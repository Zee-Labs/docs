Soyez opérationnel avec Postzee en 5 minutes. Ce guide vous accompagne dans la création de votre compte, la connexion d'un canal social, la génération de contenu IA et la publication de votre premier post.

## 1. Créez votre compte

1. Allez sur [dashboard.postzee.app](https://dashboard.postzee.app)
2. Inscrivez-vous avec votre e-mail ou via connexion sociale (Google, GitHub)
3. Choisissez votre plan — vous pouvez commencer avec **Gratuit** pour explorer les fonctionnalités IA, ou sélectionner un plan payant (**Standard**, **Team**, **Pro** ou **Ultimate**) pour débloquer la suite complète de gestion des réseaux sociaux

<Note>
Les nouveaux plans payants incluent un **essai gratuit de 7 jours**. Aucun frais pendant la période d'essai — annulez à tout moment.
</Note>

## 2. Connectez votre premier canal

1. Cliquez sur **Canaux** dans la barre de navigation supérieure
2. Cliquez sur **Ajouter un Canal**
3. Sélectionnez votre plateforme sociale (ex. : Instagram, LinkedIn, X)
4. Complétez l'autorisation OAuth pour accorder l'accès à Postzee
5. Votre canal apparaît dans la liste des canaux

<Note>
Vous avez besoin d'au moins un canal connecté pour publier des posts. Vous pouvez connecter plusieurs comptes sur différentes plateformes.
</Note>

## 3. Générez une image IA

1. Cliquez sur **Zee Studio** dans la barre de navigation supérieure
2. Vous verrez le panneau de création en bas — sélectionnez le mode **AI Image**
3. Tapez un prompt décrivant ce que vous souhaitez (ex. : "Un golden retriever portant des lunettes de soleil à la plage")
4. Cliquez sur le bouton **Améliorer avec l'IA** (icône étincelle) pour optimiser automatiquement votre prompt et obtenir de meilleurs résultats
5. Choisissez un modèle IA (ex. : **Nano Banana Pro** pour le photoréalisme ou **Recraft V4** pour le design)
6. Cliquez sur **Générer**
7. Attendez 10 à 30 secondes — votre image apparaît dans la galerie quand elle est prête

<Note>
Chaque génération coûte des crédits IA. Vérifiez votre solde dans le panneau de compte (cliquez sur votre avatar en haut à droite). Les utilisateurs du plan Gratuit peuvent acheter des crédits pour utiliser les fonctionnalités IA.
</Note>

## 4. Créez et publiez un post

1. Une fois votre image générée, cliquez dessus dans la galerie
2. Cliquez sur **Créer un Post** depuis les actions média
3. Rédigez la légende de votre post dans l'éditeur de texte — ou cliquez sur l'icône étincelle pour **générer une légende avec l'IA**
4. Sélectionnez les canaux sur lesquels publier
5. Choisissez quand publier :
   - **Publier maintenant** — publie immédiatement
   - **Programmer** — choisissez une date et une heure
   - **Brouillon** — enregistrez pour plus tard
6. Cliquez sur **Programmer** ou **Publier maintenant**

Votre post est créé ! Consultez la page **Calendrier** pour le voir dans votre planning.

## 5. Consultez vos analytics

1. Cliquez sur **Métriques** dans la barre de navigation supérieure
2. Visualisez les taux d'engagement, le nombre d'abonnés et les performances des posts sur tous vos canaux connectés
3. Filtrez par période et par plateforme
