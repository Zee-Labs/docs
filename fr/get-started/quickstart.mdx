Soyez opérationnel sur Postzee en 5 minutes. Ce guide vous accompagne pour créer votre compte, connecter un canal social, générer du contenu avec l'IA et publier votre première publication.

## 1. Créez votre compte

1. Rendez-vous sur [dashboard.postzee.app](https://dashboard.postzee.app)
2. Inscrivez-vous avec votre e-mail ou via une connexion sociale (Google, GitHub)
3. Choisissez votre forfait — vous pouvez commencer avec **Free** pour explorer les fonctionnalités d'IA, ou sélectionner un forfait payant (**Standard**, **Team**, **Pro** ou **Ultimate**) pour débloquer la suite complète de gestion des réseaux sociaux

<Note>
Les nouveaux forfaits payants incluent un **essai gratuit de 7 jours**. Aucun frais pendant la période d'essai — annulez à tout moment.
</Note>

## 2. Connectez votre premier canal

1. Cliquez sur **Canaux** dans la barre latérale gauche
2. Cliquez sur **Ajouter un canal**
3. Sélectionnez votre plateforme sociale (par exemple, Instagram, LinkedIn, X)
4. Effectuez l'autorisation OAuth pour accorder l'accès à Postzee
5. Votre canal apparaît dans l'onglet **Canaux actifs**

<Note>
Vous avez besoin d'au moins un canal connecté pour publier des posts. Vous pouvez connecter plusieurs comptes sur différentes plateformes.
</Note>

## 3. Générez une image avec l'IA

1. Cliquez sur **Zee Studio** dans la barre latérale gauche
2. Vous verrez le panneau de création en bas de l'écran — sélectionnez le mode **Image IA**
3. Saisissez un prompt décrivant ce que vous voulez (par exemple, « Un golden retriever portant des lunettes de soleil sur la plage »)
4. Cliquez sur le bouton **Améliorer avec l'IA** (icône d'étincelles) pour optimiser automatiquement votre prompt afin d'obtenir de meilleurs résultats
5. Choisissez un modèle d'IA (par exemple, **Nano Banana Pro** pour le photoréalisme ou **Recraft V4** pour le design)
6. Cliquez sur **Générer**
7. Patientez 10 à 30 secondes — votre image apparaît dans la galerie une fois prête

<Note>
Chaque génération consomme des crédits IA. Vérifiez votre solde sur la page **Crédits**. Les utilisateurs du forfait Free peuvent acheter des crédits pour utiliser les fonctionnalités d'IA.
</Note>

## 4. Créez et publiez un post

1. Une fois votre image générée, cliquez dessus dans la galerie
2. Cliquez sur **Créer un post** dans les actions de média
3. Rédigez la légende de votre post dans l'éditeur de texte — ou cliquez sur l'icône d'étincelles pour **générer une légende avec l'IA**
4. Sélectionnez les canaux sur lesquels publier
5. Choisissez quand publier :
   - **Publier maintenant** — publication immédiate
   - **Planifier** — choisissez une date et une heure
   - **Brouillon** — enregistrez pour plus tard
6. Cliquez sur **Planifier** ou **Publier maintenant**

Votre post est créé ! Consultez la page **Calendrier** pour le voir dans votre planning.

## 5. Consultez vos métriques

1. Cliquez sur **Métriques** dans la barre latérale gauche
2. Visualisez les taux d'engagement, le nombre d'abonnés et la performance des posts sur tous vos canaux connectés
3. Filtrez par période et par plateforme
