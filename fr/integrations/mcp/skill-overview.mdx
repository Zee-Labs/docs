La **Postzee Skill** est un ensemble structuré d'instructions distribué via GitHub (et ClawHub pour OpenClaw) qui apprend à votre agent IA à utiliser Postzee comme le ferait une équipe créative senior.

Une fois installée, votre agent cesse d'être un "wrapper d'outils" et commence à se comporter comme une agence créative : il fait des briefs, choisit le bon modèle, valide le coût, écrit des légendes qui convertissent et publie dans l'ordre optimal.

<Note>
La skill est distribuée publiquement et fonctionne avec tout agent compatible MCP qui supporte les skills (Claude Code, OpenClaw, Hermes Agent et autres). Tout le travail Postzee se fait via le [serveur MCP HTTP](/integrations/mcp/overview) — jamais par appels REST directs.
</Note>

## Fonctionnalités clés

### 🌍 Multilingue natif

Votre agent **répond toujours dans votre langue** — français, anglais, portugais, espagnol, allemand, japonais ou autre. Bien que la skill soit écrite en anglais, l'agent traduit tout dans votre langue et adapte le ton culturellement.

### 💳 Conscience du forfait et des crédits

Avant de générer quoi que ce soit, l'agent lit le contexte complet de votre compte — niveau du forfait, solde de crédits, stockage, statut des canaux — et **anticipe les problèmes au lieu de s'y heurter en cours de route** :

- **Forfait Free + envie de publier ?** Génère l'asset, puis propose le bon forfait payant avec une copy persuasive et offre la livraison de fichiers comme alternative.
- **Crédits faibles ?** Affiche le déficit et recommande le pack de crédits adapté — sans paywalls surprise.
- **Canaux à reconnecter ?** Vous dit lequel et comment (reconnecter vs réactiver — actions différentes).
- **Stockage presque plein ?** Vous prévient avant de générer, donc les crédits ne sont pas gaspillés sur des fichiers qui échouent à enregistrer.

### 💰 Transparence du coût

L'agent **affiche toujours le coût de génération en crédits avant de générer** — jamais en dollars, jamais comme une surprise. Il utilise l'outil live `POSTZEE_ESTIMATE_GENERATION_COST` pour calculer les chiffres exacts à partir du pricing actuel — pas de devinettes, pas de valeurs hardcodées.

### 🎨 Maîtrise des carrousels (rendu HTML — v3.4+)

Les carrousels passent par un pipeline dédié : l'agent **écrit le HTML de chaque slide** (texte au pixel près, votre typographie) et Postzee rend en PNG via Chromium headless. Les modèles d'image IA rendent mal le texte — cette approche élimine cette contrainte entièrement.

La skill impose un **workflow de 6 phases** : Brief (4 questions, analyse de concurrent optionnelle) → Scénario (plan complet avec 3 variantes de hook et 3 de légende) → Approbation (hard stop, pas de génération silencieuse) → Rendu (HTML + backgrounds Nano Banana optionnels) → Itération (remplacement chirurgical de slide via `POSTZEE_REPLACE_CAROUSEL_SLIDE` — jamais de re-rendu complet pour 1 ajustement) → Publication.

Le deep-dive vit dans `reference/carousel-mastery.md` — design system (échelle typographique, palettes, polices, espacement), traitement de logo (CSS style photo de profil IG), 5 patterns de layout (hook, card numéroté, citation, CTA, bg+overlay composé), playbook d'analyse de concurrent, variantes de légende, tactique meta-proof et le template de scénario que l'agent remplit avant de demander votre approbation.

### 🎬 Vidéo multi-scènes

Pour les histoires avec 2+ scènes connectées, la skill orchestre la cohérence des personnages et des décors entre plusieurs générations. Les stratégies sont conditionnées à ce qui est actuellement disponible via MCP — l'agent ne propose que ce qui fonctionne de bout en bout.

### 🎙️ Décisions talking-head

Quand vous demandez quelqu'un parlant un texte spécifique, la skill choisit intelligemment entre **HeyGen** (lip-sync parfait, contrôle de voix, multilingue, format long) et **Sora 2** / **Veo 3.1** (scènes cinématographiques avec audio natif) — et vous prévient à l'avance lequel facture sur les crédits Postzee et lequel utilise les crédits HeyGen.

### 🎞️ Composition avec ffmpeg + Whisper

Pour les agents avec accès shell (OpenClaw, Hermes, Claude Code avec shell), la skill inclut un cookbook complet : concat avec crossfades, mixage audio avec sidechain ducking, normalisation de loudness (-14 LUFS), conversion d'aspect ratio, Ken Burns, picture-in-picture. Plus des workflows de sous-titres Whisper avec 5 styles.

### 🔍 Conscience des tendances

Pour le travail créatif, l'agent fait une recherche web rapide des patterns viraux actuels avant de proposer des concepts — sans citer les sources sauf si vous le demandez.

### 📅 Planification intelligente

L'agent connaît les meilleurs horaires de publication par plateforme, **dans votre fuseau horaire** (lu depuis les paramètres de votre org Postzee) — et convertit en UTC ISO lors de la planification.

### 🧭 Routage intelligent des modèles (v3.2)

Vous n'avez pas besoin de savoir quel modèle utiliser. Dites *"crée une photo de coucher de soleil pour Instagram"* et l'agent choisit la bonne famille (texte-lourd vs photoréaliste vs illustration), le bon tier (fast / balanced / premium) et le bon aspect ratio pour la plateforme — automatiquement. Le tier par défaut est **fast** pour garder le coût de l'automatisation prévisible. (Les carrousels passent par `POSTZEE_RENDER_CAROUSEL` et n'ont pas besoin de sélection de modèle — le pipeline HTML s'en charge.)

Si vous SAVEZ ce que vous voulez — dites *"avec Nano Banana"* ou *"avec la meilleure qualité"* — l'agent respecte votre override. L'arbre de décision, le vocabulaire multilingue de signals (PT / EN / ES / FR / DE) et les exemples complets vivent dans `reference/smart-routing.md`.

### 🔄 Auto-vérification de version

À chaque nouvelle session, l'agent vérifie si la version installée de la skill correspond à celle actuellement publiée. Si vous êtes en retard, il vous prévient une fois (dans votre langue) avec la commande de mise à jour pour votre client — et continue de travailler avec la version que vous avez.

## Architecture

La skill est distribuée comme une **structure multi-fichiers** :

```
postzee-skill/
├── SKILL.md                          # orchestrateur (chargé en premier)
└── reference/
    ├── smart-routing.md              # Arbre de décision pour modèle / aspect / qualité (v3.2)
    ├── media-memory.md               # Pattern de manifest + réutilisation de médias entre turns/sessions (v3.3)
    ├── plans-and-pricing.md          # Les 5 forfaits + 5 packs + copy CTA
    ├── credit-aware-flow.md          # Matrice d'états forfait/crédits
    ├── carousel-mastery.md           # Pipeline rendu HTML + design system + template de scénario + replace-slide
    ├── captions-frameworks.md        # AIDA / PAS / BAB / FAB / 4Ps
    ├── hooks-library.md              # 80+ hooks éprouvés
    ├── multi-scene-workflow.md       # Stratégies multi-scènes
    ├── heygen-vs-aivideo.md          # Matrice de décision talking-head
    ├── ffmpeg-cookbook.md            # Recettes de composition
    ├── subtitle-workflows.md         # Whisper + styles de sous-titres
    └── trends-2026.md                # Macro shifts et tendances de format
```

L'orchestrateur se charge à chaque session ; les fichiers de référence sont chargés à la demande.

## Ce que la skill ne fait jamais

- ❌ Coter des prix en dollars pour les générations IA (toujours des crédits — `1 $ USD = 1 000 crédits` est le ratio d'affichage)
- ❌ Recommander une fonctionnalité non disponible via MCP
- ❌ Générer sans vous montrer d'abord le coût
- ❌ Sauter la vérification de version dans une nouvelle session
- ❌ Publier automatiquement sur un canal déconnecté ou nécessitant une reauth
- ❌ Traduire les légendes littéralement entre langues — adapte culturellement
