Connectez Postzee à **Claude Code** pour générer des images/vidéos avec l'IA, créer des vidéos avec avatar HeyGen et publier sur plus de 30 réseaux sociaux — le tout par conversation en langage naturel.

## Installer la Skill Postzee

La façon la plus simple de se connecter — installez la skill depuis GitHub.

### Option 1 : GitHub CLI

```bash
gh skill install Zee-Labs/postzee-skill
```

### Option 2 : Installation manuelle

```bash
mkdir -p ~/.claude/skills/postzee
curl -o ~/.claude/skills/postzee/SKILL.md https://raw.githubusercontent.com/Zee-Labs/postzee-skill/main/SKILL.md
```

## Configurer la Connexion MCP

Après avoir installé la skill, connectez votre compte Postzee :

1. Allez dans [Paramètres Postzee](https://dashboard.postzee.app/settings) → onglet **API Pública**
2. Copiez l'**URL MCP** dans la section "MCP (Model Context Protocol)"
3. Exécutez la commande suivante dans Claude Code :

```bash
claude mcp add postzee VOTRE_URL_MCP
```

Remplacez `VOTRE_URL_MCP` par l'URL complète copiée (ressemble à `https://api.postzee.app/mcp/.../http`).

### Vérifier la connexion

```
Liste mes canaux Postzee
```

Claude Code devrait appeler l'outil `POSTZEE_LIST_CHANNELS` et afficher vos comptes de réseaux sociaux connectés.

## Exemples d'utilisation

Une fois connecté, utilisez le langage naturel :

### Générer une image

> "Génère une photo professionnelle d'une femme dans un bureau moderne"

### Transformer une photo (image vers image)

> "Transforme cette photo en style anime" (fournissez une URL d'image)

### Animer une photo en vidéo

> "Anime cette photo en vidéo cinématographique pour les Reels Instagram"

### Créer une vidéo avatar HeyGen

> "Crée une vidéo HeyGen d'un avatar présentant notre nouveau produit"

### Générer et publier

> "Génère une photo de plage au coucher de soleil et publie-la sur mon Instagram avec la légende 'Heure dorée'"

### Vérifier les crédits

> "Combien de crédits IA me reste-t-il ?"

## Workflow recommandé

1. **Vérifier les crédits** — confirmez que vous avez assez de crédits
2. **Optimiser le prompt** — utilisez l'optimiseur pour de meilleurs résultats (gratuit)
3. **Générer** — créez l'image ou la vidéo
4. **Publier** — publiez ou programmez sur vos canaux
