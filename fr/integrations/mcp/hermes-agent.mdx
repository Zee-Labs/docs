Connectez Postzee à **Hermes Agent** de Nous Research pour générer des images/vidéos avec l'IA, créer des vidéos avatar HeyGen et publier sur plus de 30 réseaux sociaux.

## Installer la Skill Postzee

```bash
hermes skills install Zee-Labs/postzee-skill
```

## Configurer la Connexion MCP

1. Allez dans [Paramètres Postzee](https://dashboard.postzee.app/settings) → onglet **API Pública**
2. Copiez l'**URL MCP**
3. Configurez dans Hermes :

```bash
hermes mcporter add postzee --transport http VOTRE_URL_MCP
```

### Vérifier la connexion

> "Liste mes canaux Postzee"

## Exemples d'utilisation

- "Génère une photo professionnelle dans un bureau moderne"
- "Transforme cette photo en style anime"
- "Anime cette photo en vidéo pour les Reels"
- "Crée une vidéo HeyGen présentant notre produit"
- "Génère une photo de plage et publie-la sur Instagram"
