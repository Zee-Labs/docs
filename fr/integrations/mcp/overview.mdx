Postzee inclut un **serveur MCP** (Model Context Protocol) intégré permettant aux agents IA d'interagir avec votre compte. Générez des images, créez des vidéos, créez des vidéos avatar HeyGen, publiez des posts et gérez vos réseaux sociaux — le tout par conversation en langage naturel.

## Qu'est-ce que MCP ?

Le [Model Context Protocol](https://modelcontextprotocol.io) est un standard ouvert pour connecter des modèles IA à des outils externes. Le serveur MCP de Postzee expose **20 outils**.

## Agents IA supportés

| Agent | Type d'intégration | Temps de setup |
|-------|-------------------|----------------|
| **Claude Code** | Skill (GitHub) | < 1 minute |
| **OpenClaw** | Skill (ClawHub) | < 1 minute |
| **Hermes Agent** | Skill (GitHub) | < 1 minute |
| **Tout client MCP** | Connexion HTTP directe | ~ 1 minute |

## Détails de connexion

| Propriété | Valeur |
|-----------|--------|
| **Version** | 3.1.0 |
| **Transport** | HTTP Streamable (recommandé) + SSE (legacy) |
| **Endpoint HTTP** | `https://api.postzee.app/mcp/{apiKey}/http` |
| **Outils** | 12 |

## Prérequis

- Un **compte Postzee** (gratuit ou payant)
- L'**URL MCP** (copiez dans Paramètres → API Pública → section MCP)
