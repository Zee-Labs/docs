Connectez Postzee à **OpenClaw** pour générer des images/vidéos avec l'IA, créer des vidéos avec avatar HeyGen et gérer vos publications sur les réseaux sociaux via votre agent OpenClaw.

## Installer la Skill Postzee

Installez la skill depuis ClawHub :

```bash
clawhub install postzee
```

Ou dites directement à votre agent OpenClaw :

> "Installe la skill Postzee"

## Configurer la Connexion MCP

1. Allez dans [Paramètres Postzee](https://dashboard.postzee.app/settings) → onglet **API Pública**
2. Copiez l'**URL MCP** dans la section "MCP (Model Context Protocol)"
3. Dites à votre agent : "Configure Postzee avec cette URL MCP : `VOTRE_URL_MCP`"

### Vérifier la connexion

> "Liste mes canaux Postzee"

## Outils disponibles

20 outils disponibles pour votre agent :

| Outil | Description |
|-------|-------------|
| `POSTZEE_LIST_CHANNELS` | Lister les canaux de réseaux sociaux |
| `POSTZEE_GET_CREDITS` | Vérifier le solde de crédits |
| `POSTZEE_LIST_IMAGE_MODELS` | Lister les modèles d'image avec coûts |
| `POSTZEE_LIST_VIDEO_MODELS` | Lister les modèles de vidéo avec coûts |
| `POSTZEE_ENHANCE_PROMPT` | Optimiser les prompts (gratuit) |
| `POSTZEE_GENERATE_IMAGE` | Générer une image IA (images de référence, ratio, qualité, style) |
| `POSTZEE_GENERATE_VIDEO` | Générer une vidéo IA (image vers vidéo, durée, ratio) |
| `POSTZEE_GENERATE_HEYGEN_VIDEO` | Créer une vidéo avatar HeyGen |
| `POSTZEE_LIST_HEYGEN_AVATARS` | Lister les avatars HeyGen |
| `POSTZEE_LIST_HEYGEN_VOICES` | Lister les voix HeyGen |
| `POSTZEE_CHECK_JOB` | Vérifier l'état de la génération |
| `POSTZEE_CREATE_POST` | Créer, programmer ou publier un post |

## Exemples d'utilisation

- "Génère une photo professionnelle d'un produit"
- "Transforme cette photo en peinture à l'huile" (envoyez une photo par Telegram)
- "Anime ma photo en vidéo pour TikTok" (envoyez une photo par Telegram)
- "Crée une vidéo HeyGen présentant notre entreprise"
- "Publie une image sur LinkedIn avec le texte 'Encore une journée productive'"
- "Combien de crédits ai-je ?"

### Envoyer des images de référence via Telegram

Vous pouvez envoyer une photo directement dans le chat Telegram. L'agent l'utilisera automatiquement comme référence.
