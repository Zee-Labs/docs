Connectez votre compte X (anciennement Twitter) a Postzee pour publier des tweets, des threads et des medias depuis le tableau de bord.

## Prerequis

- Un compte **X (Twitter)**

## Comment connecter

1. Allez dans **Canaux** dans la barre de navigation superieure
2. Cliquez sur **Ajouter un Canal** → **X (Twitter)**
3. Connectez-vous avec votre compte X
4. Autorisez Postzee
5. Votre canal X apparait dans la liste des canaux

## Types de contenu pris en charge

| Type | Description |
|------|-------------|
| **Tweet** | Post texte (jusqu'a 280 caracteres) |
| **Thread** | Plusieurs tweets connectes |
| **Tweet avec image** | Tweet avec jusqu'a 4 images |
| **Tweet avec video** | Tweet avec une video en piece jointe |

## Conseils

- **Limite de caracteres :** 280 caracteres par tweet. Postzee vous avertit si votre contenu depasse cette limite.
- **Threads :** Creez des threads multi-tweets pour du contenu plus long. Chaque tweet du thread est connecte automatiquement.
- **Images :** Jusqu'a 4 images par tweet. Formats pris en charge : JPEG, PNG, GIF, WEBP.
- **Formats d'image :** Le paysage (16:9) et le carre (1:1) s'affichent le mieux dans le fil.
