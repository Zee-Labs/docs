Connectez votre compte Pinterest a Postzee pour creer des Pins avec des images, des descriptions et un ciblage de tableaux.

## Prerequis

- Un compte **Pinterest** (personnel ou professionnel)

## Comment connecter

1. Allez dans **Canaux** dans la barre de navigation superieure
2. Cliquez sur **Ajouter un Canal** → **Pinterest**
3. Connectez-vous avec votre compte Pinterest
4. Autorisez Postzee
5. Votre canal Pinterest apparait dans la liste des canaux

## Types de contenu pris en charge

| Type | Description |
|------|-------------|
| **Pin** | Pin image avec description et URL de lien |

## Conseils

- **Format d'image :** Les images verticales (2:3) fonctionnent le mieux sur Pinterest.
- **URL du lien :** Ajoutez une URL de destination pour diriger le trafic vers votre site web ou page d'atterrissage.
- **Selection du tableau :** Choisissez sur quel tableau epingler lors de la creation d'un post.
- **Descriptions :** Utilisez des descriptions riches en mots-cles pour la decouverte via la recherche Pinterest.
