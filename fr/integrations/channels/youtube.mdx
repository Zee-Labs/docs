Connectez votre chaine YouTube a Postzee pour telecharger des videos et des Shorts avec des titres, descriptions, miniatures et parametres de visibilite.

## Prerequis

- Un **compte Google** avec une **chaine YouTube**

## Comment connecter

1. Allez dans **Canaux** dans la barre de navigation superieure
2. Cliquez sur **Ajouter un Canal** → **YouTube**
3. Connectez-vous avec votre compte Google
4. Selectionnez la chaine YouTube que vous souhaitez gerer
5. Autorisez les permissions requises
6. Votre chaine YouTube apparait dans la liste des canaux

## Types de contenu pris en charge

| Type | Description |
|------|-------------|
| **Video** | Telechargement de video standard avec titre, description et tags |
| **Short** | Video courte verticale (9:16, jusqu'a 60 secondes) |

## Parametres specifiques a la plateforme

| Parametre | Options | Description |
|-----------|---------|-------------|
| **Titre** | Texte (2-100 caracteres) | Titre de la video — obligatoire pour tous les telechargements YouTube |
| **Visibilite** | Public, Prive, Non repertorie | Qui peut voir votre video |
| **Miniature** | Telechargement d'image | Miniature video personnalisee |
| **Tags** | Tags textuels | Mots-cles recherchables pour la decouverte |

## Conseils

- **Le titre est obligatoire :** YouTube exige un titre pour chaque video. Postzee vous invitera a en ajouter un si vous oubliez.
- **Miniatures :** Les miniatures personnalisees ameliorent considerablement les taux de clics. Telechargez une image de 1280x720.
- **Formats d'image :** Utilisez le 16:9 pour les videos standard et le 9:16 pour les Shorts.
- **Tags :** Ajoutez des tags pertinents pour aider l'algorithme de YouTube a recommander votre contenu.
