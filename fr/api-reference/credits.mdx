Verifiez votre solde de credits IA et ameliorez les prompts via l'API.

## Obtenir le solde de credits

```
GET /public/v1/credits
```

Retourne votre solde de credits actuel.

### Requete

```bash
curl -H "Authorization: YOUR_API_KEY" \
  https://api.postzee.app/public/v1/credits
```

### Reponse

```json
{
  "monthlyCredits": 5.0,
  "purchasedCredits": 2.0,
  "usedCredits": 0.5,
  "availableCredits": 6.5,
  "lastReset": "2026-04-01T00:00:00.000Z"
}
```

| Champ | Type | Description |
|-------|------|-------------|
| `monthlyCredits` | number | Credits mensuels de votre plan (en USD) |
| `purchasedCredits` | number | Credits achetes qui n'expirent jamais (en USD) |
| `usedCredits` | number | Credits utilises dans le cycle de facturation en cours (en USD) |
| `availableCredits` | number | Total des credits disponibles — mensuels + achetes - utilises (en USD) |
| `lastReset` | string | Horodatage ISO de la derniere reinitialisation mensuelle des credits |

<Note>
Toutes les valeurs de credits sont en USD. Pour convertir en nombre de credits : `valeur x 1 000`. Par exemple, `availableCredits: 6.5` = `6 500` credits disponibles.
</Note>

## Ameliorer un prompt

```
POST /public/v1/enhance-prompt
```

Optimisez un prompt pour de meilleurs resultats de generation IA. Cet endpoint est **gratuit** — aucun credit n'est consomme.

**Limite de debit :** 10 requetes par minute.

### Requete

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "a cat on a windowsill",
    "mediaType": "image"
  }' \
  https://api.postzee.app/public/v1/enhance-prompt
```

### Parametres

| Parametre | Type | Requis | Description |
|-----------|------|:---:|-------------|
| `prompt` | string | Oui | Le prompt a ameliorer (max 2 000 caracteres) |
| `mediaType` | string | Oui | `"image"` ou `"video"` |
| `model` | string | Non | ID du modele IA cible pour une optimisation specifique au modele |
| `stylePreferences` | string | Non | Indices de style (max 500 caracteres) |
| `sourceImageUrls` | string[] | Non | URLs d'images de reference (max 5) |

### Reponse

```json
{
  "enhancedPrompt": "A fluffy ginger cat perched on a sunlit windowsill, warm afternoon light casting soft shadows through lace curtains...",
  "success": true
}
```

| Champ | Type | Description |
|-------|------|-------------|
| `enhancedPrompt` | string | Le prompt optimise |
| `success` | boolean | Si l'amelioration a reussi |
| `error` | string | Message d'erreur (uniquement si `success` est `false`) |
