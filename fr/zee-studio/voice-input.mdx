Dictez vos prompts par la voix au lieu de taper. Appuyez sur l'icône microphone, dites votre idée, et Postzee la transcrit dans le champ de prompt — prête à générer ou à améliorer.

## Comment utiliser l'entrée vocale

1. Ouvrez **Zee Studio** depuis la barre de navigation supérieure (cliquez sur **Studio** → **Zee Studio**) et sélectionnez n'importe quel mode de création (**AI Image**, **AI Video** ou **Create Post**)
2. Appuyez sur l'**icône microphone** (🎤) dans le panneau de création
3. **Autorisez l'accès au microphone** quand votre navigateur le demande (première fois uniquement)
4. **Dictez votre prompt** — un indicateur rouge d'enregistrement montre que vous êtes enregistré
5. Quand vous avez fini de parler, appuyez à nouveau sur l'icône microphone pour arrêter
6. Attendez 1 à 2 secondes pendant la transcription de l'audio — un indicateur ambre montre que la transcription est en cours
7. Vos mots prononcés apparaissent dans le champ de prompt sous forme de texte

## États d'enregistrement

| État | Indicateur | Ce qui se passe |
|------|-----------|----------------|
| **Prêt** | Icône microphone par défaut | Appuyez pour commencer l'enregistrement |
| **Enregistrement** | Indicateur rouge pulsant | Votre voix est captée |
| **Transcription** | Points animés ambre | L'audio est converti en texte |

## Fonctionne partout

L'entrée vocale fonctionne sur desktop et mobile :

- **Desktop** — cliquez sur l'icône microphone dans le panneau de création
- **Mobile** — appuyez sur l'icône microphone (même emplacement, optimisé pour le tactile)

<Note>
Votre navigateur doit supporter l'accès au microphone. La plupart des navigateurs modernes (Chrome, Safari, Firefox, Edge) le supportent. Assurez-vous d'autoriser les permissions du microphone quand demandé.
</Note>

## Combiner avec l'Optimiseur de Prompt

L'entrée vocale se combine parfaitement avec l'Optimiseur de Prompt pour le workflow le plus rapide :

1. **Parlez** votre idée de façon décontractée ("un chien courant sur la plage au coucher du soleil")
2. **Améliorez** — cliquez sur le bouton étincelle (✨) pour transformer votre description décontractée en prompt professionnel
3. **Générez** — cliquez sur Générer pour créer votre image ou vidéo

Cela vous permet de passer de l'idée à la génération en quelques secondes sans taper un seul mot.

## Conseils

- **Parlez naturellement** — vous n'avez pas besoin d'utiliser des termes techniques de photographie ; l'Optimiseur de Prompt s'en charge
- **Soyez précis** — "un chiot golden retriever jouant sur une plage tropicale" donne de meilleurs résultats que "un chien"
- **Une idée à la fois** — décrivez une seule scène ou concept par enregistrement
- **Environnement calme** — enregistrer dans un endroit calme améliore la précision de la transcription
