Générez des vidéos par IA directement dans Zee Studio. Choisissez parmi plusieurs modèles optimisés pour différents styles — mouvement cinématographique, itérations rapides, qualité professionnelle et plus encore.

## Comment générer une vidéo

1. Ouvrez **Zee Studio** depuis la barre latérale
2. Sélectionnez le mode **Vidéo IA** dans le panneau de création
3. Choisissez votre mode de génération :
   - **Texte vers vidéo** — décrivez la scène que vous voulez dans le champ de prompt
   - **Image vers vidéo** — joignez une image de référence et décrivez le mouvement souhaité
4. Saisissez votre prompt (par exemple, « Un coucher de soleil doré sur un océan calme avec de douces vagues »)
5. *(Optionnel)* Cliquez sur le bouton **Améliorer avec l'IA** (✨) pour optimiser votre prompt pour la vidéo — il ajoute automatiquement les mouvements de caméra, l'éclairage et les descriptions de mouvement
6. Cliquez sur la puce du modèle pour sélectionner un modèle d'IA et ajuster les paramètres
7. Vérifiez le coût en crédits affiché sur le bouton **Générer**
8. Cliquez sur **Générer**
9. Patientez pendant le traitement — la génération de vidéo prend généralement 1 à 5 minutes selon le modèle et la durée
10. Votre vidéo apparaît dans **Mes Créations** une fois prête

<Note>
La génération de vidéo prend plus de temps que celle d'image. Vous pouvez fermer Zee Studio et revenir plus tard — votre vidéo vous attendra dans **Mes Créations** lorsqu'elle sera terminée.
</Note>

## Paramètres du modèle

Cliquez sur la puce du modèle (par exemple, **Sora 2**) pour ouvrir le panneau des paramètres :

- **Sélecteur de modèle** — parcourez et basculez entre les modèles vidéo disponibles
- **Durée** — choisissez la longueur de la vidéo :
  - Modèles Sora 2 : 10 ou 15 secondes
  - Modèles Veo 3.1 : 4, 6 ou 8 secondes
- **Format d'image** — paysage (16:9), portrait (9:16) ou carré (1:1)
- **Résolution** — 720p (HD) ou 1080p (Full HD) lorsque pris en charge
- **Génération audio** — activez l'audio synchronisé (disponible sur Veo 3.1 Fast)
- **Prompt négatif** — décrivez ce que vous ne voulez pas dans la vidéo (pris en charge par certains modèles)

### Formats d'image recommandés par plateforme

| Plateforme | Recommandé |
|----------|-------------|
| Instagram Reels, TikTok, YouTube Shorts | Portrait 9:16 |
| YouTube, Facebook Video | Paysage 16:9 |
| Fil Instagram, LinkedIn | Carré 1:1 |

## Animer des images (Image vers vidéo)

En mode **Image vers vidéo**, vous pouvez donner vie à n'importe quelle image fixe :

1. Passez en mode **Image vers vidéo** dans le panneau de création
2. Cliquez sur l'icône d'image pour joindre votre image source
3. Décrivez le mouvement souhaité dans le prompt (par exemple, « La caméra effectue un zoom lent tandis que les feuilles oscillent doucement au vent »)
4. Sélectionnez un modèle prenant en charge l'image vers vidéo :
   - **Sora 2** (Image vers vidéo)
   - **Veo 3.1** (Image vers vidéo)
5. Cliquez sur **Générer**

<Note>
Pour de meilleurs résultats, utilisez des images sources de haute qualité. L'IA préserve le style et la composition de votre image originale tout en ajoutant un mouvement naturel.
</Note>

## Conseils pour de meilleures vidéos

- **Décrivez le mouvement** — « La caméra effectue un travelling vers la gauche pour révéler une chaîne de montagnes » fonctionne mieux que « montagnes »
- **Utilisez l'Optimiseur de prompts** — il ajoute automatiquement des mouvements de caméra, des transitions lumineuses et une dynamique de mouvement
- **Précisez le travail de caméra** — « zoom lent », « plan en travelling », « vue aérienne par drone », « dolly avant »
- **Mentionnez les transitions** — « la scène passe de l'aube au lever du soleil avec une chaude lumière dorée »
- **Restez concentré** — une scène claire par génération fonctionne mieux que des descriptions complexes à plusieurs scènes
- **Essayez différentes durées** — les vidéos courtes (4-6s) sont idéales pour les boucles sur réseaux sociaux, les plus longues (10-15s) pour la narration

## Après la génération

Une fois votre vidéo prête, cliquez dessus pour :

- **Télécharger** — enregistrez-la sur votre appareil au format MP4
- **Créer un post** — ouvrez l'éditeur de post avec la vidéo attachée
- **Recréer** — réutilisez le même modèle et les mêmes paramètres avec un nouveau prompt
- **Utiliser le prompt** — copiez le prompt dans le champ de création
- **Publier** — partagez-la dans la galerie publique Explorer
