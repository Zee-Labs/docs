Générez des vidéos IA directement dans Zee Studio. Choisissez parmi plusieurs modèles optimisés pour différents styles — mouvement cinématique, itérations rapides, qualité professionnelle, et plus encore.

## Comment générer une vidéo

1. Ouvrez **Zee Studio** depuis la barre de navigation supérieure (cliquez sur **Studio** → **Zee Studio**)
2. Sélectionnez le mode **AI Video** dans le panneau de création
3. Choisissez votre mode de génération :
   - **Texte à Vidéo** — décrivez la scène souhaitée dans le champ de prompt
   - **Image à Vidéo** — joignez une image de référence et décrivez le mouvement souhaité
4. Tapez votre prompt (ex. : "Un coucher de soleil doré sur un océan calme avec des vagues douces")
5. *(Optionnel)* Cliquez sur le bouton **Améliorer avec l'IA** (✨) pour optimiser votre prompt pour la vidéo — cela ajoute automatiquement des mouvements de caméra, l'éclairage et des descriptions de mouvement
6. Cliquez sur la puce du modèle pour sélectionner un modèle IA et ajuster les paramètres
7. Vérifiez le coût en crédits affiché sur le bouton **Générer**
8. Cliquez sur **Générer**
9. Attendez le traitement — la génération de vidéo prend généralement 1 à 5 minutes selon le modèle et la durée
10. Votre vidéo apparaît dans **My Creations** quand elle est prête

<Note>
La génération de vidéo prend plus de temps que la génération d'images. Vous pouvez fermer Zee Studio et revenir plus tard — votre vidéo vous attendra dans **My Creations** quand elle sera terminée.
</Note>

## Paramètres du modèle

Cliquez sur la puce du modèle (ex. : **Sora 2**) pour ouvrir le panneau de paramètres :

- **Sélecteur de modèle** — parcourez et changez de modèle vidéo disponible
- **Durée** — choisissez la longueur de la vidéo :
  - Modèles Sora 2 : 10 secondes ou 15 secondes
  - Modèles Veo 3.1 : 4, 6 ou 8 secondes
- **Proportion** — paysage (16:9), portrait (9:16) ou carré (1:1)
- **Résolution** — 720p (HD) ou 1080p (Full HD) selon le support
- **Génération audio** — activez l'audio synchronisé (disponible sur Veo 3.1 Fast)
- **Prompt Négatif** — décrivez ce que vous ne souhaitez pas dans la vidéo (supporté par certains modèles)

### Proportions recommandées par plateforme

| Plateforme | Recommandé |
|------------|-----------|
| Instagram Reels, TikTok, YouTube Shorts | Portrait 9:16 |
| YouTube, Facebook Video | Paysage 16:9 |
| Instagram Feed, LinkedIn | Carré 1:1 |

## Animer des images (Image à Vidéo)

En mode **Image à Vidéo**, vous pouvez donner vie à n'importe quelle image fixe :

1. Passez en mode **Image à Vidéo** dans le panneau de création
2. Cliquez sur l'icône image pour joindre votre image source
3. Décrivez le mouvement souhaité dans le prompt (ex. : "La caméra zoome lentement tandis que les feuilles se balancent doucement dans le vent")
4. Sélectionnez un modèle qui supporte l'image-vers-vidéo :
   - **Sora 2** (Image to Video)
   - **Veo 3.1** (Image to Video)
5. Cliquez sur **Générer**

<Note>
Pour de meilleurs résultats, utilisez des images source de haute qualité. L'IA préserve le style et la composition de votre image originale tout en ajoutant un mouvement naturel.
</Note>

## Conseils pour de meilleurs résultats vidéo

- **Décrivez le mouvement** — "La caméra panoramique vers la gauche révèle une chaîne de montagnes" fonctionne mieux que "montagnes"
- **Utilisez l'Optimiseur de Prompt** — il ajoute automatiquement des mouvements de caméra, des transitions d'éclairage et des dynamiques de mouvement
- **Spécifiez le travail de caméra** — "zoom avant lent", "plan de suivi", "vue aérienne drone", "travelling avant"
- **Mentionnez les transitions** — "la scène passe de l'aube au lever du soleil avec une lumière dorée et chaude"
- **Restez concentré** — une seule scène claire par génération fonctionne mieux que des descriptions multi-scènes complexes
- **Essayez différentes durées** — les vidéos courtes (4-6s) sont idéales pour les boucles sur les réseaux sociaux, les plus longues (10-15s) pour la narration

## Après la génération

Une fois votre vidéo prête, cliquez dessus pour :

- **Télécharger** — enregistrez sur votre appareil au format MP4
- **Créer un Post** — ouvrez l'éditeur de post avec la vidéo jointe
- **Recréer** — réutilisez le même modèle et les mêmes paramètres avec un nouveau prompt
- **Utiliser le prompt** — copiez le prompt dans le champ de création
- **Publier** — partagez dans la galerie publique Explore
