Créez des vidéos professionnelles avec des présentateurs virtuels grâce aux avatars IA propulsés par **HeyGen**. Choisissez parmi une bibliothèque d'avatars réalistes, sélectionnez une voix, rédigez votre script et générez une vidéo — sans caméra ni microphone nécessaires.

## Que sont les avatars HeyGen ?

Les avatars HeyGen sont des présentateurs vidéo générés par IA qui synchronisent leurs lèvres avec votre script. Ils ressemblent et bougent comme de vraies personnes, ce qui les rend idéaux pour :

- Démos de produits et vidéos explicatives
- Contenu pour les réseaux sociaux avec une touche personnelle
- Vidéos de formation et éducatives
- Annonces marketing
- Contenu multilingue (même avatar, différentes langues)

## Pour commencer

### Connectez votre compte HeyGen

Avant d'utiliser les avatars, vous devez connecter votre compte HeyGen :

1. Ouvrez le panneau de compte (cliquez sur votre avatar) → **Paramètres** → onglet **Intégrations**
2. Trouvez **HeyGen** dans la liste des intégrations
3. Cliquez sur **Connecter** et suivez le flux d'autorisation
4. Une fois connecté, les avatars deviennent disponibles dans Zee Studio

<Note>
Vous avez besoin d'un compte HeyGen pour utiliser cette fonctionnalité. Si vous n'en avez pas, vous pouvez [vous inscrire sur HeyGen](https://www.heygen.com/signup) puis le connecter à Postzee.
</Note>

## Comment créer une vidéo avec avatar

1. Ouvrez **Zee Studio** depuis la barre de navigation supérieure (cliquez sur **Studio** → **Zee Studio**)
2. Sélectionnez le mode **AI Video** dans le panneau de création
3. Choisissez **HeyGen Avatar** comme type de génération
4. **Sélectionnez un avatar** :
   - Parcourez la galerie d'avatars
   - Recherchez par nom pour trouver des avatars spécifiques
   - Survolez un avatar pour prévisualiser sa vidéo
   - Cliquez pour sélectionner
5. **Sélectionnez une voix** :
   - Parcourez les voix disponibles
   - Filtrez par langue avec le menu déroulant
   - Cliquez sur le bouton de lecture pour prévisualiser une voix
   - Cliquez pour sélectionner
6. **Rédigez votre script** — tapez ou collez le texte que votre avatar prononcera
7. Vérifiez le coût en crédits et cliquez sur **Générer**
8. Attendez le traitement — les vidéos d'avatar prennent généralement 2 à 5 minutes
9. Votre vidéo apparaît dans **My Creations** quand elle est prête

## Choisir un avatar

La galerie d'avatars affiche tous les avatars HeyGen disponibles :

- **Recherche** — trouvez des avatars par nom
- **Aperçu** — survolez une carte d'avatar pour voir un court aperçu vidéo
- **Types** — avatars classiques et photos parlantes sont disponibles
- **Genre** — parcourez les présentateurs masculins et féminins

Cliquez sur un avatar pour le sélectionner. Une bordure violette indique votre sélection.

## Choisir une voix

Le sélecteur de voix vous permet de trouver la voix parfaite pour votre contenu :

- **Filtre par langue** — sélectionnez une langue pour afficher uniquement les voix dans cette langue
- **Indicateurs de genre** — chaque voix indique si elle est féminine ou masculine
- **Aperçu audio** — cliquez sur le bouton de lecture pour écouter un échantillon avant de sélectionner
- **Recherche** — trouvez des voix par nom

<Note>
Pour du contenu multilingue, vous pouvez utiliser le même avatar avec des voix de langues différentes. L'avatar synchronisera naturellement ses lèvres avec n'importe quelle langue supportée.
</Note>

## Conseils pour de meilleures vidéos d'avatar

- **Écrivez naturellement** — rédigez votre script comme vous le diriez à l'oral, avec des pauses et une formulation naturelle
- **Gardez les scripts concis** — des scripts courts et ciblés produisent de meilleurs résultats que de longs monologues
- **Adaptez la voix au contenu** — choisissez une voix qui correspond au ton de votre marque (professionnel, amical, énergique)
- **Testez avec les aperçus** — prévisualisez toujours l'avatar et la voix avant de générer

## Après la génération

Une fois votre vidéo d'avatar prête, vous pouvez :

- **Télécharger** — enregistrez la vidéo sur votre appareil
- **Créer un Post** — joignez la vidéo à un post sur les réseaux sociaux
- **Publier** — partagez dans la galerie Explore
