Créez des vidéos talking-head professionnelles à l'aide d'avatars IA propulsés par **HeyGen**. Choisissez un avatar réaliste dans une bibliothèque, sélectionnez une voix, rédigez votre script et générez une vidéo — sans caméra ni microphone.

## Que sont les Avatars HeyGen ?

Les Avatars HeyGen sont des présentateurs vidéo générés par IA qui synchronisent leurs lèvres sur votre script. Ils ressemblent et bougent comme de vraies personnes, ce qui les rend idéaux pour :

- Démonstrations de produits et vidéos explicatives
- Contenu pour réseaux sociaux avec une touche personnelle
- Vidéos de formation et pédagogiques
- Annonces marketing
- Contenu multilingue (même avatar, langues différentes)

## Pour commencer

### Connectez votre compte HeyGen

Avant d'utiliser les avatars, vous devez connecter votre compte HeyGen :

1. Allez dans **Paramètres** → **Intégrations**
2. Trouvez **HeyGen** dans la liste des intégrations
3. Cliquez sur **Connecter** et suivez le flux d'autorisation
4. Une fois connectés, les avatars deviennent disponibles dans Zee Studio

<Note>
Vous avez besoin d'un compte HeyGen pour utiliser cette fonctionnalité. Si vous n'en avez pas, vous pouvez [vous inscrire sur HeyGen](https://www.heygen.com/signup) puis le connecter à Postzee.
</Note>

## Comment créer une vidéo avec avatar

1. Ouvrez **Zee Studio** depuis la barre latérale
2. Sélectionnez le mode **Vidéo IA** dans le panneau de création
3. Choisissez **Avatar HeyGen** comme type de génération
4. **Sélectionnez un avatar** :
   - Parcourez la galerie d'avatars
   - Recherchez par nom pour trouver des avatars spécifiques
   - Survolez un avatar pour prévisualiser sa vidéo
   - Cliquez pour sélectionner
5. **Sélectionnez une voix** :
   - Parcourez les voix disponibles
   - Filtrez par langue à l'aide du menu déroulant
   - Cliquez sur le bouton de lecture pour prévisualiser une voix
   - Cliquez pour sélectionner
6. **Rédigez votre script** — saisissez ou collez le texte que votre avatar doit prononcer
7. Vérifiez le coût en crédits et cliquez sur **Générer**
8. Patientez pendant le traitement — les vidéos d'avatar prennent généralement 2 à 5 minutes
9. Votre vidéo apparaît dans **Mes Créations** une fois prête

## Choisir un avatar

La galerie d'avatars affiche tous les avatars HeyGen disponibles :

- **Recherche** — trouvez des avatars par nom
- **Aperçu** — survolez la carte d'un avatar pour voir un court aperçu vidéo
- **Types** — avatars classiques et photos parlantes disponibles
- **Genre** — parcourez des présentateurs masculins et féminins

Cliquez sur un avatar pour le sélectionner. Une bordure violette indique votre sélection.

## Choisir une voix

Le sélecteur de voix vous aide à trouver la voix parfaite pour votre contenu :

- **Filtre de langue** — sélectionnez une langue pour n'afficher que les voix dans cette langue
- **Indicateurs de genre** — chaque voix indique si elle est féminine ou masculine
- **Aperçu audio** — cliquez sur le bouton de lecture pour entendre un échantillon avant de sélectionner
- **Recherche** — trouvez des voix par nom

<Note>
Pour du contenu multilingue, vous pouvez utiliser le même avatar avec des voix dans différentes langues. L'avatar fera la synchronisation labiale naturellement dans toutes les langues prises en charge.
</Note>

## Conseils pour de meilleures vidéos d'avatar

- **Écrivez naturellement** — rédigez votre script comme vous le diriez, avec des pauses et un phrasé naturel
- **Restez concis** — des scripts plus courts et ciblés produisent de meilleurs résultats que de longs monologues
- **Adaptez la voix au contenu** — choisissez une voix qui correspond au ton de votre marque (professionnel, amical, énergique)
- **Testez avec des aperçus** — prévisualisez toujours l'avatar et la voix avant de générer

## Après la génération

Une fois votre vidéo d'avatar prête, vous pouvez :

- **Télécharger** — enregistrez la vidéo sur votre appareil
- **Créer un post** — joignez la vidéo à un post sur les réseaux sociaux
- **Publier** — partagez-la dans la galerie Explorer
