Votre médiathèque personnelle dans Zee Studio est organisée en deux onglets : **Mes Créations** pour le contenu généré par IA et **Mes Médias** pour les fichiers téléversés manuellement.

## Mes Créations

L'onglet **Mes Créations** affiche toutes les images et vidéos que vous avez générées avec l'IA. Chaque élément affiche :

- Une vignette d'aperçu
- Le modèle d'IA utilisé pour la génération
- Depuis combien de temps il a été créé (par exemple, « il y a 2 heures »)

### Filtrer vos créations

Utilisez les pastilles de filtre au-dessus de la galerie pour affiner votre vue :

| Filtre | Affiche |
|--------|-------|
| **Tout** | Tous les médias générés |
| **Images** | Uniquement les images générées par IA |
| **Vidéos** | Uniquement les vidéos générées par IA |

### Actions sur vos créations

Cliquez sur un élément de Mes Créations pour accéder à ces actions :

| Action | Description |
|--------|-------------|
| **Télécharger** | Enregistrez le fichier sur votre appareil (JPG pour les images, MP4 pour les vidéos) |
| **Créer un post** | Ouvrez l'éditeur de post avec ce média joint |
| **Recréer** | Chargez le même modèle et les mêmes paramètres pour générer une nouvelle variation |
| **Utiliser le prompt** | Copiez le prompt original dans le champ de création |
| **Voir les détails** | Consultez toutes les métadonnées de génération — modèle, prompt, paramètres, résolution |
| **Modifier l'image** | Ouvrez l'image dans l'éditeur intégré pour la modifier (images uniquement) |
| **Animer** | Transformez une image fixe en vidéo grâce au mode Image vers vidéo (images uniquement) |
| **Publier** | Partagez-le dans la galerie publique Explorer pour que la communauté le découvre |
| **Supprimer** | Supprimez définitivement le média (avec confirmation) |

### Générations en attente

Lorsqu'une génération est en cours, elle apparaît dans Mes Créations avec un indicateur de chargement. Vous pouvez :

- Voir l'état d'avancement (en file, en traitement)
- Continuer à travailler sur d'autres tâches en attendant
- Fermer Zee Studio et revenir plus tard — l'élément sera prêt à votre retour

## Mes Médias

L'onglet **Mes Médias** affiche les fichiers que vous avez téléversés manuellement. Utilisez-le pour vos propres photos, ressources de marque, logos ou tout média que vous souhaitez utiliser dans des posts.

### Comment téléverser

- **Glisser-déposer** — faites glisser les fichiers directement sur la zone de la galerie
- **Cliquer pour parcourir** — cliquez sur la zone de téléversement pour ouvrir votre sélecteur de fichiers
- **Bouton de téléversement** — utilisez le bouton de téléversement dans la barre d'outils de la galerie

Consultez [Téléverser des médias](/zee-studio/upload) pour connaître les formats pris en charge et les limites de taille.

### Actions sur les médias téléversés

| Action | Description |
|--------|-------------|
| **Télécharger** | Enregistrez le fichier sur votre appareil |
| **Créer un post** | Joignez le média à un post sur les réseaux sociaux |
| **Animer** | Générez une vidéo à partir d'une image fixe (Image vers vidéo) |
| **Supprimer** | Supprimez le fichier téléversé |

## Comportement de la galerie

- Les médias s'affichent dans une grille responsive qui s'adapte à la taille de votre écran
- Les éléments sont triés par date de création, les plus récents en premier
- La galerie charge automatiquement plus d'éléments au défilement (scroll infini)
- Les vignettes sont optimisées pour un chargement rapide
- Les vidéos affichent une icône de lecture en superposition sur leur vignette
