Votre bibliothèque média personnelle dans Zee Studio est organisée en deux onglets : **My Creations** pour le contenu généré par IA et **My Media** pour les fichiers uploadés manuellement.

## My Creations

L'onglet **My Creations** affiche toutes les images et vidéos que vous avez générées avec l'IA. Chaque élément affiche :

- Un aperçu en miniature
- Le modèle IA utilisé pour la génération
- Le temps écoulé depuis la création (ex. : "il y a 2 heures")

### Filtrer vos créations

Utilisez les filtres au-dessus de la galerie pour affiner votre vue :

| Filtre | Affiche |
|--------|---------|
| **All** | Tous les médias générés |
| **Images** | Uniquement les images générées par IA |
| **Videos** | Uniquement les vidéos générées par IA |

### Actions sur vos créations

Cliquez sur n'importe quel élément dans My Creations pour accéder à ces actions :

| Action | Description |
|--------|-------------|
| **Télécharger** | Enregistrez le fichier sur votre appareil (JPG pour les images, MP4 pour les vidéos) |
| **Créer un Post** | Ouvrez l'éditeur de post avec ce média joint |
| **Recréer** | Chargez le même modèle et les mêmes paramètres pour générer une nouvelle variante |
| **Utiliser le Prompt** | Copiez le prompt original dans le champ de création |
| **Voir les Détails** | Consultez les métadonnées complètes de génération — modèle, prompt, paramètres, résolution |
| **Éditer l'Image** | Ouvrez l'image dans l'éditeur intégré pour des modifications (images uniquement) |
| **Animer** | Transformez une image fixe en vidéo avec le mode Image à Vidéo (images uniquement) |
| **Publier** | Partagez dans la galerie publique Explore pour que la communauté la découvre |
| **Supprimer** | Supprimez définitivement le média (avec confirmation) |

### Générations en cours

Quand une génération est en cours, elle apparaît dans My Creations avec un indicateur de chargement. Vous pouvez :

- Voir le statut de progression (en file d'attente, en traitement)
- Continuer à travailler sur d'autres tâches en attendant
- Fermer Zee Studio et revenir plus tard — l'élément sera prêt à votre retour

## My Media

L'onglet **My Media** affiche les fichiers que vous avez uploadés manuellement. Utilisez-le pour vos propres photos, assets de marque, logos ou tout média que vous souhaitez utiliser dans des posts.

### Comment uploader

- **Glisser-déposer** — glissez des fichiers directement sur la zone de la galerie
- **Cliquer pour parcourir** — cliquez sur la zone d'upload pour ouvrir votre sélecteur de fichiers
- **Bouton d'upload** — utilisez le bouton d'upload dans la barre d'outils de la galerie

Voir [Uploader des Médias](/zee-studio/upload) pour les formats supportés et les limites de taille.

### Actions sur les médias uploadés

| Action | Description |
|--------|-------------|
| **Télécharger** | Enregistrez le fichier sur votre appareil |
| **Créer un Post** | Joignez le média à un post sur les réseaux sociaux |
| **Animer** | Générez une vidéo à partir d'une image fixe (Image à Vidéo) |
| **Supprimer** | Supprimez le fichier uploadé |

## Comportement de la galerie

- Les médias sont affichés dans une grille responsive qui s'adapte à la taille de votre écran
- Les éléments sont triés par date de création, les plus récents en premier
- La galerie charge plus d'éléments automatiquement au défilement (défilement infini)
- Les miniatures sont optimisées pour un chargement rapide
- Les vidéos affichent une icône de lecture sur leur miniature
