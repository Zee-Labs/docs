Générez des images IA professionnelles directement dans Zee Studio. Choisissez parmi 16 modèles optimisés pour différents styles — photoréalisme, logos, art vectoriel, illustrations, et plus encore.

## Comment générer une image

1. Ouvrez **Zee Studio** depuis la barre de navigation supérieure (cliquez sur **Zee Studio**)
2. Sélectionnez le mode **AI Image** dans le panneau de création
3. Choisissez votre mode de génération :
   - **Texte à Image** — décrivez ce que vous souhaitez dans le champ de prompt
   - **Image à Image** — joignez une image de référence et décrivez les modifications souhaitées
4. Tapez votre prompt (ex. : "Un portrait professionnel d'une femme confiante dans un bureau moderne")
5. *(Optionnel)* Cliquez sur le bouton **Améliorer avec l'IA** (✨) pour optimiser votre prompt — cela améliore considérablement les résultats
6. Cliquez sur la puce du modèle pour sélectionner un modèle IA et ajuster les paramètres
7. Vérifiez le coût en crédits affiché sur le bouton **Générer**
8. Cliquez sur **Générer**
9. Attendez 10 à 30 secondes — votre image apparaît dans **Creations** quand elle est prête

## Paramètres du modèle

Cliquez sur la puce du modèle (ex. : **Nano Banana Pro**) pour ouvrir le panneau de paramètres :

- **Sélecteur de modèle** — parcourez et changez de modèle disponible
- **Proportion** — choisissez entre paysage (16:9), portrait (9:16), carré (1:1) et d'autres formats
- **Qualité/Vitesse** — certains modèles offrent des niveaux de qualité (ex. : GPT Image : Low, Medium, High)
- **Style** — certains modèles supportent des préréglages de style (ex. : Ideogram : realistic, design, 3D, anime)
- **Prompt Négatif** — décrivez ce que vous ne souhaitez pas dans l'image (supporté par certains modèles)

## Utilisation d'images de référence

En mode **Image à Image** :

1. Cliquez sur l'icône image dans le panneau de création pour joindre une image de référence
2. Le modèle IA utilisera votre image comme point de départ
3. Décrivez les modifications souhaitées dans le prompt (ex. : "Transformez ceci en peinture aquarelle")

<Note>
Tous les modèles ne supportent pas les images de référence. Vérifiez le panneau de paramètres du modèle — les modèles avec support de référence affichent une option de pièce jointe image.
</Note>

## Générer plusieurs images à la fois

Besoin de variations du même prompt ? Générez jusqu'à **4 images à la fois** en un seul clic.

1. Dans le panneau de création, définissez la **quantité** sur 2, 3 ou 4
2. Cliquez sur **Générer**
3. Toutes les images apparaissent dans votre galerie en un seul lot

Chaque image compte comme une génération — vos crédits sont déduits en conséquence. Générer 4 images coûte 4× le coût en crédits d'une image.

<Note>
La génération par lot est idéale pour choisir la meilleure variation. L'IA génère chaque image indépendamment avec le même prompt, vous obtiendrez donc différentes interprétations de votre idée.
</Note>

## Conseils pour de meilleurs résultats

- **Soyez précis** — "Un chiot golden retriever sur une plage au coucher du soleil" fonctionne mieux que "chien sur la plage"
- **Utilisez l'Optimiseur de Prompt** — il ajoute automatiquement des détails de photographie professionnelle (éclairage, caméra, composition)
- **Mentionnez le style** — "photographie éditoriale Vogue", "plan cinématographique", "design plat minimaliste"
- **Spécifiez la qualité** — "ultra-photoréaliste 8K", "éclairage studio professionnel"
- **Essayez différents modèles** — chaque modèle a ses forces. Nano Banana excelle pour les photos, Recraft pour le design, Ideogram pour le texte

## Après la génération

Une fois votre image prête, cliquez dessus pour :

- **Télécharger** — enregistrez sur votre appareil
- **Créer un Post** — ouvrez l'éditeur de post avec l'image jointe
- **Recréer** — réutilisez le même modèle et les mêmes paramètres avec un nouveau prompt
- **Utiliser le prompt** — copiez le prompt dans le champ de création
- **Animer** — transformez l'image en vidéo (mode Image à Vidéo)
- **Publier** — partagez dans la galerie publique Explore
