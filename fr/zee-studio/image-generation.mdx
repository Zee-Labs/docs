Générez des images professionnelles par IA directement dans Zee Studio. Choisissez parmi plus de 20 modèles optimisés pour différents styles — photoréalisme, logos, art vectoriel, illustrations et plus encore.

## Comment générer une image

1. Ouvrez **Zee Studio** depuis la barre latérale
2. Sélectionnez le mode **Image IA** dans le panneau de création
3. Choisissez votre mode de génération :
   - **Texte vers image** — décrivez ce que vous voulez dans le champ de prompt
   - **Image vers image** — joignez une image de référence et décrivez les changements souhaités
4. Saisissez votre prompt (par exemple, « Un portrait professionnel d'une femme assurée dans un bureau moderne »)
5. *(Optionnel)* Cliquez sur le bouton **Améliorer avec l'IA** (✨) pour optimiser votre prompt — cela améliore considérablement les résultats
6. Cliquez sur la puce du modèle pour sélectionner un modèle d'IA et ajuster les paramètres
7. Vérifiez le coût en crédits affiché sur le bouton **Générer**
8. Cliquez sur **Générer**
9. Patientez 10 à 30 secondes — votre image apparaît dans **Mes Créations** une fois prête

## Choisir le bon modèle

| Modèle | Idéal pour | Coût en crédits |
|-------|----------|-------------|
| **Nano Banana Pro** | Portraits et photos photoréalistes | ~160 crédits |
| **FLUX 1.1 Ultra** | Photoréalisme haute résolution (2K) | ~120 crédits |
| **Recraft V4** | Logos, icônes, illustrations vectorielles, design | ~80 crédits |
| **Ideogram V3** | Texte dans des images (affiches, bannières, visuels sociaux) | ~60-180 crédits |
| **GPT Image 2** | Images créatives avec narration | ~40-440 crédits |
| **GPT Image 1.5** | Équilibre entre qualité et style | ~60-126 crédits |

<Note>
Les coûts en crédits varient selon le modèle, la résolution et les paramètres de qualité. Le coût exact est toujours affiché sur le bouton **Générer** avant la confirmation.
</Note>

Consultez la liste complète dans [Modèles d'image](/ai-models/image-models).

## Paramètres du modèle

Cliquez sur la puce du modèle (par exemple, **Nano Banana Pro**) pour ouvrir le panneau des paramètres :

- **Sélecteur de modèle** — parcourez et basculez entre les modèles disponibles
- **Format d'image** — choisissez entre paysage (16:9), portrait (9:16), carré (1:1) et d'autres formats
- **Qualité/Vitesse** — certains modèles offrent des niveaux de qualité (par exemple, GPT Image : Faible, Moyenne, Élevée)
- **Style** — certains modèles prennent en charge des préréglages de style (par exemple, Ideogram : réaliste, design, 3D, anime)
- **Prompt négatif** — décrivez ce que vous ne voulez pas dans l'image (pris en charge par certains modèles)

## Utiliser des images de référence

En mode **Image vers image** :

1. Cliquez sur l'icône d'image dans le panneau de création pour joindre une image de référence
2. Le modèle d'IA utilisera votre image comme point de départ
3. Décrivez les changements souhaités dans le prompt (par exemple, « Transformer ceci en peinture aquarelle »)

<Note>
Tous les modèles ne prennent pas en charge les images de référence. Consultez le panneau des paramètres du modèle — les modèles compatibles affichent une option de pièce jointe d'image.
</Note>

## Conseils pour de meilleurs résultats

- **Soyez précis** — « Un chiot golden retriever sur une plage au coucher du soleil » fonctionne mieux que « un chien sur la plage »
- **Utilisez l'Optimiseur de prompts** — il ajoute automatiquement des détails photographiques professionnels (éclairage, caméra, composition)
- **Mentionnez le style** — « photographie éditoriale Vogue », « image fixe cinématographique », « design plat minimaliste »
- **Précisez la qualité** — « ultra-photoréaliste 8K », « éclairage de studio professionnel »
- **Essayez différents modèles** — chaque modèle a ses points forts. Nano Banana excelle pour les photos, Recraft pour le design, Ideogram pour le texte

## Après la génération

Une fois votre image prête, cliquez dessus pour :

- **Télécharger** — enregistrez-la sur votre appareil
- **Créer un post** — ouvrez l'éditeur de post avec l'image attachée
- **Recréer** — réutilisez le même modèle et les mêmes paramètres avec un nouveau prompt
- **Utiliser le prompt** — copiez le prompt dans le champ de création
- **Animer** — transformez l'image en vidéo (mode Image vers vidéo)
- **Publier** — partagez-la dans la galerie publique Explorer
