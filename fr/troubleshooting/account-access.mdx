## J'ai oublie mon mot de passe

1. Allez sur la page de connexion
2. Cliquez sur **Mot de passe oublie ?**
3. Entrez votre adresse email
4. Verifiez votre boite de reception pour un lien de reinitialisation de mot de passe (verifiez aussi le dossier spam/courrier indesirable)
5. Cliquez sur le lien et definissez un nouveau mot de passe

<Note>
Le lien de reinitialisation expire au bout de **20 minutes**. S'il a expire, demandez-en un nouveau.
</Note>

<Warning>
La reinitialisation du mot de passe ne fonctionne que pour les comptes crees avec email/mot de passe. Si vous vous etes inscrit avec **Google**, utilisez le bouton "Se connecter avec Google" a la place — il n'y a pas de mot de passe a reinitialiser.
</Warning>

## Je n'arrive pas a me connecter

**Verifiez ces problemes courants :**

| Probleme | Solution |
|----------|----------|
| Mauvais mot de passe | Utilisez le lien "Mot de passe oublie ?" pour reinitialiser |
| Mauvais email | Assurez-vous d'utiliser le meme email avec lequel vous vous etes inscrit |
| Inscription via Google | Cliquez sur "Se connecter avec Google" au lieu d'entrer un mot de passe |
| Compte non active | Verifiez votre email pour le lien d'activation (envoye lors de l'inscription) |
| Compte bloque | Votre paiement a peut-etre echoue — verifiez votre email pour les notifications de paiement |

## Je me suis inscrit avec Google mais je veux definir un mot de passe

Si vous vous etes inscrit avec Google et souhaitez egalement avoir un mot de passe :

1. Connectez-vous avec Google
2. Allez dans **Parametres** → **Compte**
3. Dans la section mot de passe, definissez un nouveau mot de passe

Apres cela, vous pouvez vous connecter avec Google ou avec votre email/mot de passe.

## Je veux changer mon email

Actuellement, votre adresse email ne peut pas etre modifiee apres la creation du compte. Si vous devez utiliser un email different :

1. Creez un nouveau compte avec l'email souhaite
2. Contactez le support pour transferer votre abonnement et vos donnees

## Je veux supprimer mon compte

Pour supprimer votre compte et toutes les donnees associees, contactez le support via le widget de chat. Incluez :
- L'email de votre compte
- La confirmation que vous souhaitez la suppression de toutes les donnees

Nous traiterons votre demande conformement aux exigences LGPD/GDPR.

## Je n'arrive pas a acceder a mon equipe

**Si vous avez ete invite dans une equipe :**
1. Verifiez votre email pour le lien d'invitation
2. Cliquez sur le lien pour accepter l'invitation
3. Si le lien a expire, demandez a l'administrateur de l'equipe de le renvoyer

**Si vous etes l'administrateur :**
1. Allez dans **Parametres** → **Equipe**
2. Consultez les membres de votre equipe
3. Envoyez de nouvelles invitations si necessaire

## Mon compte affiche "Activation requise"

Apres l'inscription, vous devez activer votre compte :

1. Verifiez votre boite de reception (et le dossier spam) pour l'email d'activation
2. Cliquez sur le lien d'activation
3. Si l'email n'est pas arrive, allez sur la page de connexion et demandez un nouvel email d'activation

<Note>
Vous pouvez demander un nouvel email d'activation jusqu'a 3 fois. Attendez au moins 60 secondes entre chaque demande.
</Note>

## Vous avez toujours des problemes ?

Contactez le support via le widget de chat avec :
- L'email de votre compte
- Une description du probleme
- Tout message d'erreur que vous voyez sur la page de connexion
