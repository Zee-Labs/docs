## Le telechargement de fichier echoue

**Verifiez ces exigences :**

| Type de fichier | Taille max. | Formats supportes |
|----------------|-------------|-------------------|
| Images | 10 Mo | JPG, PNG, WebP, GIF |
| Videos | 1 Go | MP4 uniquement |

**Corrections courantes :**
- **Mauvais format :** Convertissez votre video en MP4 avant de telecharger. Les formats MOV, AVI et MKV ne sont pas supportes.
- **Fichier trop volumineux :** Compressez votre image ou video avant de telecharger.
- **Stockage plein :** Verifiez votre utilisation du stockage dans le menu du compte. Supprimez d'anciens medias ou mettez a niveau votre plan pour plus d'espace.

## Le stockage est plein

Lorsque votre stockage atteint la limite, vous ne pouvez plus telecharger de nouveaux fichiers ni generer de contenu IA.

**Comment liberer de l'espace :**
1. Allez dans **Zee Studio** → onglet **Fichiers**
2. Supprimez les medias dont vous n'avez plus besoin
3. L'espace de stockage est libere immediatement apres la suppression

**Limites de stockage par plan :**

| Plan | Stockage |
|------|----------|
| Gratuit | 10 Go |
| Standard | 15 Go |
| Team | 30 Go |
| Pro | 50 Go |
| Ultimate | 100 Go |

Pour obtenir plus de stockage, mettez a niveau votre plan dans **Facturation** depuis le menu lateral.

## La page charge indefiniment

**Essayez ces etapes dans l'ordre :**

1. **Rafraichissement force :** Appuyez sur `Ctrl+Shift+R` (Windows) ou `Cmd+Shift+R` (Mac)
2. **Videz le cache du navigateur :** Allez dans les parametres du navigateur → Effacer les donnees de navigation → Images et fichiers en cache
3. **Essayez un autre navigateur :** Si le probleme persiste, essayez Chrome, Firefox ou Edge
4. **Verifiez votre connexion internet :** Assurez-vous que votre connexion est stable
5. **Desactivez les extensions du navigateur :** Les bloqueurs de publicites ou les extensions de confidentialite peuvent parfois interferer

Si le probleme persiste apres toutes ces etapes, il peut s'agir d'un probleme temporaire de serveur. Attendez quelques minutes et reessayez.

## Le calendrier n'affiche pas mes posts

**Causes courantes :**

- **Mauvais mois/semaine :** Naviguez vers la bonne date en utilisant les controles du calendrier
- **Filtres actifs :** Verifiez si des filtres sont appliques (plateforme, statut) et effacez-les
- **Canal non selectionne :** Assurez-vous que le canal concerne est selectionne dans le filtre de la barre laterale

## Les notifications n'apparaissent pas

- **Verifiez l'icone de cloche** dans la barre de navigation superieure — le badge indique le nombre de notifications non lues
- **Ouvrez le panneau de notifications** en cliquant sur la cloche
- Si le panneau est vide, fermez-le et rouvrez-le — les notifications se rafraichissent a l'ouverture du panneau
- Assurez-vous que les emails de notification sont actives dans **Parametres** → **Compte** → **Preferences email**

## Les analyses affichent zero donnee

**Causes courantes :**

- **Nouveau canal :** Les donnees d'analyse mettent 24 a 48 heures a apparaitre apres la connexion d'un canal
- **Aucun post publie :** Les analyses n'affichent des donnees que pour les posts publies
- **Delai de la plateforme :** Certaines plateformes (notamment TikTok et LinkedIn) mettent plus de temps a fournir les donnees d'analyse

## Je ne vois pas une fonctionnalite mentionnee dans la documentation

Certaines fonctionnalites ne sont disponibles que sur des plans specifiques :

| Fonctionnalite | Plan requis |
|----------------|-------------|
| Suite reseaux sociaux (canaux, calendrier, publication) | Standard+ |
| Generation d'images/videos par IA | Tous les plans (le plan Gratuit necessite l'achat de credits) |
| Membres d'equipe | Team+ |
| Acces API | Standard+ |
| Recharge automatique | Standard+ |

Si vous etes sur le plan Gratuit, vous pouvez acceder aux fonctionnalites IA en achetant des packs de credits, mais vous ne pouvez pas connecter de canaux ni programmer de posts.

## Le theme sombre/clair ne s'affiche pas correctement

1. Cliquez sur votre **avatar** en haut a droite
2. Cliquez sur l'icone des **parametres d'apparence** (palette de peinture)
3. Choisissez entre le mode **Clair**, **Sombre** ou **Systeme**

Si le theme ne s'applique pas immediatement, essayez de rafraichir la page.

## J'ai un probleme non liste ici

Contactez le support via le widget de chat (disponible sur ordinateur). Lorsque vous contactez le support, incluez :
- Ce que vous essayiez de faire
- Ce qui s'est passe a la place
- Tout message d'erreur que vous voyez
- Votre navigateur et appareil (par ex., Chrome sur Mac)

Notre equipe de support repond generalement en quelques heures pendant les heures ouvrables.
