Les credits IA sont la monnaie utilisee pour generer des images et des videos dans Postzee. Chaque generation IA coute un nombre specifique de credits selon le modele, la resolution et les parametres.

## Comment fonctionnent les credits

**1 dollar = 1 000 credits**

Lorsque vous generez une image ou une video, le cout en credits est deduit de votre solde. Le cout exact est toujours affiche sur le bouton **Generer** avant confirmation.

## Types de credits

Il existe deux types de credits dans Postzee — ils fonctionnent differemment :

| Type | Source | Expiration | En cas d'annulation |
|------|--------|------------|---------------------|
| **Credits du plan** | Inclus mensuellement avec les plans payants | Reinitialises a chaque cycle de facturation | Retires lors de l'annulation |
| **Credits achetes** | Achetes separement sous forme de packs de credits | **N'expirent jamais** | **Conserves pour toujours** |

### Credits du plan

Les plans payants incluent une allocation mensuelle de credits :

| Plan | Credits mensuels |
|------|-----------------|
| Standard | 2 000 |
| Team | 3 000 |
| Pro | 5 000 |
| Ultimate | 10 000 |

Les credits du plan sont reinitialises au debut de chaque cycle de facturation. Les credits du plan non utilises ne sont pas reportes.

### Credits achetes

Les credits achetes via des packs de credits sont permanents :

- **N'expirent jamais** — utilisez-les quand vous le souhaitez
- **Survivent a l'annulation** — si vous annulez votre plan, les credits achetes restent sur votre compte
- **Disponibles sur tous les plans** — y compris le plan Gratuit

## Verifier votre solde

Votre solde de credits actuel est visible dans le **panneau de compte** (cliquez sur votre avatar en haut a droite). Il affiche a la fois vos credits disponibles totaux et une repartition par type.

Vous pouvez egalement verifier votre solde detaille en cliquant sur l'affichage des credits dans le panneau de compte, ce qui vous amene a la page **Credits**.

## Utilisation des credits

Chaque generation IA affiche son cout en credits avant confirmation :

- Le bouton **Generer** affiche le cout estime (par ex., "Generer · 160")
- Les couts en credits varient selon le modele — certains modeles sont plus chers que d'autres
- Des resolutions plus elevees, des durees plus longues et des parametres de qualite augmentent le cout

### Exemples de couts

| Generation | Cout approximatif |
|-----------|-------------------|
| Image avec Nano Banana (original) | ~78 credits |
| Image avec Nano Banana 2 | ~160 credits |
| Image avec Nano Banana Pro | ~300 credits |
| Image avec GPT Image Mini | ~22 credits |
| Image avec Recraft V4 (raster) | ~80 credits |
| Image avec FLUX 1.1 Ultra | ~120 credits |
| Image avec GPT Image 2 (High) | ~440 credits |
| Video avec Sora 2 (10s) | ~300 credits |
| Video avec Veo 3.1 Fast (8s, sans audio) | ~800 credits |
| Video avec Veo 3.1 (8s, sans audio) | ~1 600 credits |
| Optimiseur de Prompts | **Gratuit** (0 credit) |

<Note>
Les couts exacts dependent du modele specifique, de la resolution, des parametres de qualite et de la duree. Verifiez le bouton Generer pour le cout precis avant de confirmer.
</Note>

Consultez [Tarification des modeles IA](/ai-models/pricing) pour la reference complete des couts.

## Quand les credits sont-ils deduits ?

Les credits sont deduits **au moment ou vous cliquez sur Generer**. Voici le flux complet :

1. **Vous cliquez sur Generer** — les credits sont deduits immediatement de votre solde
2. **La generation demarre** — l'IA traite votre demande
3. **La generation reussit** — votre media apparait dans la galerie, aucun autre changement de solde
4. **La generation echoue** — les credits sont **automatiquement rembourses** sur votre solde

### Remboursements pour generations echouees

Si une generation echoue pour une raison quelconque (erreur du fournisseur, violation de politique de contenu, expiration, etc.), les credits sont retournes automatiquement sur votre solde. Vous n'avez pas besoin de contacter le support.

Vous verrez une notification lorsqu'un remboursement se produit, et votre solde se met a jour en temps reel.

<Note>
Les remboursements ne se produisent que pour les echecs techniques. Les generations que vous annulez manuellement en cours de processus ou qui se terminent avec succes (meme si vous n'aimez pas le resultat) ne sont pas remboursees — c'est pourquoi nous recommandons d'utiliser l'**Optimiseur de Prompts** avant de generer pour maximiser la qualite.
</Note>

## Ce qui ne coute pas de credits

Ces fonctionnalites sont toujours gratuites :

- **Optimiseur de Prompts** — ameliorez vos prompts sans frais
- **Entree vocale** — dictez vos prompts a la voix
- **Explorer la galerie** — parcourez la galerie communautaire
- **Telecharger des medias** — telechargez vos propres fichiers dans Mes Medias
- **Programmer et publier des posts** — la suite reseaux sociaux utilise votre plan, pas des credits
