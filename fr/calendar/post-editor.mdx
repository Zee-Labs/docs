L'Éditeur de Post est l'endroit où vous composez, personnalisez et publiez votre contenu pour les réseaux sociaux. Rédigez une fois, personnalisez par plateforme et publiez sur plusieurs canaux simultanément.

## Ouvrir l'Éditeur de Post

Vous pouvez ouvrir l'Éditeur de Post depuis plusieurs endroits :

- **Calendrier** — cliquez sur une date ou un post existant
- **Zee Studio** — sélectionnez le mode **Create Post**, ou cliquez sur **Créer un Post** sur une image/vidéo générée

## Rédiger votre post

### Champ de contenu

L'éditeur de texte principal supporte :

- **Texte enrichi** avec formatage adapté à la plateforme
- **Jusqu'à 2 200 caractères** — un compteur affiche les caractères restants
- **Support des emojis** — utilisez le sélecteur d'emojis ou tapez des emojis directement
- **Suggestions de hashtags** — hashtags populaires par catégorie (marketing, business, technologie, lifestyle)

<Note>
Les limites de caractères varient selon la plateforme. Instagram autorise 2 200 caractères, X (Twitter) autorise 280, LinkedIn autorise 3 000. Postzee vous avertira si votre contenu dépasse la limite d'une plateforme.
</Note>

### Joindre des médias

Ajoutez des images et vidéos à votre post :

- **Depuis Zee Studio** — cliquez sur **Créer un Post** sur n'importe quelle image ou vidéo générée
- **Depuis My Media** — sélectionnez des fichiers uploadés depuis votre bibliothèque
- **Upload direct** — glissez-déposez ou cliquez pour uploader de nouveaux fichiers

Les types de médias supportés dépendent de la plateforme :

| Plateforme | Image unique | Images multiples | Vidéo | Carrousel |
|------------|:---:|:---:|:---:|:---:|
| Instagram | Oui | Oui | Oui (Reels) | Oui |
| Facebook | Oui | Oui | Oui | — |
| X (Twitter) | Oui | Oui (jusqu'à 4) | Oui | — |
| LinkedIn | Oui | Oui | Oui | Oui |
| TikTok | — | — | Oui | — |
| YouTube | — | — | Oui | — |
| Pinterest | Oui | — | — | — |
| Threads | Oui | Oui | — | — |

## Sélectionner les canaux

Choisissez quels canaux reçoivent ce post :

1. Dans le sélecteur de canaux, cochez les canaux sur lesquels vous souhaitez publier
2. Les canaux sélectionnés affichent une coche et le compteur se met à jour (ex. : "Canaux (3)")
3. Chaque canal affiche son avatar, son nom et sa plateforme

Vous pouvez sélectionner des canaux de différentes plateformes — Postzee gère automatiquement les différences de format.

<Note>
Seuls les canaux activés apparaissent dans le sélecteur. Si un canal nécessite une reconnexion, il s'affichera comme désactivé. Allez dans **Canaux** pour corriger la connexion.
</Note>

## Paramètres spécifiques à la plateforme

Certaines plateformes requièrent ou supportent des paramètres supplémentaires. Ceux-ci apparaissent quand vous sélectionnez un canal de cette plateforme :

### TikTok

| Paramètre | Options |
|-----------|---------|
| **Titre** | Jusqu'à 90 caractères |
| **Confidentialité** | Public, Amis mutuels, Abonnés uniquement, Privé |
| **Autoriser les Duos** | Activé / Désactivé |
| **Autoriser les Stitches** | Activé / Désactivé |
| **Autoriser les Commentaires** | Activé / Désactivé |
| **Ajout automatique de musique** | Oui / Non |
| **Contenu de marque** | Basculer pour la divulgation de contenu sponsorisé/de marque |
| **Créé avec l'IA** | Basculer pour la divulgation de contenu généré par IA |

### YouTube

| Paramètre | Options |
|-----------|---------|
| **Titre** | 2-100 caractères (obligatoire) |
| **Visibilité** | Public, Privé, Non répertorié |
| **Miniature** | Uploader une image de miniature personnalisée |
| **Tags** | Ajouter des tags recherchables |

### Instagram

| Paramètre | Options |
|-----------|---------|
| **Type de post** | Post dans le fil ou Story |
| **Collaborateurs** | Taguer des comptes collaborateurs |

### LinkedIn

| Paramètre | Options |
|-----------|---------|
| **Carrousel** | Basculer pour afficher plusieurs images en carrousel déroulant |

## Tags

Organisez vos posts avec des tags pour un filtrage facile :

1. Dans l'éditeur de post, cliquez sur le champ **Tags**
2. Sélectionnez des tags existants ou créez-en de nouveaux
3. Les tags apparaissent sous forme de puces colorées sur votre post
4. Utilisez les tags pour filtrer les posts dans la vue Calendrier

Les tags sont partagés au sein de votre équipe — tout le monde peut créer, modifier ou filtrer par tags.

## Liens courts

Postzee peut automatiquement raccourcir les URLs dans vos posts pour un formatage plus propre et un suivi des clics :

- Activez **Liens courts** dans l'éditeur de post
- Les liens raccourcis sont suivis — consultez les statistiques de clics dans les détails du post

## Contenu depuis Zee Studio

La façon la plus rapide de créer des posts avec du contenu IA :

1. Générez une image ou vidéo dans **Zee Studio**
2. Cliquez sur **Créer un Post** sur le média généré
3. L'éditeur de post s'ouvre avec le média déjà joint
4. Rédigez votre légende, sélectionnez les canaux et programmez

Vous pouvez aussi utiliser le mode **Create Post** directement dans Zee Studio pour rédiger votre post aux côtés des outils de génération IA.
