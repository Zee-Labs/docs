Le **Calendrier** est votre centre de commande pour le contenu des réseaux sociaux. Visualisez tous vos posts d'un coup d'oeil, glissez pour reprogrammer, filtrez par canal ou statut, et gérez l'ensemble de votre pipeline de contenu depuis une seule vue.

## Comment y accéder

Cliquez sur **Calendrier** dans la barre de navigation supérieure. Vos posts sont affichés sur un calendrier visuel avec des statuts codés par couleur.

## Vues du calendrier

Le calendrier supporte quatre vues, sélectionnables depuis la barre d'outils :

| Vue | Ce qu'elle affiche |
|-----|-------------------|
| **Mois** | Grille mensuelle complète avec jusqu'à 3 posts par jour (par défaut sur desktop) |
| **Semaine** | Vue sur 7 jours avec créneaux horaires |
| **Jour** | Journée unique avec décomposition horaire détaillée |
| **Agenda** | Liste chronologique des posts à venir (par défaut sur mobile) |

<Note>
Sur les appareils mobiles, le calendrier passe automatiquement en vue **Agenda** pour une lecture plus facile. Sur desktop, il affiche par défaut la vue **Mois**.
</Note>

## Couleurs des statuts de post

Chaque post sur le calendrier est codé par couleur selon son statut actuel :

| Couleur | Statut | Signification |
|---------|--------|--------------|
| **Gris** | Brouillon | Enregistré mais non programmé |
| **Bleu** | Programmé | En file d'attente pour publication à une date/heure spécifique |
| **Violet** | Publié | Publié avec succès sur la plateforme |
| **Rouge** | Erreur | Échec de publication — action requise |

## Interagir avec le calendrier

### Voir un post

Cliquez sur n'importe quel post dans le calendrier pour ouvrir les détails du post. De là, vous pouvez voir le contenu, modifier, reprogrammer ou supprimer.

### Glisser pour reprogrammer

Glissez un post programmé d'une date/heure à une autre pour le reprogrammer instantanément. Cela fonctionne dans les vues Mois, Semaine et Jour.

<Note>
Seuls les posts **Brouillon** et **Programmé** peuvent être reprogrammés par glissement. Les posts Publiés et en Erreur sont verrouillés.
</Note>

### Créer un nouveau post

Cliquez sur n'importe quelle date dans le calendrier pour ouvrir l'éditeur de post avec cette date pré-sélectionnée. Ou utilisez le bouton **Créer un Post** dans la barre d'outils.

### Naviguer dans les dates

- Utilisez les **flèches gauche/droite** dans la barre d'outils pour naviguer entre les mois, semaines ou jours
- Cliquez sur **Aujourd'hui** pour revenir à la date actuelle

## Filtrage

Utilisez les contrôles de filtre au-dessus du calendrier pour affiner votre vue :

| Filtre | Options |
|--------|---------|
| **Canal** | Afficher les posts pour des canaux spécifiques uniquement |
| **Statut** | Filtrer par Brouillon, Programmé, Publié ou Erreur |
| **Tags** | Filtrer par les tags de contenu que vous avez attribués |

Les filtres actifs s'affichent sous forme de puces que vous pouvez effacer individuellement, ou utilisez **Réinitialiser les filtres** pour tout supprimer.

## Groupes de posts récurrents

Si vous avez des posts récurrents (posts qui se répètent à un intervalle défini), ils apparaissent avec des indicateurs de couleur correspondants sur le calendrier. Une légende sous le calendrier affiche :

- L'indicateur de couleur pour chaque groupe récurrent
- Le titre du groupe et le nombre de posts
- La plage de dates de la série récurrente

Survolez un groupe dans la légende pour mettre en surbrillance tous ses posts sur le calendrier. Cliquez pour naviguer vers un groupe spécifique.

## Fonctionnalités clés

- **La semaine commence le lundi** — la grille du calendrier commence toujours le lundi
- **Sensible au fuseau horaire** — tous les horaires sont affichés dans le fuseau horaire configuré de votre équipe
- **Responsive** — s'adapte au desktop, à la tablette et au mobile
- **Multilingue** — les libellés et dates du calendrier respectent votre paramètre de langue (anglais, portugais, espagnol, français)
