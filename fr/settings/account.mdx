Gerez votre profil personnel, votre mot de passe et vos preferences de langue depuis les parametres du compte.

## Comment y acceder

Cliquez sur votre **avatar** en haut a droite pour ouvrir le panneau de compte, puis cliquez sur **Parametres**. La page des parametres du compte s'ouvre avec des onglets pour chaque section.

## Informations du profil

Mettez a jour vos informations personnelles :

| Champ | Description |
|-------|-------------|
| **Prenom** | Votre prenom (obligatoire, minimum 2 caracteres) |
| **Nom de famille** | Votre nom de famille (optionnel) |
| **Bio** | Une courte description a votre sujet (optionnel) |
| **Avatar** | Telechargez une photo de profil |
| **Telephone** | Votre numero de telephone (optionnel) |

### Informations d'adresse (optionnel)

| Champ | Description |
|-------|-------------|
| **Pays** | Votre pays |
| **Etat** | Etat ou region |
| **Ville** | Votre ville |
| **Adresse** | Adresse postale |
| **Code postal** | Code postal |

## Changer votre mot de passe

1. Allez dans **Parametres** → **Mon Compte**
2. Trouvez la section **Changer le mot de passe**
3. Entrez votre mot de passe actuel (si vous en avez un defini)
4. Entrez votre nouveau mot de passe (minimum 6 caracteres)
5. Confirmez et enregistrez

<Note>
Si vous vous etes inscrit avec une connexion sociale (Google, GitHub) et n'avez pas de mot de passe defini, vous pouvez en ajouter un sans entrer de mot de passe actuel.
</Note>

## Langue

Changez la langue de l'interface :

1. Ouvrez le **panneau de compte** (cliquez sur votre avatar)
2. Selectionnez votre langue preferee dans le selecteur de **Langue**
3. L'interface se met a jour immediatement

Langues prises en charge :

| Langue | Code |
|--------|------|
| **Anglais** | en |
| **Portugais (Bresil)** | pt-BR |
| **Espagnol** | es |
| **Francais** | fr |

Le parametre de langue affecte l'ensemble de l'interface Postzee, y compris les menus, boutons, notifications et e-mails.

## Supprimer votre compte

Pour demander la suppression de votre compte :

1. Envoyez un e-mail a **support@postzee.app** depuis l'adresse e-mail enregistree sur votre compte
2. Incluez "Demande de suppression de compte" dans l'objet
3. Notre equipe confirmera la demande et traitera la suppression

### Que se passe-t-il lorsque vous supprimez votre compte

- **Facturation** — votre abonnement est annule immediatement. Vous ne serez plus facture.
- **Donnees** — toutes vos donnees personnelles (profil, posts, medias generes, analytiques, canaux) sont supprimees definitivement dans un delai de 30 jours
- **Posts publies** — le contenu deja publie sur vos plateformes de reseaux sociaux y reste. Pour le supprimer, supprimez-le directement sur chaque plateforme.
- **Appartenance a une equipe** — si vous etes membre d'autres equipes, vous en etes retire
- **Equipe que vous possedez** — si vous possedez une equipe avec d'autres membres, contactez-nous d'abord pour organiser un transfert de propriete

<Note>
La suppression de compte est definitive et ne peut pas etre annulee. Si vous souhaitez seulement faire une pause, envisagez plutot d'annuler votre abonnement — votre compte reste actif et vous pouvez reprendre a tout moment.
</Note>

Pour les demandes d'export de donnees (RGPD, LGPD), contactez **support@postzee.app** avant d'initier la suppression.
