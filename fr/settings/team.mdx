Invitez des membres d'equipe pour collaborer sur votre contenu de reseaux sociaux. Attribuez des roles pour controler ce que chaque personne peut acceder et gerer.

## Comment y acceder

Ouvrez le panneau de compte (cliquez sur votre avatar en haut a droite) → **Parametres** → onglet **Equipe**.

## Plans avec support d'equipe

| Plan | Membres d'equipe |
|------|-----------------|
| **Gratuit** | — |
| **Standard** | — |
| **Team** | Illimites |
| **Pro** | Illimites |
| **Ultimate** | Illimites |

<Note>
Les membres d'equipe sont disponibles sur les plans **Team**, **Pro** et **Ultimate**. Si vous etes sur Standard ou Gratuit, passez a Team ou superieur pour inviter des collaborateurs.
</Note>

## Inviter des membres d'equipe

1. Allez dans **Parametres** → **Equipe**
2. Cliquez sur **Inviter un membre**
3. Entrez l'**adresse e-mail** de la personne
4. Selectionnez un **role** (Utilisateur ou Administrateur)
5. Choisissez si vous souhaitez **envoyer une invitation par e-mail**
6. Cliquez sur **Inviter**

La personne invitee recoit un e-mail avec un lien pour rejoindre votre equipe. Vous pouvez egalement copier le lien d'invitation et le partager directement.

## Roles

Chaque membre d'equipe a un role qui controle ses permissions :

| Role | Permissions |
|------|------------|
| **Utilisateur** | Creer des posts, voir le calendrier, utiliser la generation IA, voir les analytiques |
| **Administrateur** | Tout ce qu'un Utilisateur peut faire, plus : gerer les canaux, gerer les membres d'equipe, acceder a la facturation, gerer les parametres |
| **Super Administrateur** | Tout ce qu'un Administrateur peut faire, plus : supprimer l'equipe, gerer les cles API, changer le nom de l'equipe |

<Note>
Le createur du compte est automatiquement le **Super Administrateur**. Ce role ne peut pas etre retire ou transfere.
</Note>

### Ce que chaque role peut faire

| Action | Utilisateur | Administrateur | Super Administrateur |
|--------|:-----------:|:--------------:|:--------------------:|
| Creer et programmer des posts | Oui | Oui | Oui |
| Generer des images/videos IA | Oui | Oui | Oui |
| Voir les analytiques | Oui | Oui | Oui |
| Connecter/gerer les canaux | — | Oui | Oui |
| Inviter/supprimer des membres d'equipe | — | Oui | Oui |
| Acceder a la facturation | — | Oui | Oui |
| Gerer les cles API | — | — | Oui |
| Changer le nom de l'equipe | — | — | Oui |

## Gerer les membres d'equipe

### Changer le role d'un membre

1. Trouvez le membre dans la liste d'equipe
2. Cliquez sur le menu deroulant du role a cote de son nom
3. Selectionnez le nouveau role
4. Le changement prend effet immediatement

### Supprimer un membre

1. Trouvez le membre dans la liste d'equipe
2. Cliquez sur le bouton **Supprimer**
3. Confirmez la suppression
4. Le membre perd immediatement l'acces a l'equipe

<Note>
Les membres avec un role superieur peuvent gerer les membres avec un role inferieur. Un Utilisateur ne peut pas supprimer un Administrateur, mais un Administrateur peut supprimer un Utilisateur.
</Note>

## Basculer entre les equipes

Si vous appartenez a plusieurs equipes (par ex., votre propre equipe et une equipe a laquelle vous avez ete invite), vous pouvez basculer entre elles :

1. Cliquez sur votre **avatar** dans le coin superieur droit pour ouvrir le panneau de compte
2. Cliquez sur le **selecteur d'equipe** en haut du panneau (affiche le nom de l'equipe actuelle)
3. Choisissez une autre equipe dans la liste
4. Le tableau de bord se recharge avec les canaux, posts et analytiques de l'equipe selectionnee

Chaque equipe possede ses propres :
- Canaux connectes
- Posts et calendrier
- Solde de credits IA
- Membres et roles
- Facturation

Votre compte (e-mail, nom, langue) reste le meme dans toutes les equipes.

## Ressources partagees

Tous les membres de l'equipe partagent :

- **Canaux** — tout le monde peut publier sur les canaux connectes de l'equipe (selon leur role)
- **Calendrier** — tous les posts de tous les membres apparaissent sur le meme calendrier
- **Credits IA** — le solde de credits de l'equipe est partage entre tous les membres
- **Mediatheque** — les medias generes et telecharges sont accessibles a l'equipe
- **Stockage** — la limite de stockage du plan est partagee
