Configurez le fuseau horaire, le nom de l'organisation et d'autres preferences globales de votre equipe qui affectent le fonctionnement de Postzee pour tous les membres de votre equipe.

## Comment y acceder

Ouvrez le panneau de compte (cliquez sur votre avatar en haut a droite) → **Parametres** → onglet **Preferences**.

## Fuseau horaire

Le parametre de fuseau horaire affecte toutes les fonctionnalites liees au temps de votre equipe :

- **Calendrier** — les posts sont affiches a l'heure locale correcte
- **Programmation** — le selecteur de date/heure utilise ce fuseau horaire
- **Analytiques** — les donnees sont regroupees par ce fuseau horaire
- **Cycle de facturation** — les reinitalisations de credits s'alignent sur ce fuseau horaire

### Changer le fuseau horaire

1. Allez dans **Parametres** → **Preferences**
2. Trouvez le selecteur de **Fuseau horaire**
3. Choisissez votre fuseau horaire dans le menu deroulant
4. Cliquez sur **Enregistrer**

<Note>
Le parametre de fuseau horaire s'applique a toute l'equipe. Tous les membres de l'equipe voient les heures dans le fuseau horaire configure de l'equipe. Les remplacements de fuseau horaire individuels ne sont pas pris en charge — coordonnez-vous avec votre equipe lors du choix du fuseau horaire.
</Note>

## Nom de l'equipe

Mettez a jour le nom d'affichage de votre equipe :

1. Allez dans **Parametres** → **Preferences**
2. Modifiez le champ **Nom de l'equipe**
3. Cliquez sur **Enregistrer**

<Note>
Changer le nom de l'equipe necessite le role **Super Administrateur**.
</Note>

Le nom de l'equipe apparait dans :
- La barre de navigation
- Les invitations des membres d'equipe
- Les notifications par e-mail

## Langue

La langue de votre interface est configuree par utilisateur (pas par equipe) dans vos parametres de compte :

- Allez dans **Mon Compte** → selecteur de **Langue**
- Choisissez entre anglais, portugais (BR), espagnol ou francais
- Consultez [Parametres du compte](/settings/account) pour plus de details

## Apparence (Mode sombre)

Postzee prend en charge les themes clair et sombre :

- Ouvrez le panneau de compte (cliquez sur votre avatar dans le coin superieur droit)
- Cliquez sur **Apparence**
- Choisissez :
  - **Clair** — toujours utiliser le theme clair
  - **Sombre** — toujours utiliser le theme sombre
  - **Systeme** — suivre la preference de votre systeme d'exploitation

Le parametre d'apparence est par utilisateur et conserve entre les sessions.

## Notifications

Postzee affiche des notifications in-app pour les evenements importants : post publie, post echoue, generation IA terminee, canal deconnecte, membre d'equipe ajoute, et plus encore.

### Panneau de notifications

Cliquez sur l'**icone de cloche** dans le coin superieur droit de l'en-tete pour ouvrir le panneau de notifications :

- **Onglet Non lues** — uniquement les notifications que vous n'avez pas encore vues
- **Onglet Toutes** — historique complet des notifications
- **Marquer comme lu** — les notifications que vous faites defiler sont marquees comme lues automatiquement lorsque vous fermez le panneau
- **Tout marquer comme lu** — effacer toutes les notifications non lues d'un coup

### Preferences de notifications

Vous pouvez configurer quelles notifications vous recevez :

- Allez dans **Mon Compte** → onglet **Notifications**
- Activez/desactivez chaque type de notification
- Choisissez entre in-app, e-mail ou les deux

Les notifications par e-mail sont envoyees a l'adresse e-mail de votre compte.
