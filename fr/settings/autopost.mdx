title: Autopost (RSS)
icon: rss
---

Autopost vous permet d'importer automatiquement du contenu depuis des flux RSS et de le publier sur vos canaux sociaux. Connectez un blog, un podcast ou toute source compatible RSS, et Postzee créera des publications dès que de nouveaux éléments seront détectés.

## Comment y accéder

Ouvrez le tiroir du compte (cliquez sur votre avatar en haut à droite) → **Paramètres** → onglet **Autopost**.

## Qu'est-ce qu'Autopost ?

Autopost surveille les flux RSS que vous configurez et crée automatiquement des publications dans Postzee dès qu'un nouvel élément est diffusé dans le flux. Vous décidez sur quels canaux chaque flux est publié et comment les publications sont mises en forme.

Cas d'usage courants :

- **Partager automatiquement les articles de blog** sur vos canaux sociaux dès leur publication
- **Diffuser en cross-posting des épisodes de podcast** sur LinkedIn et X
- **Distribuer des actualités** issues de flux externes vers votre communauté
- **Republier votre propre contenu** d'une plateforme à une autre

## Comment configurer

1. Accédez à **Paramètres** → **Autopost**
2. Cliquez sur **Ajouter un Autopost**
3. Saisissez :
   - **Titre** — un nom convivial pour cet autopost
   - **URL RSS** — le flux à surveiller (Postzee valide l'URL avant l'enregistrement)
   - **Canaux** — choisissez **Tous les canaux** ou sélectionnez des canaux spécifiques
4. Configurez les options de publication :
   - **Publier au créneau programmé** — publiez aux horaires de publication configurés par votre équipe (ou immédiatement si désactivé)
   - **Synchroniser les dernières entrées** — importez également les éléments récents déjà présents dans le flux (par défaut : uniquement les nouveaux éléments à venir)
   - **Générer le contenu avec l'IA** — laissez l'IA rédiger la légende des réseaux sociaux à partir de l'élément du flux
   - **Ajouter une image** — joignez une image extraite de l'élément du flux
5. *(Facultatif)* **Contenu personnalisé** — fournissez un modèle si vous ne souhaitez pas de contenu généré par l'IA
6. Cliquez sur **Enregistrer**

## Filtrage par canal

Vous pouvez limiter chaque flux Autopost à des canaux spécifiques :

- **Tous les canaux** — publiez chaque nouvel élément sur chaque canal connecté
- **Canaux spécifiques** — publiez uniquement sur les canaux sélectionnés (utile pour adapter le contenu à la plateforme)

## Contenu généré par l'IA vs contenu personnalisé

Par défaut, Autopost utilise l'IA pour générer la légende des réseaux sociaux à partir de chaque élément du flux, en l'optimisant selon le style et les limites de caractères de chaque plateforme.

Si vous préférez un contrôle manuel :

- Désactivez **Générer le contenu avec l'IA**
- Fournissez votre propre modèle dans le champ **Contenu personnalisé**
- Utilisez le titre et le lien de l'élément du flux dans votre modèle

## Validation du flux

Avant l'enregistrement, cliquez sur **Tester** pour vérifier que Postzee peut lire votre flux RSS. La validation contrôle :

- Que l'URL est accessible
- Que le flux est au format RSS/Atom valide
- Que les éléments contiennent les champs requis

## Disponibilité selon le plan

Autopost est disponible avec les plans **Team**, **Pro** et **Ultimate**.

<Note>
Autopost utilise votre quota mensuel de publications. Chaque élément publié à partir d'un flux compte comme une publication. Le contenu généré par l'IA consomme des crédits — voir [Crédits](/billing/credits) pour plus de détails.
</Note>

## Désactiver ou supprimer un flux

Pour mettre en pause l'autoposting :

1. Ouvrez le flux dans **Paramètres** → **Autopost**
2. Cliquez sur **Modifier** → désactivez l'option, ou cliquez sur **Supprimer** pour le retirer définitivement

Les publications déjà dans la file d'attente sont tout de même diffusées, mais aucun nouvel élément ne sera détecté après la désactivation.
