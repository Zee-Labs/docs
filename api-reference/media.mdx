Generate AI images and videos, and upload files to your media library.

## Generate image

```
POST /public/v1/generate-image
```

Generate an AI image from a text prompt. This is an **async operation** — it returns a job ID. Use the [Jobs endpoint](/api-reference/jobs) to poll for completion.

**Rate limit:** 10 requests per minute.

### Request

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-pro",
    "prompt": "A professional headshot of a confident woman in a modern office"
  }' \
  https://api.postzee.app/public/v1/generate-image
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `model` | string | Yes | Model ID (see [Image Models](/ai-models/image-models)) |
| `prompt` | string | Yes | Image description (5+ characters) |
| `aspectRatio` | string | No | `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `4:5`, `5:4` |
| `imageUrls` | string[] | No | Reference image URLs for image-to-image (max 16) |
| `modelParams` | object | No | Model-specific parameters (e.g., `quality`, `rendering_speed`) |

### Valid model IDs

`nano-banana`, `nano-banana-2`, `nano-banana-pro`, `gpt-image-2`, `gpt-image-1.5`, `gpt-image-1`, `gpt-image-1-mini`, `flux-2-pro`, `flux-1.1-pro`, `flux-1.1-ultra`, `recraft-v3`, `recraft-v4`, `recraft-v4-pro`, `ideogram-v2`, `ideogram-v3`

### Response

```json
{
  "jobId": "task-uuid-here",
  "jobType": "task",
  "status": "processing"
}
```

## Generate video

```
POST /public/v1/generate-video-ai
```

Generate an AI video from a text prompt. This is an **async operation**.

**Rate limit:** 10 requests per minute.

### Request

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2-t2v",
    "prompt": "A golden sunset over a calm ocean with gentle waves",
    "duration": 10,
    "aspectRatio": "16:9"
  }' \
  https://api.postzee.app/public/v1/generate-video-ai
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `model` | string | Yes | Video model ID (see below) |
| `prompt` | string | Yes | Video description (5-10,000 characters) |
| `duration` | number | No | Duration in seconds (2-25) |
| `aspectRatio` | string | No | `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `4:5`, `5:4` |
| `resolution` | string | No | `360p`, `480p`, `540p`, `720p`, `1080p`, `4k` |
| `generateAudio` | boolean | No | Generate synced audio (Veo 3.1 models) |
| `negativePrompt` | string | No | What to avoid (max 2,000 characters) |
| `imageUrl` | string | No | Source image URL for image-to-video models |

### Valid video model IDs

**Sora 2:** `sora-2-t2v`, `sora-2-i2v`, `sora-2-t2v-pro`, `sora-2-i2v-pro`

**Veo 3.1:** `veo3.1`, `veo3.1/fast`, `veo3.1/image-to-video`, `veo3.1/fast/image-to-video`

**Kling:** `kling-3.0-pro`, `kling-2.6-pro`, `kling-2.5-turbo-pro`, `kling-o3-standard`, `kling-2.1-master`

**Luma Ray:** `luma-ray-2`, `luma-ray-2-flash`

**Seedance:** `seedance-2.0`, `seedance-1.0-pro`, `seedance-1.0-lite`

**Wan:** `wan-2.5`, `wan-2.1-pro`, `wan-2.1`, `wan-flf2v`

**Other:** `vidu-q1`, `pixverse-v4.5`, `minimax-video-01`, `minimax-video-01-live`, `hunyuan-video`

### Response

```json
{
  "jobId": "task-uuid-here",
  "jobType": "task",
  "status": "processing"
}
```

## Generate batch (1-4 images)

```
POST /public/v1/generate-media
```

Generate multiple AI images in a single request. Use this for batch generation instead of multiple `generate-image` calls.

**Rate limit:** 10 requests per minute.

### Request

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-pro",
    "prompt": "A professional headshot of a confident woman",
    "quantity": 4,
    "aspectRatio": "1:1"
  }' \
  https://api.postzee.app/public/v1/generate-media
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `model` | string | Yes | Image model ID (see [Image Models](/ai-models/image-models)) |
| `prompt` | string | Yes | Image description |
| `quantity` | number | No | Number of images (1-4, default: 1) |
| `aspectRatio` | string | No | `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `4:5`, `5:4` |
| `imageUrls` | string[] | No | Reference image URLs (max 5) |
| `modelParams` | object | No | Model-specific parameters |

### Response

For `quantity > 1`, returns a group job ID:

```json
{
  "jobId": "group-uuid-here",
  "jobType": "group",
  "totalImages": 4,
  "status": "processing"
}
```

For `quantity === 1`, returns a regular task:

```json
{
  "jobId": "task-uuid-here",
  "jobType": "task",
  "status": "processing"
}
```

Use the [Jobs endpoint](/api-reference/jobs) to poll for completion. For group jobs, the response includes a `medias` array with all generated images and their `order` field.

## Upload file

```
POST /public/v1/upload
```

Upload an image or video file to your media library.

### Request

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -F "file=@/path/to/image.jpg" \
  https://api.postzee.app/public/v1/upload
```

### Response

Returns a media object with `id`, `path` (URL), `type` (image/video), and `size`.

Use the `id` in the Create Post endpoint to attach uploaded media to a post.
