Check your AI credit balance and enhance prompts via the API.

## Get credit balance

```
GET /public/v1/credits
```

Returns your current credit balance.

### Request

```bash
curl -H "Authorization: YOUR_API_KEY" \
  https://api.postzee.app/public/v1/credits
```

### Response

```json
{
  "monthlyCredits": 5.0,
  "purchasedCredits": 2.0,
  "usedCredits": 0.5,
  "availableCredits": 6.5,
  "lastReset": "2026-04-01T00:00:00.000Z"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `monthlyCredits` | number | Monthly credits from your plan (in USD) |
| `purchasedCredits` | number | Purchased credits that never expire (in USD) |
| `usedCredits` | number | Credits used in the current billing cycle (in USD) |
| `availableCredits` | number | Total available credits — monthly + purchased - used (in USD) |
| `lastReset` | string | ISO timestamp of the last monthly credit reset |

<Note>
All credit values are in USD. To convert to credit count: `value × 1,000`. For example, `availableCredits: 6.5` = `6,500` credits available.
</Note>

## Enhance prompt

```
POST /public/v1/enhance-prompt
```

Optimize a prompt for better AI generation results. This endpoint is **free** — no credits are consumed.

**Rate limit:** 10 requests per minute.

### Request

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "a cat on a windowsill",
    "mediaType": "image"
  }' \
  https://api.postzee.app/public/v1/enhance-prompt
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `prompt` | string | Yes | The prompt to enhance (max 2,000 characters) |
| `mediaType` | string | Yes | `"image"` or `"video"` |
| `model` | string | No | Target AI model ID for model-specific optimization |
| `stylePreferences` | string | No | Style hints (max 500 characters) |
| `sourceImageUrls` | string[] | No | Reference image URLs (max 5) |

### Response

```json
{
  "enhancedPrompt": "A fluffy ginger cat perched on a sunlit windowsill, warm afternoon light casting soft shadows through lace curtains...",
  "success": true
}
```

| Field | Type | Description |
|-------|------|-------------|
| `enhancedPrompt` | string | The optimized prompt |
| `success` | boolean | Whether enhancement succeeded |
| `error` | string | Error message (only if `success` is `false`) |
