## Meine KI-Generierung ist fehlgeschlagen — wo sind meine Credits?

**Credits werden automatisch erstattet**, wenn eine Generierung fehlschlaegt. Sie muessen keine Rueckerstattung anfordern — es geschieht sofort.

Ueberpruefen Sie Ihr Credit-Guthaben im Bereich **Credits**. Der erstattete Betrag entspricht genau dem, was fuer diese Generierung abgezogen wurde.

<Note>
Sie erhalten eine Benachrichtigung, wenn eine Generierung fehlschlaegt. Die Benachrichtigung enthaelt den Grund fuer den Fehler (Inhalt blockiert, Zeitueberschreitung usw.).
</Note>

## Haeufige Fehlergruende

### Inhalt blockiert

**Ursache:** Der Sicherheitsfilter des KI-Modells hat potenziell unangemessene Inhalte in Ihrem Prompt erkannt.

**Loesung:**
- Formulieren Sie Ihren Prompt um, um sensible Themen zu vermeiden
- Vermeiden Sie Anfragen fuer realistische Darstellungen realer Personen
- Entfernen Sie explizite oder gewalttaetige Beschreibungen
- Probieren Sie ein anderes KI-Modell — jedes Modell hat unterschiedliche Inhaltsrichtlinien

### Generierung hat das Zeitlimit ueberschritten

**Ursache:** Das KI-Modell hat zu lange fuer die Generierung gebraucht. Dies tritt haeufiger bei Videomodellen und hochaufloesenden Bildern auf.

**Loesung:**
- Versuchen Sie es erneut — Zeitueberschreitungen sind normalerweise voruebergehende Serverlastprobleme
- Fuer Videos: Versuchen Sie eine kuerzere Dauer (z.B. 5 Sekunden statt 10)
- Fuer Bilder: Versuchen Sie eine niedrigere Aufloesung oder ein anderes Seitenverhaeltnis
- Versuchen Sie es ausserhalb der Spitzenzeiten

### Probleme mit dem Referenzbild

**Ursache:** Das von Ihnen bereitgestellte Referenzbild konnte nicht verarbeitet werden — es ist moeglicherweise zu klein, zu gross oder in einem nicht unterstuetzten Format.

**Loesung:**
- Verwenden Sie Bilder zwischen 512x512 und 4096x4096 Pixeln
- Verwenden Sie JPG-, PNG- oder WebP-Format
- Stellen Sie sicher, dass die Bilddatei nicht beschaedigt ist
- Versuchen Sie es ohne das Referenzbild, um das Problem einzugrenzen

### Keine Ausgabe generiert

**Ursache:** Das KI-Modell konnte aus Ihrem Prompt kein Ergebnis erzeugen. Dies kann bei sehr abstrakten oder widerspruechlichen Prompts vorkommen.

**Loesung:**
- Seien Sie spezifischer in Ihrer Beschreibung
- Verwenden Sie den **Prompt-Optimierer** (Schaltflaeche ✨), um Ihren Prompt zu verbessern
- Probieren Sie ein anderes Modell, das Ihren Prompt besser interpretieren koennte

## Meine Bildqualitaet ist schlecht

**Tipps fuer bessere Ergebnisse:**

1. **Verwenden Sie den Prompt-Optimierer** — Klicken Sie auf die Schaltflaeche ✨, um die KI Ihren Prompt verbessern zu lassen. Dies verbessert die Ergebnisse drastisch.

2. **Seien Sie spezifisch** — "Ein Golden-Retriever-Welpe am Strand bei Sonnenuntergang, professionelle Fotografie, geringe Schaerfentiefe" funktioniert besser als "Hund am Strand."

3. **Waehlen Sie das richtige Modell:**
   - **Fotorealismus:** Nano Banana Pro, FLUX.2 Pro, GPT Image 2 (HD-Qualitaet)
   - **Illustrationen/Kunst:** Recraft V4, Ideogram V3
   - **Schnelle Entwuerfe:** GPT Image Mini, Nano Banana
   - **Text in Bildern:** Ideogram V3 (am besten fuer praezise Textdarstellung)

4. **Das Seitenverhaeltnis ist wichtig** — Verwenden Sie das Seitenverhaeltnis, das zu Ihrer Zielplattform passt (9:16 fuer Stories/Reels, 1:1 fuer Feed-Beitraege, 16:9 fuer YouTube).

## Mein Video hat visuelle Artefakte

**Haeufige Ursachen:**
- Sehr schnelle Bewegung im Prompt — KI-Videomodelle haben Schwierigkeiten mit schnellen Bewegungen
- Sehr detaillierte Szenen mit vielen kleinen Elementen
- Unrealistische Physikanforderungen

**Loesung:**
- Vereinfachen Sie die Szenenbeschreibung
- Verwenden Sie Prompts, die langsame, fliessende Bewegungen beschreiben
- Probieren Sie ein anderes Videomodell (jedes hat unterschiedliche Staerken)
- Fuer beste Qualitaet verwenden Sie **Kling 2.0 Master** oder **Veo 3.1**

## Die KI hat falschen Text im Bild generiert

**Ursache:** Die meisten KI-Modelle haben Schwierigkeiten mit der Textdarstellung. Sie koennen Woerter falsch schreiben oder unleserlichen Text erzeugen.

**Beste Optionen fuer Text in Bildern:**
1. Verwenden Sie **Ideogram V3** — es ist speziell fuer praezise Textdarstellung konzipiert
2. Verwenden Sie **Recraft V4** — hervorragend fuer Logos und Design mit Text
3. Generieren Sie das Bild ohne Text und fuegen Sie den Text dann mit dem **Foto-Editor** hinzu

## Ich habe keine Credits mehr

**Moeglichkeiten, mehr Credits zu erhalten:**
1. **Credit-Pakete kaufen** — Gehen Sie zu **Credits** → waehlen Sie ein Paket
2. **Auf monatliche Zuruecksetzung warten** — Bezahlte Plaene erhalten bei jedem Abrechnungszyklus neue Credits
3. **Automatisches Aufladen aktivieren** — Unter **Credits** → **Automatisches Aufladen** legen Sie einen Schwellenwert fest, um automatisch mehr Credits zu kaufen, wenn Ihr Guthaben niedrig wird

<Note>
Monatliche Plan-Credits werden bei jedem Abrechnungszyklus zurueckgesetzt. Gekaufte Credits verfallen nie, auch wenn Sie Ihr Abonnement kuendigen.
</Note>

## Welches Modell soll ich verwenden?

| Ziel | Empfohlenes Modell |
|------|-------------------|
| Fotorealistische Bilder | Nano Banana Pro, FLUX.2 Pro |
| Logos und Icons | Recraft V4 |
| Text in Bildern | Ideogram V3 |
| Schnell und guenstig | GPT Image Mini, Nano Banana |
| Beste Gesamtqualitaet | GPT Image 2 (HD) |
| Kurze Videos (5-10s) | Kling 2.0, Wan 2.5 |
| Filmische Videos | Veo 3.1 |
| KI-Avatar-Videos | HeyGen |

Kontaktieren Sie den Support ueber das Chat-Widget mit:
- Dem verwendeten Modell
- Ihrem Prompt (oder einer Beschreibung dessen, was Sie generieren wollten)
- Der Fehlermeldung aus der Benachrichtigung
