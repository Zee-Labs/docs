Verbinden Sie Postzee mit **Claude Code**, um KI-Bilder/Videos zu generieren, HeyGen-Avatar-Videos zu erstellen und auf 30+ Social-Media-Plattformen zu veroeffentlichen -- alles durch natuerlichsprachliche Konversation.

## Postzee Skill installieren

Der einfachste Weg zur Verbindung -- installieren Sie den Postzee Skill von GitHub.

### Option 1: GitHub CLI

```bash
gh skill install Zee-Labs/postzee-skill
```

### Option 2: Manuelle Installation

```bash
mkdir -p ~/.claude/skills/postzee
curl -o ~/.claude/skills/postzee/SKILL.md https://raw.githubusercontent.com/Zee-Labs/postzee-skill/main/SKILL.md
```

## MCP-Verbindung konfigurieren

Nach der Installation des Skills verbinden Sie sich mit Ihrem Postzee-Konto:

1. Gehen Sie zu [Postzee-Einstellungen](https://dashboard.postzee.app/settings) → Registerkarte **API Publica**
2. Kopieren Sie die **MCP-URL** aus dem Abschnitt "MCP (Model Context Protocol)"
3. Fuehren Sie folgenden Befehl in Claude Code aus:

```bash
claude mcp add postzee YOUR_MCP_URL
```

Ersetzen Sie `YOUR_MCP_URL` durch die vollstaendige URL, die Sie kopiert haben (sie sieht aus wie `https://api.postzee.app/mcp/.../http`).

### Verbindung ueberpruefen

```
List my Postzee channels
```

Claude Code sollte das Tool `POSTZEE_LIST_CHANNELS` aufrufen und Ihre verbundenen Social-Media-Konten anzeigen.

## Anwendungsbeispiele

Nach der Verbindung interagieren Sie in natuerlicher Sprache mit Postzee:

### Ein Bild generieren

> "Generate a professional headshot of a woman in a modern office"

### Ein Foto transformieren (Bild-zu-Bild)

> "Transform this photo into anime style" (Bild-URL bereitstellen)

### Ein Foto als Video animieren

> "Animate this photo into a cinematic video for Instagram Reels"

### Ein HeyGen-Avatar-Video erstellen

> "Create a HeyGen video of an avatar presenting our new product"

### Generieren und posten

> "Generate a sunset beach photo and post it to my Instagram with the caption 'Golden hour vibes'"

### Credits pruefen

> "How many AI credits do I have left?"

## Empfohlener Workflow

Fuer beste Ergebnisse:

1. **Credits pruefen** -- stellen Sie sicher, dass Sie genuegend Credits haben
2. **Prompt verbessern** -- nutzen Sie den Prompt Optimizer fuer bessere Ergebnisse (kostenlos)
3. **Generieren** -- erstellen Sie das Bild oder Video
4. **Posten** -- veroeffentlichen oder planen Sie auf Ihren Kanaelen
