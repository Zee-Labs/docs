Vollstaendige Referenz fuer alle **20 Tools**, die von Postzees MCP-Server bereitgestellt werden. Diese Tools stehen jedem MCP-kompatiblen KI-Agenten zur Verfuegung (Claude Code, OpenClaw, Hermes Agent oder direkte MCP-Clients).

<Note>
Alle Tools akzeptieren den API-Key, der in Ihrer MCP-URL eingebettet ist. Sie sind auf eine einzelne Organisation beschraenkt. **Kosten fuer KI-Generierung werden immer in Credits zurueckgegeben, niemals in Dollar.** ($1 USD = 1.000 Credits.)
</Note>

## Schnellreferenz

| Kategorie | Tools |
|-----------|-------|
| **Kontext und Discovery** | `POSTZEE_GET_CONTEXT`, `POSTZEE_LIST_PLANS`, `POSTZEE_LIST_CREDIT_PACKAGES` |
| **Modelle und Specs** | `POSTZEE_LIST_MODELS_DETAILED`, `POSTZEE_LIST_PLATFORM_SPECS`, `POSTZEE_GET_BEST_POSTING_TIMES` |
| **Kosten und Validierung** | `POSTZEE_ESTIMATE_GENERATION_COST`, `POSTZEE_VALIDATE_GENERATION` |
| **Kanaele und Credits** | `POSTZEE_LIST_CHANNELS`, `POSTZEE_GET_CREDITS` |
| **Prompt-Optimierung** | `POSTZEE_ENHANCE_PROMPT` |
| **Generierung** | `POSTZEE_GENERATE_IMAGE`, `POSTZEE_GENERATE_VIDEO`, `POSTZEE_CHECK_JOB` |
| **HeyGen** | `POSTZEE_LIST_HEYGEN_AVATARS`, `POSTZEE_LIST_HEYGEN_VOICES`, `POSTZEE_GENERATE_HEYGEN_VIDEO` |
| **Veroeffentlichung** | `POSTZEE_CREATE_POST` |
| **Legacy (veraltet)** | `POSTZEE_LIST_IMAGE_MODELS`, `POSTZEE_LIST_VIDEO_MODELS` |

## Kontext und Discovery

### POSTZEE_GET_CONTEXT

Kontext-Aggregation in einem einzigen Aufruf. **Empfohlen als erster Aufruf jeder Session.** Gibt Plan, Credits, Speicher, Kanaele, Features und Skill-Version in einem Round-Trip zurueck.

**Parameter:** Keine

**Gibt zurueck:**
```json

  "plan": {
    "tier": "FREE | STANDARD | TEAM | PRO | ULTIMATE",
    "isLifetime": false,
    "period": "MONTHLY | YEARLY | null",
    "cancelAt": "string | null",
    "canPost": true,
    "canConnectChannels": true,
    "canUseAi": true,
    "channelsLimit": 5,
    "postsPerMonth": 400,
    "postsRemaining": 380,
    "monthlyCredits": 2000,
    "storageLimitGB": 15,
    "monthPriceUSD": 27,
    "yearPriceUSD": 252
  },
  "credits": {
    "available": 4500,
    "monthly": 2000,
    "purchased": 2500,
    "used": 0,
    "lastResetAt": "2026-04-15T10:00:00Z",
    "nextResetAt": "2026-05-15T10:00:00Z"
  },
  "storage": { "usedGB": 1.2, "limitGB": 15, "percentUsed": 8 },
  "channels": {
    "connected": 5,
    "active": 4,
    "requiresReauth": 1,
    "disabled": 0,
    "withIssues": 1
  },
  "organization": { "id": "...", "timezone": "Europe/Berlin" },
  "features": {
    "img2vid": true,
    "veoR2V": false,
    "soraStoryboard": false,
    "sora1080p": true,
    "audioControl": true,
    "firstLastFrame": true,

  }
}
```

Die `features.*`-Flags spiegeln wider, was das MCP **gerade jetzt** ausfuehren kann — Agenten sollten nur Features vorschlagen, deren entsprechende Flag `true` ist.

## Modelle und Specs

### POSTZEE_LIST_MODELS_DETAILED

Capability-Matrix fuer alle verfuegbaren Bild- und Video-Modelle — Dauern, Aufloesungen, Audio-Faehigkeiten, akzeptierte Custom-Parameter und relativer Cost-Tier.

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `type` | string | Nein | `"image"`, `"video"` oder `"all"` (default) |
| `includeHeygen` | boolean | Nein | Inkludiert den synthetischen HeyGen-Eintrag. Default: `true` |

**Gibt zurueck (pro Modell):**
```json
{
  "modelId": "sora-2-t2v-pro",
  "displayName": "Sora 2 Pro Text-to-Video",
  "type": "video",
  "audio": "speech-lipsync",
  "supportsImg2Img": false,
  "supportsImg2Vid": false,
  "supportsVector": false,
  "durations": [10, 15],
  "resolutions": ["720p", "1080p"],
  "aspectRatios": ["16:9", "9:16"],
  "customParamsAccepted": {
    "duration": { "type": "enum", "enum": [10, 15], "description": "..." },
    "quality": { "type": "enum", "enum": ["standard", "high"], "description": "..." }
  },
  "costTier": "high",
  "popular": true,
  "bestFor": ["cinematic dialogue", "native lip-sync", "1080p quality"],
  "notRecommendedFor": []
}
```

`costTier` ist einer von `very-low`, `low`, `mid`, `high`, `premium` — die absoluten Credit-Kosten werden absichtlich nicht offengelegt; verwenden Sie stattdessen `POSTZEE_ESTIMATE_GENERATION_COST`.

Der synthetische `heygen-avatar-video`-Eintrag hat `costTier: "external"` und einen `billingNote`, der angibt, dass HeyGen ueber das HeyGen-Konto des Benutzers abrechnet, nicht ueber Postzee-Credits.

### POSTZEE_LIST_PLATFORM_SPECS

Specs pro Plattform (Aspect Ratios, max. Slides fuer Carousels, Zeichenlimits fuer Captions, Hashtag-Empfehlungen).

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `platforms` | string[] | Nein | Filtert auf bestimmte Plattformen (z. B. `["instagram", "tiktok"]`). Weglassen fuer alle. |

Abgedeckte Plattformen: instagram, facebook, linkedin, x, tiktok, youtube, pinterest, threads, bluesky, reddit.

### POSTZEE_GET_BEST_POSTING_TIMES

Heuristische Best-Posting-Time-Fenster pro Kanal, in der Zeitzone der Organisation zurueckgegeben.

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `channelIds` | string[] | Nein | Filtert auf bestimmte Kanaele. Weglassen fuer alle aktiven Kanaele. |

**Gibt zurueck:**
```json
[
  {
    "channelId": "...",
    "platform": "instagram",
    "name": "My Instagram",
    "timezone": "Europe/Berlin",
    "bestTimes": [
      { "time": "11:00", "timezone": "Europe/Berlin", "nextDateLocal": "2026-05-06", "score": 90, "reason": "Lunchtime peak", "source": "heuristic" },
      { "time": "19:00", "timezone": "Europe/Berlin", "nextDateLocal": "2026-05-06", "score": 95, "reason": "Evening prime", "source": "heuristic" }
    ]
  }
]
```

Kanaele mit Status `disabled` oder `requiresReauth` werden gefiltert — Empfehlungen erscheinen nur fuer Kanaele, die tatsaechlich posten koennen.

## Kosten und Validierung

### POSTZEE_ESTIMATE_GENERATION_COST

Single Source of Truth fuer Kostenschaetzungen. **Gibt immer Credits zurueck, niemals Dollar.**

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `type` | string | Ja | `"image"` oder `"video"` |
| `model` | string | Ja | Model-ID aus `POSTZEE_LIST_MODELS_DETAILED` |
| `duration` | number | Nein | Sekunden (Video, fuer Modelle mit Pro-Sekunde-Pricing) |
| `resolution` | string | Nein | `"480p"`, `"540p"`, `"720p"`, `"1080p"`, `"4k"` |
| `quality` | string | Nein | Sora 2 Pro: `"standard"` (720p) oder `"high"` (1080p) |
| `generateAudio` | boolean | Nein | Beeinflusst Kosten bei Veo, Pixverse, Wan 2.5 |
| `voiceControl` | boolean | Nein | Kling 2.6/3.0 Pro |
| `aspectRatio` | string | Nein | Informativ |

**Gibt zurueck:**
```json
{
  "modelId": "sora-2-t2v-pro",
  "displayName": "Sora 2 Pro Text-to-Video",
  "estimatedCredits": 5000,
  "currentBalance": 4500,
  "willExceedBalance": true,
  "creditsAfter": 0,
  "shortfall": 500,
  "warnings": []
}
```

---

### POSTZEE_VALIDATE_GENERATION

Pre-Flight-Validierung — faengt Parameterfehler, unzureichende Credits, Speicherprobleme und Plan-Blockaden ab, **bevor** Credits verbrannt werden.

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `type` | string | Ja | `"image"` oder `"video"` |
| `model` | string | Ja | Model-ID |
| `duration` | number | Nein | Sekunden |
| `aspectRatio` | string | Nein | z. B. `"4:5"` |
| `resolution` | string | Nein | Ausgabe-Aufloesung |
| `quality` | string | Nein | Sora Pro `"standard"` / `"high"` |
| `generateAudio` | boolean | Nein | |
| `voiceControl` | boolean | Nein | |
| `imageUrl` | string | Nein | Erforderlich fuer image-to-video-Modelle |
| `slideCount` | number | Nein | Fuer Carousels — multipliziert die Kosten |

**Gibt zurueck:**
```json
{
  "valid": true,
  "errors": [],
  "warnings": ["Storage at 85% — approaching limit."],
  "estimatedCreditsPerItem": 160,
  "totalEstimatedCredits": 1600,
  "slideCount": 10,
  "currentBalance": 4500,
  "willExceedBalance": false,
  "shortfall": 0
}
```

Wenn `valid: false`, enthaelt das Array `errors[]` lesbare Probleme — z. B. Dauer ist nicht im erlaubten Set des Modells, fehlende `imageUrl` fuer i2v-Modell, Speicher bei 100%, oder Modell ist registriert, aber noch nicht ueber MCP fahrbar.

## Kanaele und Credits

### POSTZEE_LIST_CHANNELS

Listet verbundene Social-Media-Kanaele mit expliziten Action-Flags auf.

**Parameter:** Keine

**Gibt zurueck:**
```json
[
  {
    "id": "channel-uuid",
    "name": "My Instagram",
    "platform": "instagram",
    "picture": "https://...",
    "profile": "https://...",
    "disabled": false,
    "requiresReauth": false,
    "canPostNow": true,
    "actionRequired": "none",
    "actionUrl": null,
    "actionMessage": null,
    "status": "active"
  }
]
```

`actionRequired` ist einer von `"none"` (bereit zu posten), `"reconnect"` (Token abgelaufen — der Benutzer verbindet neu) oder `"reenable"` (Admin/Billing — der Benutzer reaktiviert, **nicht** dasselbe wie reconnect).

## Generierung

### POSTZEE_GENERATE_IMAGE

Generiert ein KI-Bild aus einem Prompt. **Asynchrone Operation** — gibt eine Job-ID zurueck, die mit `POSTZEE_CHECK_JOB` gepollt werden muss.

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `prompt` | string | Ja | Detaillierte Bildbeschreibung |
| `model` | string | Ja | Model-ID aus `POSTZEE_LIST_MODELS_DETAILED` (type `"image"`) |
| `aspectRatio` | string | Nein | z. B. `"1:1"`, `"4:5"`, `"9:16"`, `"16:9"`. Default: `"1:1"` |
| `imageUrls` | string[] | Nein | Referenz-Bild-URLs fuer image-to-image |
| `quality` | string | Nein | GPT Image 2: `"low"`, `"medium"` oder `"high"`. Default: `"high"` |
| `style` | string | Nein | Recraft V3/V4: `"realistic_image"`, `"digital_illustration"`, `"vector_illustration"` |
| `renderingSpeed` | string | Nein | Ideogram-V3-Tier: `"turbo"`, `"balanced"` oder `"quality"` |
| `negativePrompt` | string | Nein | Dinge, die aus dem Bild ausgeschlossen werden sollen |
| `seed` | number | Nein | Optionaler deterministischer Seed |

**Gibt zurueck (Erfolg):**
```json
{ "success": true, "jobId": "job-uuid", "status": "processing", "message": "Image generation started..." }
```

**Gibt zurueck (Fehler):**
```json
{ "success": false, "error": "unsupported_model | organization_not_found | generation_failed", "message": "..." }
```

---

### POSTZEE_GENERATE_VIDEO

Generiert ein KI-Video aus einem Prompt oder Referenzbild. **Asynchrone Operation.**

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `prompt` | string | Ja | Detaillierte Videobeschreibung |
| `model` | string | Ja | Model-ID aus `POSTZEE_LIST_MODELS_DETAILED` (type `"video"`) |
| `duration` | number | Nein | Sekunden. Erlaubte Werte variieren pro Modell — siehe `model.durations` |
| `aspectRatio` | string | Nein | `"16:9"` oder `"9:16"`. Default: `"16:9"` |
| `imageUrl` | string | Nein | Referenz-Bild-URL fuer image-to-video-Modelle |
| `endImageUrl` | string | Nein | URL des End-Frames fuer First-Last-Frame-Interpolation (z. B. `wan-flf2v`) |
| `resolution` | string | Nein | `"480p"`, `"540p"`, `"720p"`, `"1080p"`, `"4k"` (modellabhaengig) |
| `quality` | string | Nein | Nur Sora 2 Pro: `"standard"` (720p) oder `"high"` (1080p) |
| `generateAudio` | boolean | Nein | Generiert Umgebungsgeraeusche (Veo 3.1, Pixverse, Wan 2.5). Beeinflusst Credits. |
| `voiceControl` | boolean | Nein | Stimmkontrolle (Kling 2.6/3.0 Pro). Beeinflusst Credits. |
| `negativePrompt` | string | Nein | Dinge, die im Video vermieden werden sollen |
| `seed` | number | Nein | Optionaler deterministischer Seed |

**Gibt zurueck:** gleiche Struktur wie `POSTZEE_GENERATE_IMAGE`.

<Warning>
Einige Model-IDs im Katalog sind noch nicht ueber MCP fahrbar und werden im Voraus abgelehnt (Sora 2 storyboard-Varianten, einige Veo 3.1-Pfade). Verwenden Sie `POSTZEE_LIST_MODELS_DETAILED`, um aktuell unterstuetzte Modelle zu sehen — sie werden gefiltert.
</Warning>

### POSTZEE_CHECK_JOB

Pollt den Status eines asynchronen Bild- oder Video-Generierungsjobs.

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `jobId` | string | Ja | Job-ID aus `POSTZEE_GENERATE_*` |

**Gibt zurueck (in Bearbeitung):**
```json
{ "jobId": "...", "status": "processing", "progress": 45 }
```

**Gibt zurueck (Erfolg):**
```json
{ "jobId": "...", "status": "success", "progress": 100, "mediaUrl": "https://cdn.postzee.app/...", "mediaId": "media-uuid" }
```

**Gibt zurueck (Fehler):**
```json
{ "jobId": "...", "status": "failed", "error": "Generation failed: ..." }
```

**Gibt zurueck (nicht gefunden):**
```json
{ "jobId": "...", "status": "not_found" }
```

## Veroeffentlichung

### POSTZEE_CREATE_POST

Erstellt, plant oder veroeffentlicht einen Post auf einem Social-Media-Kanal.

**Parameter:**

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `type` | string | Ja | `"now"` (sofort veroeffentlichen), `"draft"` oder `"schedule"` |
| `date` | string | Nur Schedule | UTC-Datetime (z. B. `"2026-04-24T15:00:00Z"`) — **erforderlich**, wenn `type` `"schedule"` ist |
| `channelId` | string | Ja | Channel-ID aus `POSTZEE_LIST_CHANNELS` |
| `text` | string | Ja | Post-Inhalt |
| `mediaUrls` | string[] | Nein | Medien-URLs (in Anzeigereihenfolge — Array-Reihenfolge = Slide-Reihenfolge fuer Carousels) |

**Gibt zurueck (Erfolg):**
```json
{ "success": true, "postId": "post-uuid", "message": "Post published successfully." }
```

**Gibt zurueck (Fehler):**
```json
{ "success": false, "error": "<error_code>", "message": "...", "upgradeUrl": "https://dashboard.postzee.app/billing" }
```

**Moegliche `error`-Codes:**

| Code | Bedeutung |
|------|-----------|
| `subscription_required` | Der aktuelle Plan erlaubt keine Veroeffentlichung oder das monatliche Post-Limit ist erreicht. `upgradeUrl` ist enthalten. |
| `date_required` | `type: "schedule"` wurde ohne `date` gesendet. |
| `invalid_date` | Der `date`-String ist kein gueltiges UTC-ISO-Timestamp. |
| `organization_not_found` | Der MCP-Key wurde zu einer nicht existierenden Organisation aufgeloest (selten). |
| `post_failed` | Provider-/Netzwerkfehler waehrend der Veroeffentlichung. Einmal wiederholen; bei Persistenz die `message` anzeigen. |

## Legacy-Tools (veraltet)

Die folgenden Tools funktionieren noch, sind aber abgeloest:

- **`POSTZEE_LIST_IMAGE_MODELS`** → verwenden Sie `POSTZEE_LIST_MODELS_DETAILED({ type: "image" })`
- **`POSTZEE_LIST_VIDEO_MODELS`** → verwenden Sie `POSTZEE_LIST_MODELS_DETAILED({ type: "video" })`

Sie werden mindestens 90 Tage lang fuer Rueckwaertskompatibilitaet beibehalten und in einer zukuenftigen Major-Version entfernt.

## Asynchrones Workflow-Pattern

Bild-, Video- und HeyGen-Generierung ist asynchron. Der empfohlene Workflow:

1. Validieren Sie zuerst — `POSTZEE_VALIDATE_GENERATION` faengt Parameterfehler und unzureichende Credits ab, ohne Credits zu verbrennen
2. Schaetzen Sie die Kosten — `POSTZEE_ESTIMATE_GENERATION_COST`, falls Sie die genaue Zahl benoetigen, um sie dem Benutzer zu zeigen
3. Optimieren Sie den Prompt — `POSTZEE_ENHANCE_PROMPT` (kostenlos)
4. Generieren — `POSTZEE_GENERATE_IMAGE` / `POSTZEE_GENERATE_VIDEO` / `POSTZEE_GENERATE_HEYGEN_VIDEO` → gibt `jobId` zurueck
5. Pollen — `POSTZEE_CHECK_JOB` alle paar Sekunden. Typische Latenzen: 10-60s fuer Bilder, 30-180s fuer Videos, bis zu 5 Min fuer HeyGen
6. Bei Erfolg — verwenden Sie `mediaUrl` in `POSTZEE_CREATE_POST`

## Standard-Fehlerformat

Alle write/generate-Tools geben zurueck:

```json
{
  "success": true | false,
  "error": "<machine_code>",
  "message": "<human readable>",
  "...": "tool-spezifische Felder bei Erfolg"
}
```

Verzweigen Sie auf den `error`-Machine-Code fuer Logik; verwenden Sie `message`, um dem Benutzer (in seine Sprache uebersetzt) zu erzaehlen.

| Endpoint | Limit |
|----------|-------|
| HTTP POST | 60 Anfragen/Minute |
| SSE-Verbindung (legacy) | 30 Anfragen/Minute |
| KI-Generierung (`GENERATE_*`) | 10 Anfragen/Minute |
