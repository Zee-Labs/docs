Die **Postzee Skill** ist ein strukturiertes Set von Anweisungen, das ueber GitHub (und ClawHub fuer OpenClaw) verteilt wird und Ihrem KI-Agenten beibringt, Postzee so zu nutzen, wie es ein erfahrenes Creative-Team tun wuerde.

Nach der Installation hoert Ihr Agent auf, ein blosser "Tool-Wrapper" zu sein, und beginnt sich wie eine Kreativagentur zu verhalten: erstellt Briefs, waehlt das richtige Modell, validiert die Kosten, schreibt Captions, die konvertieren, und veroeffentlicht in optimaler Reihenfolge.

<Note>
Die Skill wird oeffentlich verteilt und funktioniert mit jedem MCP-kompatiblen Agenten, der Skills unterstuetzt (Claude Code, OpenClaw, Hermes Agent und andere). Die gesamte Postzee-Arbeit wird ueber den [MCP-HTTP-Server](/integrations/mcp/overview) abgewickelt — niemals ueber direkte REST-Aufrufe.
</Note>

## Hauptfunktionen

### 🌍 Mehrsprachigkeit von Haus aus

Ihr Agent **antwortet immer in Ihrer Sprache** — Deutsch, Englisch, Portugiesisch, Spanisch, Franzoesisch, Japanisch oder andere. Obwohl die Skill auf Englisch geschrieben ist, uebersetzt der Agent alles in Ihre Sprache und passt den Ton kulturell an.

### 💳 Plan- und Kreditbewusstsein

Bevor irgendetwas generiert wird, liest der Agent den vollstaendigen Account-Kontext — Plan-Stufe, Kreditstand, Speicherplatz, Kanalstatus — und **antizipiert Probleme, statt mittendrin auf sie zu stossen**:

- **Free-Plan + moechte posten?** Generiert das Asset, schlaegt dann den richtigen kostenpflichtigen Plan mit ueberzeugender Copy vor und bietet Dateilieferung als Alternative.
- **Wenig Credits?** Zeigt das Defizit und empfiehlt das passende Kreditpaket — keine Paywall-Ueberraschungen.
- **Kanaele muessen neu verbunden werden?** Sagt Ihnen welche und wie (Reconnect vs Reenable — verschiedene Aktionen).
- **Speicher fast voll?** Warnt Sie vor der Generierung, damit keine Credits an Dateien verschwendet werden, die beim Speichern fehlschlagen.

### 💰 Kostentransparenz

Der Agent **zeigt immer die Generierungskosten in Credits vor der Generierung** — nie in Dollar, nie als Ueberraschung. Er nutzt das Live-Tool `POSTZEE_ESTIMATE_GENERATION_COST`, um exakte Zahlen aus der aktuellen Preisgestaltung zu berechnen — kein Raten, keine hardcoded Werte.

### 🎨 Bild- und Karussell-Meisterschaft (HTML-Render -- v3.5+ fuer Karussells, v3.7+ fuer einzelne Bilder)

Karussells und textreiche einzelne Bilder laufen ueber eine dedizierte Pipeline: der Agent **schreibt das HTML jeder Folie** (pixelgenauer Text, Ihre Typografie) und Postzee rendert per Headless-Chromium zu PNG. KI-Bildmodelle rendern Text schlecht — dieser Ansatz beseitigt diese Einschraenkung vollstaendig.

Die Skill bringt eine vollstaendige redaktionelle Methodologie, die Arbeit produziert, die nicht von dem zu unterscheiden ist, was ein erstklassiges menschliches Redaktionsteam veroeffentlichen wuerde — starke Winkelauswahl, rigorose Qualitaetspruefung und ein visuelles System, das den Scroll erobert. Der Nutzer sieht den Vorschlag, gibt frei, und der Agent liefert die Assets via `POSTZEE_RENDER_IMAGE` (redaktionelles Einzelbild) oder `POSTZEE_RENDER_CAROUSEL` (Multi-Slide). Iteration nach dem Render ist chirurgisch — `POSTZEE_REPLACE_CAROUSEL_SLIDE` fuer eine punktuelle Anpassung, `POSTZEE_APPEND_CAROUSEL_SLIDE` zum Erweitern eines wachsenden Karussells.

### ⚙️ Plattform-Settings-Awareness (v3.7+)

Die Skill kennt die Settings jedes sozialen Netzwerks — Instagram Story vs Feed, TikTok-Privatsphaere + Duet + Musik-Steuerung, YouTube-Sichtbarkeit + Tags, LinkedIn-Karussell-Modus, Pinterest-Board und mehr. Wenn Sie sagen „poste das als Story" oder „privater Upload auf TikTok", routet der Agent das richtige Setting automatisch an `POSTZEE_CREATE_POST` — Sie muessen die Plattform-Mechaniken nicht ausspecifizieren.

### ✅ Plattformbewusste Publish-Guards (v3.5.1+)

Postzee setzt die nativen Karussell-Limits jedes sozialen Netzwerks durch, **bevor** der Plattform-API-Aufruf erfolgt — keine 4 Minuten Retries und 30+ Media-Container, die fuer ein Karussell verschwendet werden, das das Netzwerk sowieso abgelehnt haette.

| Netzwerk | Max. Slides pro Karussell |
|---|---|
| Instagram / Facebook | 10 |
| LinkedIn | 20 |
| TikTok | 35 |
| Threads / Reddit | 20 |
| Pinterest | 5 |
| X | 4 |
| Bluesky / Mastodon | 4 |
| Telegram / VK | 10 |

Wenn ein Karussell das Limit des Ziel-Netzwerks ueberschreitet (z. B. 13 Slides → Instagram), bricht Postzee sofort ab — mit einer Meldung, die **in die Sprache des Nutzers uebersetzt** ist — *"Instagram akzeptiert maximal 10 Slides pro Karussell. Dein Beitrag hat 13."* — statt die Netzwerk-API einen `fbtrace_id` zurueckgeben zu lassen. Skill v3.5.1+ deckelt zudem die Slide-Anzahl bereits im Brief, wenn die Zielplattform bekannt ist, sodass der Nutzer vor dem ersten Render korrigiert wird.

### 🎬 Multi-Szenen-Video

Fuer Geschichten mit 2+ verbundenen Szenen orchestriert die Skill Charakter- und Schauplatz-Konsistenz ueber mehrere Generierungen hinweg. Strategien sind an das gebunden, was im MCP aktuell verfuegbar ist — der Agent schlaegt nur vor, was Ende-zu-Ende funktioniert.

### 🎙️ Talking-Head-Entscheidungen

Wenn Sie jemanden bitten, einen bestimmten Text zu sprechen, waehlt die Skill intelligent zwischen **HeyGen** (perfekte Lip-Sync, Stimmkontrolle, mehrsprachig, Langform) und **Sora 2** / **Veo 3.1** (cinematische Szenen mit nativem Audio) — und sagt Ihnen vorher, welche Postzee-Credits abrechnet und welche HeyGen-Credits nutzt.

### 🎞️ Komposition mit ffmpeg + Whisper

Fuer Agenten mit Shell-Zugriff (OpenClaw, Hermes, Claude Code mit Shell) enthaelt die Skill ein vollstaendiges Cookbook: Concat mit Crossfades, Audio-Mischung mit Sidechain-Ducking, Loudness-Normalisierung (-14 LUFS), Aspect-Ratio-Konvertierung, Ken Burns, Picture-in-Picture. Plus Whisper-Untertitel-Workflows mit 5 Stilen.

### 🔍 Trend-Bewusstsein

Fuer Kreativarbeit fuehrt der Agent eine schnelle Websuche nach aktuellen viralen Mustern durch, bevor er Konzepte vorschlaegt — ohne Quellenangabe, sofern Sie nicht danach fragen.

### 📅 Intelligente Terminplanung

Der Agent kennt die besten Veroeffentlichungszeiten pro Plattform, **in Ihrer Zeitzone** (gelesen aus Ihren Postzee-Org-Einstellungen) — und konvertiert beim Planen in UTC ISO.

### 🧭 Intelligentes Modell-Routing (v3.2)

Sie muessen nicht wissen, welches Modell zu verwenden ist. Sagen Sie *"erstelle ein Sonnenuntergangs-Foto fuer Instagram"* und der Agent waehlt die richtige Familie (textlastig vs fotorealistisch vs Illustration), das richtige Tier (fast / balanced / premium) und das richtige Aspect-Ratio fuer die Plattform — automatisch. Das Standard-Tier ist **fast**, um die Automatisierungskosten vorhersehbar zu halten. (Karussells laufen ueber `POSTZEE_RENDER_CAROUSEL` und brauchen keine Modellauswahl -- die HTML-Pipeline erledigt sie.)

Wenn Sie WISSEN, was Sie wollen — sagen Sie *"mit Nano Banana"* oder *"mit hoechster Qualitaet"* — respektiert der Agent Ihren Override. Das mehrsprachige Signal-Vokabular (PT / EN / ES / FR / DE) ist nativ eingebaut.

### 🔄 Selbst-Versionspruefung

Bei jeder neuen Session prueft der Agent, ob die installierte Skill-Version mit der aktuell veroeffentlichten uebereinstimmt. Falls Sie veraltet sind, sagt er es Ihnen einmal (in Ihrer Sprache) mit dem Update-Befehl fuer Ihren Client — und arbeitet weiter mit der Version, die Sie haben.

## Architektur

Die Skill wird als **Orchestrator-Datei** verteilt, die bei jeder Session geladen wird, plus einer Reihe von Referenzdateien, die der Orchestrator bei Bedarf laedt (kreatives Briefing, Model-Routing, Video-Komposition usw.). Das haelt den Session-Kontext schlank und gibt dem Agenten tiefes Wissen genau dort, wo jeder Teil des Workflows es benoetigt.

Die Skill wird als offenes Archiv ueber GitHub Releases publiziert — Sie koennen sie von Anfang bis Ende lesen, wenn Sie genau verstehen wollen, was Ihr Agent tut.

## Was die Skill nie tut

- ❌ Preise in Dollar fuer KI-Generierungen nennen (immer Credits — `1 USD = 1.000 Credits` ist das Anzeigeverhaeltnis)
- ❌ Eine Funktion empfehlen, die nicht ueber MCP verfuegbar ist
- ❌ Generieren, ohne Ihnen zuerst die Kosten zu zeigen
- ❌ Die Versionspruefung in einer neuen Session ueberspringen
- ❌ Automatisch auf einen getrennten oder reauth-pflichtigen Kanal posten
- ❌ Captions zwischen Sprachen woertlich uebersetzen — passt kulturell an
