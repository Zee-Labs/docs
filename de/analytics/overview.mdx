Verfolgen Sie die Performance Ihrer Social-Media-Inhalte mit den integrierten Analysen von Postzee. Sehen Sie Engagement-Metriken, Zielgruppen-Einblicke und Content-Performance über alle Ihre verbundenen Kanäle hinweg.

## Zugang

Klicken Sie auf **Metrics** in der oberen Navigationsleiste. Wählen Sie einen Kanal aus, um dessen Metriken anzuzeigen, oder sehen Sie eine aggregierte Ansicht über alle Kanäle.

## Verfügbare Metriken

Analysedaten stammen direkt aus der API jeder Plattform. Die verfügbaren Metriken hängen von der Plattform ab:

### Allgemeine Metriken

| Metrik | Beschreibung |
|--------|-------------|
| **Impressionen** | Wie oft Ihr Inhalt angezeigt wurde |
| **Reichweite** | Wie viele einzelne Konten Ihren Inhalt gesehen haben |
| **Engagement** | Gesamte Interaktionen (Likes, Kommentare, Shares, Saves) |
| **Engagement-Rate** | Engagement geteilt durch Reichweite oder Impressionen |
| **Follower** | Aktuelle Follower-Anzahl und Wachstum im Zeitverlauf |
| **Klicks** | Link-Klicks und Profilbesuche |

### Plattformspezifische Metriken

| Plattform | Zusätzliche Metriken |
|-----------|---------------------|
| **Instagram** | Story-Aufrufe, Reel-Wiedergaben, Saves, Profilbesuche |
| **Facebook** | Seiten-Likes, Beitragsreichweite, Videoaufrufe |
| **LinkedIn** | Beitragsimpressionen, Klickrate |
| **YouTube** | Wiedergabezeit, Abonnenten, Videoaufrufe |
| **X (Twitter)** | Tweet-Impressionen, Profilbesuche, Erwähnungen |
| **TikTok** | Videoaufrufe, Shares, Wiedergabezeit |

## Daten filtern

Verwenden Sie die Steuerelemente oben auf der Analyseseite zum Filtern:

| Filter | Optionen |
|--------|----------|
| **Kanal** | Einen bestimmten Kanal auswählen oder alle anzeigen |
| **Zeitraum** | Benutzerdefinierte Datumsbereichsauswahl |
| **Metriktyp** | Auf bestimmte Metriken fokussieren |

## Datenaktualität

Analysedaten werden aus Performance-Gründen zwischengespeichert:

- Daten werden **stündlich** in der Produktion aktualisiert
- Metriken werden in regelmäßigen Abständen über die API jeder Plattform abgerufen
- Einige Plattformen haben möglicherweise eigene Verzögerungen bei der Berichterstattung (z. B. können Instagram-Metriken bis zu 24 Stunden dauern)

<Note>
Die Verfügbarkeit von Analysen hängt von der API der jeweiligen Plattform ab. Einige Plattformen bieten detailliertere Metriken als andere. Postzee zeigt alle verfügbaren Daten jedes verbundenen Kanals an.
</Note>

## Beitragsperformance

Über die Kanal-Level-Analysen hinaus können Sie die Performance einzelner Beiträge verfolgen:

- **Klick-Tracking** — Beiträge mit Kurzlinks enthalten Klickstatistiken
- **Veröffentlichungsstatus** — sehen Sie, welche Beiträge kanalübergreifend erfolgreich waren oder fehlgeschlagen sind
- **Kanalübergreifender Vergleich** — vergleichen Sie, wie derselbe Inhalt auf verschiedenen Plattformen abschneidet
