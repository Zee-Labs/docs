Der **Prompt Optimizer** verwandelt Ihre einfachen Beschreibungen in professionelle Prompts, die die Ergebnisse der KI-Generierung erheblich verbessern. Die Nutzung ist kostenlos — keine Kreditkosten.

## Was er bewirkt

Wenn Sie auf die Schaltfläche „Verbessern" klicken, analysiert der Prompt Optimizer Ihre Beschreibung und erweitert sie um:

- **Kameraspezifikationen** — Objektivtyp, Brennweite, Winkel, Schärfentiefe
- **Lichtdesign** — natürliche Lichtrichtung, Studioaufbauten, Goldene Stunde, dramatische Schatten
- **Komposition** — Drittelregel, Führungslinien, Rahmung, Vordergrund-/Hintergrund-Balance
- **Farbkorrektur** — Palette, Ton, Sättigung, Filmstock-Emulation
- **Texturen und Materialien** — Oberflächendetails, Stoff, Haut, Metall, Glas
- **Künstlerischer Stil** — Fotografie-Genre, Kunstbewegung, visuelle Referenz
- **Bewegungsdetails** (für Videos) — Kamerabewegungen, Szenenübergänge, Bewegungsdynamik

## So verwenden Sie ihn

1. Geben Sie einen einfachen Prompt in das Erstellungsfeld ein (z. B. „eine Katze auf einer Fensterbank")
2. Klicken Sie auf die Schaltfläche **Enhance with AI** (✨) — das lila Funkeln-Symbol neben dem Prompt-Feld
3. Warten Sie 2–3 Sekunden, während die KI Ihren Prompt optimiert
4. Der verbesserte Prompt ersetzt Ihren Originaltext im Feld
5. Überprüfen Sie das Ergebnis und klicken Sie auf **Generate**

### Beispiel

**Vorher (Ihre Eingabe):**
> Eine Frau, die im Regen spaziert

**Nachher (optimiert):**
> Eine junge Frau in einem maßgeschneiderten Trenchcoat geht durch eine regennasse Stadtstraße in der Abenddämmerung. Aufgenommen mit einem 50-mm-Objektiv bei f/1.8, geringe Schärfentiefe. Weiches bernsteinfarbenes Straßenlicht spiegelt sich auf nassem Kopfsteinpflaster. Filmische Farbkorrektur mit kühlen blauen Schatten und warmen bernsteinfarbenen Highlights. Regentropfen in der Bewegung eingefangen mit subtiler Bewegungsunschärfe. Redaktioneller Modefotografie-Stil, Vogue-inspirierte Komposition mit Fluchtlinien aus der Straßenperspektive.

## Modellspezifische Optimierung

Der Prompt Optimizer passt seine Ausgabe an das ausgewählte KI-Modell an:

| Modelltyp | Optimierungsschwerpunkt |
|-----------|------------------------|
| **Fotorealistische Modelle** (Nano Banana, FLUX) | Technische Fotografie-Begriffe, Kameraeinstellungen, Beleuchtungsaufbauten, Farbwissenschaft |
| **Design-Modelle** (Recraft, Ideogram) | Visuelle Hierarchie, Typografie-Platzierung, markensichere Kompositionen, Vektorklarheit |
| **Video-Modelle** (Sora 2, Veo 3.1) | Kamerabewegungen (Schwenk, Neigung, Zoom, Dolly), Szenenübergänge, Bewegungsdynamik, zeitliche Progression |
| **Image-to-Video** | Natürliche Animation vorhandener Elemente, Kamera-Tracking, atmosphärische Effekte, sanfte Übergänge |

## Kosten

Der Prompt Optimizer ist **vollständig kostenlos**. Er verbraucht keine KI-Kredite. Nutzen Sie ihn so oft Sie möchten.

<Note>
Kredite werden nur belastet, wenn Sie auf **Generate** klicken, um das eigentliche Bild oder Video zu erstellen. Der Schritt der Prompt-Verbesserung ist immer kostenlos.
</Note>

## Tipps

- **Fangen Sie einfach an** — der Optimizer funktioniert am besten, wenn Sie ihm eine klare, prägnante Idee zum Erweitern geben
- **Nicht zu detailliert beschreiben** — wenn Sie einen langen, detaillierten Prompt schreiben, hat der Optimizer weniger Spielraum, Mehrwert hinzuzufügen
- **Ergebnis überprüfen** — Sie können den verbesserten Prompt vor der Generierung bearbeiten, um das Ergebnis fein abzustimmen
- **Auch für Videos nutzen** — der Optimizer fügt Kamerabewegungen und Bewegungsbeschreibungen hinzu, die die Videoqualität erheblich verbessern
- **Mit Spracheingabe kombinieren** — sprechen Sie Ihre Idee über das Mikrofon ein und verbessern Sie sie dann mit dem Optimizer
