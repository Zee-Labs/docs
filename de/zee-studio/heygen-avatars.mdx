Erstellen Sie professionelle Talking-Head-Videos mit KI-Avataren, die von **HeyGen** betrieben werden. Wählen Sie aus einer Bibliothek realistischer Avatare, wählen Sie eine Stimme, schreiben Sie Ihr Skript und generieren Sie ein Video — keine Kamera und kein Mikrofon erforderlich.

## Was sind HeyGen Avatars?

HeyGen Avatars sind KI-generierte Video-Moderatoren, die lippensynchron zu Ihrem Skript sprechen. Sie sehen aus und bewegen sich wie echte Personen und sind daher ideal für:

- Produktdemos und Erklärvideos
- Social-Media-Inhalte mit persönlicher Note
- Schulungs- und Bildungsvideos
- Marketing-Ankündigungen
- Mehrsprachige Inhalte (gleicher Avatar, verschiedene Sprachen)

## Erste Schritte

### Ihr HeyGen-Konto verbinden

Bevor Sie Avatare verwenden können, müssen Sie Ihr HeyGen-Konto verbinden:

1. Öffnen Sie das Kontomenü (klicken Sie auf Ihren Avatar) → **Settings** → Reiter **Integrations**
2. Suchen Sie **HeyGen** in der Integrationsliste
3. Klicken Sie auf **Connect** und folgen Sie dem Autorisierungsablauf
4. Sobald die Verbindung hergestellt ist, stehen Avatare in Zee Studio zur Verfügung

<Note>
Sie benötigen ein HeyGen-Konto, um diese Funktion zu nutzen. Falls Sie noch keines haben, können Sie sich [bei HeyGen registrieren](https://www.heygen.com/signup) und es dann mit Postzee verbinden.
</Note>

## So erstellen Sie ein Avatar-Video

1. Öffnen Sie **Zee Studio** über die obere Navigationsleiste (klicken Sie auf **Zee Studio**)
2. Wählen Sie den Modus **AI Video** im Erstellungspanel
3. Wählen Sie **HeyGen Avatar** als Generierungstyp
4. **Wählen Sie einen Avatar**:
   - Durchsuchen Sie die Avatar-Galerie
   - Suchen Sie nach Name, um bestimmte Avatare zu finden
   - Bewegen Sie den Mauszeiger über einen Avatar, um eine Videovorschau anzuzeigen
   - Klicken Sie zur Auswahl
5. **Wählen Sie eine Stimme**:
   - Durchsuchen Sie die verfügbaren Stimmen
   - Filtern Sie nach Sprache über das Dropdown-Menü
   - Klicken Sie auf die Wiedergabetaste, um eine Stimme vorab anzuhören
   - Klicken Sie zur Auswahl
6. **Schreiben Sie Ihr Skript** — geben Sie den Text ein oder fügen Sie ihn ein, den Ihr Avatar sprechen soll
7. Überprüfen Sie die Kreditkosten und klicken Sie auf **Generate**
8. Warten Sie auf die Verarbeitung — Avatar-Videos dauern typischerweise 2–5 Minuten
9. Ihr Video erscheint unter **Creations**, sobald es fertig ist

## Einen Avatar auswählen

Die Avatar-Galerie zeigt alle verfügbaren HeyGen Avatare:

- **Suche** — Avatare nach Name finden
- **Vorschau** — bewegen Sie den Mauszeiger über eine Avatar-Karte, um eine kurze Videovorschau zu sehen
- **Typen** — reguläre Avatare und sprechende Fotos sind verfügbar
- **Geschlecht** — durchsuchen Sie männliche und weibliche Moderatoren

Klicken Sie auf einen Avatar, um ihn auszuwählen. Ein lila Rahmen zeigt Ihre Auswahl an.

## Eine Stimme auswählen

Der Stimmenauswähler hilft Ihnen, die perfekte Stimme für Ihren Inhalt zu finden:

- **Sprachfilter** — wählen Sie eine Sprache, um nur Stimmen in dieser Sprache anzuzeigen
- **Geschlechtsanzeige** — bei jeder Stimme wird angezeigt, ob sie weiblich oder männlich ist
- **Audio-Vorschau** — klicken Sie auf die Wiedergabetaste, um vor der Auswahl eine Hörprobe zu erhalten
- **Suche** — Stimmen nach Name finden

<Note>
Für mehrsprachige Inhalte können Sie denselben Avatar mit verschiedensprachigen Stimmen verwenden. Der Avatar synchronisiert die Lippenbewegungen natürlich zu jeder unterstützten Sprache.
</Note>

## Tipps für bessere Avatar-Videos

- **Schreiben Sie natürlich** — verfassen Sie Ihr Skript so, wie Sie sprechen würden, mit Pausen und natürlicher Formulierung
- **Halten Sie Skripte fokussiert** — kürzere, fokussierte Skripte liefern bessere Ergebnisse als lange Monologe
- **Stimme zum Inhalt passend wählen** — wählen Sie eine Stimme, die zu Ihrem Markenton passt (professionell, freundlich, energisch)
- **Vorschauen testen** — hören und sehen Sie sich immer sowohl Avatar als auch Stimme an, bevor Sie generieren

## Nach der Generierung

Sobald Ihr Avatar-Video fertig ist, können Sie:

- **Download** — das Video auf Ihrem Gerät speichern
- **Create Post** — das Video an einen Social-Media-Beitrag anhängen
- **Publish** — in der Explore-Galerie teilen
