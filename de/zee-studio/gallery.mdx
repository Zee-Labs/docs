Ihre persönliche Medienbibliothek in Zee Studio ist in zwei Tabs organisiert: **Creations** für KI-generierte Inhalte und **Files** für manuell hochgeladene Dateien.

## Kreationen

Der Tab **Creations** zeigt alle Bilder und Videos, die Sie mit KI generiert haben. Jedes Element zeigt:

- Eine Miniaturvorschau
- Das für die Generierung verwendete KI-Modell
- Wie lange die Erstellung zurückliegt (z. B. „vor 2 Stunden")

### Kreationen filtern

Verwenden Sie die Filter-Buttons oberhalb der Galerie, um Ihre Ansicht einzugrenzen:

| Filter | Zeigt |
|--------|-------|
| **All** | Alle generierten Medien |
| **Images** | Nur KI-generierte Bilder |
| **Videos** | Nur KI-generierte Videos |

### Aktionen für Ihre Kreationen

Klicken Sie auf ein beliebiges Element unter Creations, um auf diese Aktionen zuzugreifen:

| Aktion | Beschreibung |
|--------|-------------|
| **Download** | Die Datei auf Ihrem Gerät speichern (JPG für Bilder, MP4 für Videos) |
| **Create Post** | Den Beitragseditor mit diesem Medium öffnen |
| **Recreate** | Dasselbe Modell und dieselben Einstellungen laden, um eine neue Variante zu generieren |
| **Use Prompt** | Den ursprünglichen Prompt zurück in das Erstellungsfeld kopieren |
| **View Details** | Vollständige Generierungsmetadaten anzeigen — Modell, Prompt, Einstellungen, Auflösung |
| **Edit Image** | Das Bild im integrierten Editor bearbeiten (nur Bilder) |
| **Animate** | Ein Standbild im Image-to-Video-Modus in ein Video umwandeln (nur Bilder) |
| **Publish** | In der öffentlichen Explore-Galerie teilen, damit die Community es entdecken kann |
| **Delete** | Das Medium dauerhaft löschen (mit Bestätigung) |

### Laufende Generierungen

Wenn eine Generierung in Bearbeitung ist, erscheint sie unter Creations mit einem Lade-Indikator. Sie können:

- Den Fortschrittsstatus sehen (in Warteschlange, in Bearbeitung)
- Während des Wartens an anderen Aufgaben weiterarbeiten
- Zee Studio schließen und später zurückkommen — das Element ist fertig, wenn Sie zurückkehren

## Dateien

Der Tab **Files** zeigt Dateien, die Sie manuell hochgeladen haben. Nutzen Sie dies für Ihre eigenen Fotos, Marken-Assets, Logos oder andere Medien, die Sie in Beiträgen verwenden möchten.

### So laden Sie hoch

- **Drag and Drop** — ziehen Sie Dateien direkt auf den Galeriebereich
- **Klicken zum Durchsuchen** — klicken Sie auf die Upload-Zone, um Ihren Dateiauswähler zu öffnen
- **Upload-Button** — verwenden Sie den Upload-Button in der Galerie-Werkzeugleiste

Unter [Medien hochladen](/zee-studio/upload) finden Sie unterstützte Formate und Größenlimits.

### Aktionen für hochgeladene Medien

| Aktion | Beschreibung |
|--------|-------------|
| **Download** | Die Datei auf Ihrem Gerät speichern |
| **Create Post** | Das Medium an einen Social-Media-Beitrag anhängen |
| **Animate** | Ein Video aus einem Standbild generieren (Image to Video) |
| **Delete** | Die hochgeladene Datei entfernen |

## Galerie-Verhalten

- Medien werden in einem responsiven Raster angezeigt, das sich an Ihre Bildschirmgröße anpasst
- Elemente sind nach Erstellungsdatum sortiert, neueste zuerst
- Die Galerie lädt automatisch weitere Elemente beim Scrollen (Infinite Scroll)
- Miniaturansichten sind für schnelles Laden optimiert
- Videos zeigen ein Wiedergabesymbol über ihrer Miniaturansicht
