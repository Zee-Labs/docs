---

Diktieren Sie Ihre Prompts per Sprache, statt zu tippen. Tippen Sie auf das Mikrofon-Symbol, sprechen Sie Ihre Idee, und Postzee transkribiert sie in das Prompt-Feld — bereit zur Generierung oder Verbesserung.

## So verwenden Sie die Spracheingabe

1. Öffnen Sie **Zee Studio** über die obere Navigationsleiste (klicken Sie auf **Zee Studio**) und wählen Sie einen beliebigen Erstellungsmodus (**AI Image**, **AI Video** oder **Create Post**)
2. Tippen Sie auf das **Mikrofon-Symbol** (🎤) im Erstellungspanel
3. **Erlauben Sie den Mikrofonzugriff**, wenn Ihr Browser danach fragt (nur beim ersten Mal)
4. **Sprechen Sie Ihren Prompt** — ein roter Aufnahme-Indikator zeigt an, dass aufgenommen wird
5. Wenn Sie fertig sind, tippen Sie erneut auf das Mikrofon-Symbol, um die Aufnahme zu beenden
6. Warten Sie 1–2 Sekunden, während das Audio transkribiert wird — ein bernsteinfarbener Indikator zeigt die laufende Transkription an
7. Ihre gesprochenen Worte erscheinen als Text im Prompt-Feld

## Aufnahme-Zustände

| Zustand | Indikator | Was passiert |
|---------|-----------|-------------|
| **Bereit** | Standard-Mikrofon-Symbol | Tippen, um die Aufnahme zu starten |
| **Aufnahme** | Rot pulsierender Indikator | Ihre Stimme wird aufgenommen |
| **Transkription** | Bernsteinfarbene animierte Punkte | Audio wird in Text umgewandelt |

## Funktioniert überall

Die Spracheingabe funktioniert sowohl auf Desktop als auch auf Mobilgeräten:

- **Desktop** — klicken Sie auf das Mikrofon-Symbol im Erstellungspanel
- **Mobilgerät** — tippen Sie auf das Mikrofon-Symbol (gleiche Position, für Touch optimiert)

<Note>
Ihr Browser muss den Mikrofonzugriff unterstützen. Die meisten modernen Browser (Chrome, Safari, Firefox, Edge) unterstützen dies. Stellen Sie sicher, dass Sie die Mikrofonberechtigungen erlauben, wenn Sie dazu aufgefordert werden.
</Note>

## Kombination mit dem Prompt Optimizer

Die Spracheingabe ergänzt sich perfekt mit dem Prompt Optimizer für den schnellsten Workflow:

1. **Sprechen** Sie Ihre Idee ungezwungen aus („ein Hund, der am Strand bei Sonnenuntergang läuft")
2. **Verbessern** — klicken Sie auf die Funkeln-Schaltfläche (✨), um Ihre ungezwungene Beschreibung in einen professionellen Prompt zu verwandeln
3. **Generieren** — klicken Sie auf Generate, um Ihr Bild oder Video zu erstellen

So gelangen Sie in Sekunden von der Idee zur Generierung, ohne ein einziges Wort zu tippen.

## Tipps

- **Sprechen Sie natürlich** — Sie müssen keine technischen Fotografie-Begriffe verwenden; der Prompt Optimizer übernimmt das
- **Seien Sie spezifisch** — „ein Golden-Retriever-Welpe, der an einem tropischen Strand spielt" liefert bessere Ergebnisse als „ein Hund"
- **Eine Idee pro Aufnahme** — beschreiben Sie eine einzelne Szene oder ein einzelnes Konzept pro Aufnahme
- **Ruhige Umgebung** — Aufnehmen in einer ruhigen Umgebung verbessert die Transkriptionsgenauigkeit
