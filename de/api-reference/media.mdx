Generieren Sie KI-Bilder und -Videos und laden Sie Dateien in Ihre Medienbibliothek hoch.

## Bild generieren

```
POST /public/v1/generate-image
```

Generieren Sie ein KI-Bild aus einem Text-Prompt. Dies ist eine **asynchrone Operation** — sie gibt eine Job-ID zurueck. Verwenden Sie den [Jobs-Endpunkt](/api-reference/jobs), um den Abschluss abzufragen.

**Rate-Limit:** 10 Anfragen pro Minute.

### Anfrage

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-pro",
    "prompt": "A professional headshot of a confident woman in a modern office"
  }' \
  https://api.postzee.app/public/v1/generate-image
```

### Parameter

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `model` | string | Ja | Modell-ID (siehe [Bildmodelle](/ai-models/image-models)) |
| `prompt` | string | Ja | Bildbeschreibung (mind. 5 Zeichen) |
| `aspectRatio` | string | Nein | `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `4:5`, `5:4` |
| `imageUrls` | string[] | Nein | Referenzbild-URLs fuer Bild-zu-Bild (max. 16) |
| `modelParams` | object | Nein | Modellspezifische Parameter (z.B. `quality`, `rendering_speed`) |

### Gueltige Modell-IDs

`nano-banana`, `nano-banana-2`, `nano-banana-pro`, `gpt-image-2`, `gpt-image-1.5`, `gpt-image-1`, `gpt-image-1-mini`, `flux-2-pro`, `flux-1.1-pro`, `flux-1.1-ultra`, `recraft-v3`, `recraft-v4`, `recraft-v4-pro`, `ideogram-v2`, `ideogram-v3`

### Antwort

```json
{
  "jobId": "task-uuid-here",
  "jobType": "task",
  "status": "processing"
}
```

## Video generieren

```
POST /public/v1/generate-video-ai
```

Generieren Sie ein KI-Video aus einem Text-Prompt. Dies ist eine **asynchrone Operation**.

**Rate-Limit:** 10 Anfragen pro Minute.

### Anfrage

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2-t2v",
    "prompt": "A golden sunset over a calm ocean with gentle waves",
    "duration": 10,
    "aspectRatio": "16:9"
  }' \
  https://api.postzee.app/public/v1/generate-video-ai
```

### Parameter

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `model` | string | Ja | Video-Modell-ID (siehe unten) |
| `prompt` | string | Ja | Videobeschreibung (5-10.000 Zeichen) |
| `duration` | number | Nein | Dauer in Sekunden (2-25) |
| `aspectRatio` | string | Nein | `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `4:5`, `5:4` |
| `resolution` | string | Nein | `360p`, `480p`, `540p`, `720p`, `1080p`, `4k` |
| `generateAudio` | boolean | Nein | Synchronisierten Ton generieren (Veo 3.1-Modelle) |
| `negativePrompt` | string | Nein | Was vermieden werden soll (max. 2.000 Zeichen) |
| `imageUrl` | string | Nein | Quellbild-URL fuer Bild-zu-Video-Modelle |

### Gueltige Video-Modell-IDs

**Sora 2:** `sora-2-t2v`, `sora-2-i2v`, `sora-2-t2v-pro`, `sora-2-i2v-pro`

**Veo 3.1:** `veo3.1`, `veo3.1/fast`, `veo3.1/image-to-video`, `veo3.1/fast/image-to-video`

**Kling:** `kling-3.0-pro`, `kling-2.6-pro`, `kling-2.5-turbo-pro`, `kling-o3-standard`, `kling-2.1-master`

**Luma Ray:** `luma-ray-2`, `luma-ray-2-flash`

**Seedance:** `seedance-2.0`, `seedance-1.0-pro`, `seedance-1.0-lite`

**Wan:** `wan-2.5`, `wan-2.1-pro`, `wan-2.1`, `wan-flf2v`

**Andere:** `vidu-q1`, `pixverse-v4.5`, `minimax-video-01`, `minimax-video-01-live`, `hunyuan-video`

### Antwort

```json
{
  "jobId": "task-uuid-here",
  "jobType": "task",
  "status": "processing"
}
```

## Batch generieren (1-4 Bilder)

```
POST /public/v1/generate-media
```

Generieren Sie mehrere KI-Bilder in einer einzigen Anfrage. Verwenden Sie dies fuer die Batch-Generierung anstelle mehrerer `generate-image`-Aufrufe.

**Rate-Limit:** 10 Anfragen pro Minute.

### Anfrage

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-pro",
    "prompt": "A professional headshot of a confident woman",
    "quantity": 4,
    "aspectRatio": "1:1"
  }' \
  https://api.postzee.app/public/v1/generate-media
```

### Parameter

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|:---:|--------------|
| `model` | string | Ja | Bildmodell-ID (siehe [Bildmodelle](/ai-models/image-models)) |
| `prompt` | string | Ja | Bildbeschreibung |
| `quantity` | number | Nein | Anzahl der Bilder (1-4, Standard: 1) |
| `aspectRatio` | string | Nein | `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `4:5`, `5:4` |
| `imageUrls` | string[] | Nein | Referenzbild-URLs (max. 5) |
| `modelParams` | object | Nein | Modellspezifische Parameter |

### Antwort

Fuer `quantity > 1` wird eine Gruppen-Job-ID zurueckgegeben:

```json
{
  "jobId": "group-uuid-here",
  "jobType": "group",
  "totalImages": 4,
  "status": "processing"
}
```

Fuer `quantity === 1` wird ein regulaerer Task zurueckgegeben:

```json
{
  "jobId": "task-uuid-here",
  "jobType": "task",
  "status": "processing"
}
```

Verwenden Sie den [Jobs-Endpunkt](/api-reference/jobs), um den Abschluss abzufragen. Bei Gruppen-Jobs enthaelt die Antwort ein `medias`-Array mit allen generierten Bildern und deren `order`-Feld.

## Datei hochladen

```
POST /public/v1/upload
```

Laden Sie eine Bild- oder Videodatei in Ihre Medienbibliothek hoch.

### Anfrage

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -F "file=@/path/to/image.jpg" \
  https://api.postzee.app/public/v1/upload
```

### Antwort

Gibt ein Medienobjekt mit `id`, `path` (URL), `type` (image/video) und `size` zurueck.

Verwenden Sie die `id` im Post-erstellen-Endpunkt, um hochgeladene Medien an einen Post anzuhaengen.
