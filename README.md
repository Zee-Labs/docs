# Postzee Documentation

Official documentation for [Postzee](https://postzee.app) — AI-powered social media management platform.

**Live:** [docs.postzee.app](https://docs.postzee.app)

## Development

```bash
npm install
npx mintlify dev --port 3004
```

Opens at [localhost:3004](http://localhost:3004).

## Structure

```
get-started/       → Introduction, quickstart, supported platforms
zee-studio/        → AI image/video generation, prompt optimizer, gallery
calendar/          → Scheduling, post editor, post lifecycle
channels/          → Connecting social accounts, platform settings
analytics/         → Engagement metrics
billing/           → Plans, credits, subscriptions
settings/          → Account, team, API keys, preferences
ai-models/         → Image & video model catalog, pricing
integrations/      → Channels detail, n8n, MCP/AI agents
api-reference/     → REST API documentation
```
