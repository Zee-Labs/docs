Postzee includes a built-in **MCP server** (Model Context Protocol) that lets AI agents interact with your Postzee account. Generate images, create videos, create HeyGen avatar videos, publish posts, and manage your social media — all through natural language conversation with your AI agent.

## What is MCP?

The [Model Context Protocol](https://modelcontextprotocol.io) is an open standard for connecting AI models to external tools and data sources. Postzee's MCP server exposes **20 tools** that any MCP-compatible AI agent can use — covering plan/credit awareness, model and platform discovery, cost estimation, pre-flight validation, generation, and posting.

## Supported AI agents

| Agent | Integration type | Setup time |
|-------|-----------------|------------|
| **Claude Code** | Skill (GitHub) | < 1 minute |
| **OpenClaw** | Skill (ClawHub) | < 1 minute |
| **Hermes Agent** | Skill (GitHub) | < 1 minute |
| **Any MCP client** | Direct HTTP connection | ~ 1 minute |

## What agents can do

With Postzee's MCP tools, an AI agent can:

- **Read full account context** — plan tier, credit balance, storage, channels, and feature availability in a single call (`POSTZEE_GET_CONTEXT`)
- **Discover plans and credit packages** — list all 5 subscription tiers and 5 one-time credit packs with current pricing
- **Pick the right model** — capability matrix for every image/video model (durations, resolutions, audio, accepted params, cost tier)
- **Read platform specs** — aspect ratios, max slides for carousels, caption limits, hashtag recommendations per network
- **Find the best posting time** — heuristic windows per channel, in the organization's timezone
- **Estimate cost** — credits required for a generation **before** spending them
- **Validate parameters** — pre-flight check that catches duration mismatches, missing image inputs, storage issues, and plan blocks
- **Generate images** — with reference photos, aspect ratio, quality, style, rendering speed, negative prompt, seed
- **Generate videos** — text-to-video, image-to-video, first-last-frame, with duration, resolution, audio control
- **Create HeyGen avatar videos** — AI avatars speaking your script (uses HeyGen credits)
- **Enhance prompts** — optimize prompts for better AI generation (free)
- **Create posts** — publish, schedule, or draft posts to any channel (carousels supported)

## How it works

```
You (conversation) → AI Agent → MCP Protocol → Postzee API → Social Platforms
```

1. You describe what you want in natural language (e.g., "Generate a sunset photo and post it to Instagram")
2. The AI agent uses Postzee's MCP tools to execute the task
3. Postzee handles the AI generation, scheduling, and publishing
4. The agent reports back with the results

## Connection details

| Property | Value |
|----------|-------|
| **Server name** | Postzee |
| **Skill version** | 3.0.0 |
| **Transport** | HTTP Streamable (recommended) + SSE (legacy) |
| **HTTP Endpoint** | `https://api.postzee.app/mcp/{apiKey}/http` |
| **SSE Endpoint** | `https://api.postzee.app/mcp/{apiKey}/sse` (legacy) |
| **Authentication** | API key (embedded in URL) |
| **Tools** | 20 |

## Requirements

- A **Postzee account** (free or paid — free users can buy AI credits)
- A **Postzee MCP URL** (copy from Settings → API Pública → MCP section)
- An **MCP-compatible AI agent** (Claude Code, OpenClaw, Hermes Agent, or any MCP client)
