Complete reference for all **20 tools** exposed by Postzee's MCP server. These tools are available to any MCP-compatible AI agent (Claude Code, OpenClaw, Hermes Agent, or direct MCP clients).

<Note>
All tools accept the API key embedded in your MCP URL. They are scoped to a single organization. **Costs for AI generation are always returned in credits, never in USD.** ($1 USD = 1,000 credits.)
</Note>

## Quick reference

| Category | Tools |
|----------|-------|
| **Context & Discovery** | `POSTZEE_GET_CONTEXT`, `POSTZEE_LIST_PLANS`, `POSTZEE_LIST_CREDIT_PACKAGES` |
| **Models & Specs** | `POSTZEE_LIST_MODELS_DETAILED`, `POSTZEE_LIST_PLATFORM_SPECS`, `POSTZEE_GET_BEST_POSTING_TIMES` |
| **Cost & Validation** | `POSTZEE_ESTIMATE_GENERATION_COST`, `POSTZEE_VALIDATE_GENERATION` |
| **Channels & Credits** | `POSTZEE_LIST_CHANNELS`, `POSTZEE_GET_CREDITS` |
| **Prompt Enhancement** | `POSTZEE_ENHANCE_PROMPT` |
| **Generation** | `POSTZEE_GENERATE_IMAGE`, `POSTZEE_GENERATE_VIDEO`, `POSTZEE_CHECK_JOB` |
| **HeyGen** | `POSTZEE_LIST_HEYGEN_AVATARS`, `POSTZEE_LIST_HEYGEN_VOICES`, `POSTZEE_GENERATE_HEYGEN_VIDEO` |
| **Posting** | `POSTZEE_CREATE_POST` |
| **Legacy (deprecated)** | `POSTZEE_LIST_IMAGE_MODELS`, `POSTZEE_LIST_VIDEO_MODELS` |

## Context & Discovery

Single-shot context aggregation. **Recommended first call of every session.** Returns plan, credits, storage, channels, features, and skill version in one round-trip.

```json
    "isLifetime": false,
    "period": "MONTHLY | YEARLY | null",

    "canPost": true,
    "canConnectChannels": true,
    "canUseAi": true,
    "channelsLimit": 5,
    "postsPerMonth": 400,

    "monthPriceUSD": 27,
    "yearPriceUSD": 252
  },
  "credits": {
    "available": 4500,
    "monthly": 2000,
    "purchased": 2500,
    "used": 0,
    "lastResetAt": "2026-04-15T10:00:00Z",
    "nextResetAt": "2026-05-15T10:00:00Z"
  },
  "storage": { "usedGB": 1.2, "limitGB": 15, "percentUsed": 8 },

    "connected": 5,
```

## Models & Specs

### POSTZEE_LIST_MODELS_DETAILED

Capability matrix for all available image and video models — durations, resolutions, audio capabilities, accepted custom params, and a relative cost tier.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `type` | string | No | `"image"`, `"video"`, or `"all"` (default) |
| `includeHeygen` | boolean | No | Include the synthetic HeyGen entry. Default: `true` |

**Returns (per model):**
```json
{
  "modelId": "sora-2-t2v-pro",
  "displayName": "Sora 2 Pro Text-to-Video",
  "type": "video",
  "audio": "speech-lipsync",
  "supportsImg2Img": false,
  "supportsImg2Vid": false,
  "supportsVector": false,
  "durations": [10, 15],
  "resolutions": ["720p", "1080p"],
  "aspectRatios": ["16:9", "9:16"],
  "customParamsAccepted": {
    "duration": { "type": "enum", "enum": [10, 15], "description": "..." },
    "quality": { "type": "enum", "enum": ["standard", "high"], "description": "..." }
  },
  "costTier": "high",
  "popular": true,
  "bestFor": ["cinematic dialogue", "native lip-sync", "1080p quality"],
  "notRecommendedFor": []
}
```

`costTier` is one of `very-low`, `low`, `mid`, `high`, `premium` — the absolute credit cost is intentionally not exposed; use `POSTZEE_ESTIMATE_GENERATION_COST` instead.

The synthetic `heygen-avatar-video` entry has `costTier: "external"` and a `billingNote` indicating that HeyGen bills on the user's HeyGen account, not Postzee credits.

### POSTZEE_LIST_PLATFORM_SPECS

Per-platform specs (aspect ratios, max slides for carousels, caption character limits, hashtag recommendations).

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `platforms` | string[] | No | Filter to specific platforms (e.g. `["instagram", "tiktok"]`). Omit for all. |

Covered platforms: instagram, facebook, linkedin, x, tiktok, youtube, pinterest, threads, bluesky, reddit.

### POSTZEE_GET_BEST_POSTING_TIMES

Curated heuristic best-posting-time windows per channel, returned in the organization's timezone.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `channelIds` | string[] | No | Filter to specific channels. Omit for all active channels. |

**Returns:**
```json
[
  {
    "channelId": "...",
    "platform": "instagram",
    "name": "My Instagram",
    "timezone": "America/Sao_Paulo",
    "bestTimes": [
      { "time": "11:00", "timezone": "America/Sao_Paulo", "nextDateLocal": "2026-05-06", "score": 90, "reason": "Lunchtime peak", "source": "heuristic" },
      { "time": "19:00", "timezone": "America/Sao_Paulo", "nextDateLocal": "2026-05-06", "score": 95, "reason": "Evening prime", "source": "heuristic" }
    ]
  }
]
```

Channels with `disabled` or `requiresReauth` status are filtered out — recommendations are only returned for channels that can actually post.

## Cost & Validation

### POSTZEE_ESTIMATE_GENERATION_COST

Single source of truth for cost estimates. **Always returns credits, never USD.**

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `type` | string | Yes | `"image"` or `"video"` |
| `model` | string | Yes | Model id from `POSTZEE_LIST_MODELS_DETAILED` |
| `duration` | number | No | Seconds (video, for per-second-priced models) |
| `resolution` | string | No | `"480p"`, `"540p"`, `"720p"`, `"1080p"`, `"4k"` |
| `quality` | string | No | Sora 2 Pro: `"standard"` (720p) or `"high"` (1080p) |
| `generateAudio` | boolean | No | Affects cost on Veo, Pixverse, Wan 2.5 |
| `voiceControl` | boolean | No | Kling 2.6/3.0 Pro |
| `aspectRatio` | string | No | Informational |

**Returns:**
```json
{
  "modelId": "sora-2-t2v-pro",
  "displayName": "Sora 2 Pro Text-to-Video",
  "estimatedCredits": 5000,
  "currentBalance": 4500,
  "willExceedBalance": true,
  "creditsAfter": 0,
  "shortfall": 500,
  "warnings": []
}
```

---

### POSTZEE_VALIDATE_GENERATION

Pre-flight validation — catches param errors, insufficient credits, storage issues, and plan blocks **before** burning credits.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `type` | string | Yes | `"image"` or `"video"` |
| `model` | string | Yes | Model id |
| `duration` | number | No | Seconds |
| `aspectRatio` | string | No | e.g. `"4:5"` |
| `resolution` | string | No | Output resolution |
| `quality` | string | No | Sora Pro `"standard"` / `"high"` |
| `generateAudio` | boolean | No | |
| `voiceControl` | boolean | No | |
| `imageUrl` | string | No | Required for image-to-video models |
| `slideCount` | number | No | For carousels — multiplies cost |

**Returns:**
```json
{
  "valid": true,
  "errors": [],
  "warnings": ["Storage at 85% — approaching limit."],
  "estimatedCreditsPerItem": 160,
  "totalEstimatedCredits": 1600,
  "slideCount": 10,
  "currentBalance": 4500,
  "willExceedBalance": false,
  "shortfall": 0
}
```

When `valid: false`, the `errors[]` array contains human-readable problems — e.g. duration not in the model's allowed set, missing `imageUrl` for an i2v model, storage 100%, or a model that's registered but not yet driveable through MCP.

## Channels & Credits

### POSTZEE_LIST_CHANNELS

List connected social media channels with explicit action flags.

**Parameters:** None

**Returns:**
```json
[
  {
    "id": "channel-uuid",
    "name": "My Instagram",
    "platform": "instagram",
    "picture": "https://...",
    "profile": "https://...",
    "disabled": false,
    "requiresReauth": false,
    "canPostNow": true,
    "actionRequired": "none",
    "actionUrl": null,
    "actionMessage": null,
    "status": "active"
  }
]
```

`actionRequired` is one of `"none"` (ready to post), `"reconnect"` (token expired — user reconnects), or `"reenable"` (admin/billing — user re-enables, **not** the same as reconnect).

```json
  "used": 0,
  "purchased": 2500
}
```

All values in credits ($1 USD = 1,000 credits).

## Generation

### POSTZEE_GENERATE_IMAGE

Generate an AI image from a text prompt. **Async operation** — returns a job id to poll with `POSTZEE_CHECK_JOB`.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `prompt` | string | Yes | Detailed image description |
| `model` | string | Yes | Model id from `POSTZEE_LIST_MODELS_DETAILED` (type `"image"`) |
| `aspectRatio` | string | No | e.g. `"1:1"`, `"4:5"`, `"9:16"`, `"16:9"`. Default: `"1:1"` |
| `imageUrls` | string[] | No | Reference image URLs for image-to-image |
| `quality` | string | No | GPT Image 2: `"low"`, `"medium"`, or `"high"`. Default: `"high"` |
| `style` | string | No | Recraft V3/V4: `"realistic_image"`, `"digital_illustration"`, `"vector_illustration"` |
| `renderingSpeed` | string | No | Ideogram V3 tier: `"turbo"`, `"balanced"`, or `"quality"` |
| `negativePrompt` | string | No | Things to exclude from the image |
| `seed` | number | No | Optional deterministic seed |

**Returns (success):**
```json
{ "success": true, "jobId": "job-uuid", "status": "processing", "message": "Image generation started..." }
```

**Returns (failure):**
```json
{ "success": false, "error": "unsupported_model | organization_not_found | generation_failed", "message": "..." }
```

---

### POSTZEE_GENERATE_VIDEO

Generate an AI video from a text prompt or reference image. **Async operation.**

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `prompt` | string | Yes | Detailed video description |
| `model` | string | Yes | Model id from `POSTZEE_LIST_MODELS_DETAILED` (type `"video"`) |
| `duration` | number | No | Seconds. Allowed values vary per model — check `model.durations` |
| `aspectRatio` | string | No | `"16:9"` or `"9:16"`. Default: `"16:9"` |
| `imageUrl` | string | No | Reference image URL for image-to-video models |
| `endImageUrl` | string | No | End frame URL for first-last-frame interpolation (e.g. `wan-flf2v`) |
| `resolution` | string | No | `"480p"`, `"540p"`, `"720p"`, `"1080p"`, `"4k"` (model-dependent) |
| `quality` | string | No | Sora 2 Pro only: `"standard"` (720p) or `"high"` (1080p) |
| `generateAudio` | boolean | No | Generate ambient audio (Veo 3.1, Pixverse, Wan 2.5). Affects credits. |
| `voiceControl` | boolean | No | Voice control (Kling 2.6/3.0 Pro). Affects credits. |
| `negativePrompt` | string | No | Things to avoid in the video |
| `seed` | number | No | Optional deterministic seed |

**Returns:** same shape as `POSTZEE_GENERATE_IMAGE`.

<Warning>
A few model ids in the catalog are not yet driveable through MCP and will be rejected upfront (Sora 2 storyboard variants, some Veo 3.1 paths). Use `POSTZEE_LIST_MODELS_DETAILED` to see currently supported models — it filters them out.
</Warning>

### POSTZEE_CHECK_JOB

Poll the status of an async image or video generation job.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `jobId` | string | Yes | Job id from `POSTZEE_GENERATE_*` |

**Returns (processing):**
```json
{ "jobId": "...", "status": "processing", "progress": 45 }
```

**Returns (success):**
```json
{ "jobId": "...", "status": "success", "progress": 100, "mediaUrl": "https://cdn.postzee.app/...", "mediaId": "media-uuid" }
```

**Returns (failed):**
```json
{ "jobId": "...", "status": "failed", "error": "Generation failed: ..." }
```

**Returns (not found):**
```json
{ "jobId": "...", "status": "not_found" }
```

### POSTZEE_GENERATE_HEYGEN_VIDEO

Create an avatar video with HeyGen. **Async operation** — poll with `POSTZEE_CHECK_JOB`.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `script` | string | Yes | Text for the avatar to speak (20-1500 chars) |
| `avatarId` | string | Yes | From `POSTZEE_LIST_HEYGEN_AVATARS` |

## Posting

### POSTZEE_CREATE_POST

Create, schedule, or publish a post to a social media channel.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:---:|-------------|
| `type` | string | Yes | `"now"` (publish immediately), `"draft"`, or `"schedule"` |
| `date` | string | Schedule only | UTC datetime (e.g. `"2026-04-24T15:00:00Z"`) — **required** when `type` is `"schedule"` |
| `channelId` | string | Yes | Channel id from `POSTZEE_LIST_CHANNELS` |
| `text` | string | Yes | Post text content |
| `mediaUrls` | string[] | No | Media URLs (in display order — array order = slide order for carousels) |

**Returns (success):**
```json
{ "success": true, "postId": "post-uuid", "message": "Post published successfully." }
```

**Returns (failure):**
```json
{ "success": false, "error": "<error_code>", "message": "...", "upgradeUrl": "https://dashboard.postzee.app/billing" }
```

**Possible `error` codes:**

| Code | Meaning |
|------|---------|
| `subscription_required` | The current plan doesn't allow publishing or the monthly post cap was reached. `upgradeUrl` is included. |
| `date_required` | `type: "schedule"` was sent without a `date`. |
| `invalid_date` | The `date` string is not a valid UTC ISO timestamp. |
| `organization_not_found` | The MCP key resolved to an organization that doesn't exist (rare). |
| `post_failed` | Provider/network error during publish. Retry once; if it persists, surface the `message`. |

## Legacy tools (deprecated)

The following tools still work but are superseded:

- **`POSTZEE_LIST_IMAGE_MODELS`** → use `POSTZEE_LIST_MODELS_DETAILED({ type: "image" })`
- **`POSTZEE_LIST_VIDEO_MODELS`** → use `POSTZEE_LIST_MODELS_DETAILED({ type: "video" })`

They will be kept for at least 90 days for backwards compatibility, then removed in a future major version.

## Async workflow pattern

Image, video, and HeyGen generation are async. The recommended workflow:

1. Validate first — `POSTZEE_VALIDATE_GENERATION` to catch param errors and insufficient-credit cases without burning credits
2. Estimate cost — `POSTZEE_ESTIMATE_GENERATION_COST` if you need the precise number to show the user
3. Enhance the prompt — `POSTZEE_ENHANCE_PROMPT` (free)
4. Generate — `POSTZEE_GENERATE_IMAGE` / `POSTZEE_GENERATE_VIDEO` / `POSTZEE_GENERATE_HEYGEN_VIDEO` → returns `jobId`
5. Poll — `POSTZEE_CHECK_JOB` every few seconds. Typical latencies: 10-60s for images, 30-180s for videos, up to 5 min for HeyGen
6. On success — use `mediaUrl` in `POSTZEE_CREATE_POST`

## Standard error format

All write/generate tools return:

```json
{
  "success": true | false,
  "error": "<machine_code>",  // only on failure
  "message": "<human readable>",
  "...": "tool-specific fields on success"
}
```

Branch on the `error` machine code for logic; use `message` to narrate to the user (translated to their language).

## Rate limits

| Endpoint | Limit |
|----------|-------|
| HTTP POST | 60 requests/minute |
| SSE connection (legacy) | 30 requests/minute |
| AI generation (`GENERATE_*`) | 10 requests/minute |
