Connect Postzee to **OpenClaw** to generate AI images/videos, create HeyGen avatar videos, and manage social media posts through your OpenClaw AI agent.

## Install the Postzee Skill

Install the Postzee skill from ClawHub:

```bash
clawhub install postzee
```

Or tell your OpenClaw agent directly:

> "Install the Postzee skill"

## Configure the MCP Connection

After installing the skill, connect to your Postzee account:

1. Go to [Postzee Settings](https://dashboard.postzee.app/settings) → **API Pública** tab
2. Copy the **MCP URL** from the "MCP (Model Context Protocol)" section
3. Tell your OpenClaw agent: "Configure Postzee with this MCP URL: `YOUR_MCP_URL`"

The agent will store the configuration automatically.

### Verify the connection

Tell your agent:

> "List my Postzee channels"

It should show your connected social media accounts.

## Available tools

Once installed, 12 tools are available to your OpenClaw agent:

| Tool | Description |
|------|-------------|
| `POSTZEE_LIST_CHANNELS` | List connected social media channels |
| `POSTZEE_GET_CREDITS` | Check AI credit balance |
| `POSTZEE_LIST_IMAGE_MODELS` | List available image models with costs |
| `POSTZEE_LIST_VIDEO_MODELS` | List available video models with costs |
| `POSTZEE_ENHANCE_PROMPT` | Optimize prompts for better results (free) |
| `POSTZEE_GENERATE_IMAGE` | Generate an AI image (supports reference images, aspect ratio, quality, style) |
| `POSTZEE_GENERATE_VIDEO` | Generate an AI video (supports image-to-video, duration, aspect ratio) |
| `POSTZEE_GENERATE_HEYGEN_VIDEO` | Create an avatar video with HeyGen AI |
| `POSTZEE_LIST_HEYGEN_AVATARS` | List available HeyGen avatars |
| `POSTZEE_LIST_HEYGEN_VOICES` | List available HeyGen voices |
| `POSTZEE_CHECK_JOB` | Check generation job status |
| `POSTZEE_CREATE_POST` | Create, schedule, or publish a post |

## Usage examples

Once configured, interact with Postzee through your OpenClaw agent:

- "Generate a professional product photo of a coffee cup"
- "Transform this photo into an oil painting" (send a photo via Telegram)
- "Animate my photo into a video for TikTok" (send a photo via Telegram)
- "Create a HeyGen avatar video presenting our company"
- "Post a sunset image to my LinkedIn with the caption 'Another beautiful day'"
- "How many credits do I have?"
- "Generate a 10-second video of ocean waves and post to YouTube"

### Sending reference images via Telegram

You can send a photo directly in the Telegram chat. The agent will use it automatically as a reference for image-to-image or image-to-video generation.
