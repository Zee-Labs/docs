Connect Postzee to **Claude Code** to generate AI images/videos, create HeyGen avatar videos, and publish to 30+ social media platforms — all through natural language conversation.

## Install the Postzee Skill

The easiest way to connect — install the Postzee skill from GitHub.

### Option 1: GitHub CLI

```bash
gh skill install Zee-Labs/postzee-skill
```

### Option 2: Manual installation

```bash
mkdir -p ~/.claude/skills/postzee
curl -o ~/.claude/skills/postzee/SKILL.md https://raw.githubusercontent.com/Zee-Labs/postzee-skill/main/SKILL.md
```

## Configure the MCP Connection

After installing the skill, connect to your Postzee account:

1. Go to [Postzee Settings](https://dashboard.postzee.app/settings) → **API Pública** tab
2. Copy the **MCP URL** from the "MCP (Model Context Protocol)" section
3. Run the following command in Claude Code:

```bash
claude mcp add postzee YOUR_MCP_URL
```

Replace `YOUR_MCP_URL` with the full URL you copied (it looks like `https://api.postzee.app/mcp/.../http`).

### Verify the connection

```
List my Postzee channels
```

Claude Code should call the `POSTZEE_LIST_CHANNELS` tool and show your connected social media accounts.

## Usage examples

Once connected, use natural language to interact with Postzee:

### Generate an image

> "Generate a professional headshot of a woman in a modern office"

### Transform a photo (image-to-image)

> "Transform this photo into anime style" (provide an image URL)

### Animate a photo into video

> "Animate this photo into a cinematic video for Instagram Reels"

### Create a HeyGen avatar video

> "Create a HeyGen video of an avatar presenting our new product"

### Generate and post

> "Generate a sunset beach photo and post it to my Instagram with the caption 'Golden hour vibes'"

### Check your credits

> "How many AI credits do I have left?"

## Recommended workflow

For best results:

1. **Check credits** — verify you have enough credits
2. **Enhance prompt** — use the prompt optimizer for better results (free)
3. **Generate** — create the image or video
4. **Post** — publish or schedule to your channels
