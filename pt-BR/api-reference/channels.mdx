Recupere seus canais de redes sociais conectados.

## Listar canais

```
GET /public/v1/channels
```

Retorna todos os canais de redes sociais conectados da sua conta.

### Requisição

```bash
curl -H "Authorization: YOUR_API_KEY" \
  https://api.postzee.app/public/v1/channels
```

### Resposta

```json
[
  {
    "id": "abc123-def456",
    "name": "My Instagram",
    "platform": "instagram",
    "picture": "https://...",
    "disabled": false,
    "profile": "username",
    "customer": {
      "id": "cust-123",
      "name": "Cliente A"
    }
  },
  {
    "id": "ghi789-jkl012",
    "name": "Company LinkedIn",
    "platform": "linkedin-page",
    "picture": "https://...",
    "disabled": false,
    "profile": "Company Name",
    "customer": null
  }
]
```

### Campos da resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | string | ID único do canal — use em Create Post |
| `name` | string | Nome de exibição do canal |
| `platform` | string | Identificador da plataforma (ex.: `instagram`, `linkedin-page`, `tiktok`, `x`) |
| `picture` | string | URL da foto de perfil |
| `disabled` | boolean | Se o canal está desativado |
| `profile` | string | Nome de usuário ou nome da página na plataforma |
| `customer` | object/null | Agrupamento de cliente (se atribuído) |

<Note>
O campo `id` é o que você passa como `channel.id` ao criar posts. O campo `platform` indica a qual rede este canal pertence.
</Note>

## Obter um único canal

```
GET /public/v1/channels/:id
```

Recupera os metadados de um canal específico. Retorna `404` se o canal não pertencer à sua organização.

```bash
curl -H "Authorization: YOUR_API_KEY" \
  https://api.postzee.app/public/v1/channels/abc123-def456
```

O formato da resposta é idêntico a uma entrada individual do endpoint de listagem.
