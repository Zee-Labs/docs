Sua biblioteca pessoal de mídia no Zee Studio é organizada em duas abas: **Minhas Criações** para conteúdo gerado por IA e **Minhas Mídias** para arquivos enviados manualmente.

## Minhas Criações

A aba **Minhas Criações** mostra todas as imagens e vídeos que você gerou com IA. Cada item exibe:

- Uma prévia em miniatura
- O modelo de IA usado para a geração
- Há quanto tempo foi criado (ex.: "2 horas atrás")

### Filtrando suas criações

Use os filtros acima da galeria para refinar sua visualização:

| Filtro | Mostra |
|--------|--------|
| **Todos** | Todas as mídias geradas |
| **Imagens** | Apenas imagens geradas por IA |
| **Vídeos** | Apenas vídeos gerados por IA |

### Ações nas suas criações

Clique em qualquer item em Minhas Criações para acessar estas ações:

| Ação | Descrição |
|------|-----------|
| **Baixar** | Salvar o arquivo no seu dispositivo (JPG para imagens, MP4 para vídeos) |
| **Criar Post** | Abrir o editor de post com esta mídia anexada |
| **Recriar** | Carregar o mesmo modelo e configurações para gerar uma nova variação |
| **Usar Prompt** | Copiar o prompt original de volta para o campo de criação |
| **Ver Detalhes** | Ver metadados completos da geração — modelo, prompt, configurações, resolução |
| **Editar Imagem** | Abrir a imagem no editor integrado para modificações (apenas imagens) |
| **Animar** | Transformar uma imagem estática em vídeo usando o modo Imagem para Vídeo (apenas imagens) |
| **Publicar** | Compartilhar na galeria pública Explore para a comunidade descobrir |
| **Excluir** | Remover a mídia permanentemente (com confirmação) |

### Gerações pendentes

Quando uma geração está em andamento, ela aparece em Minhas Criações com um indicador de carregamento. Você pode:

- Ver o status do progresso (na fila, processando)
- Continuar trabalhando em outras tarefas enquanto espera
- Fechar o Zee Studio e voltar depois — o item estará pronto quando você retornar

## Minhas Mídias

A aba **Minhas Mídias** mostra arquivos que você enviou manualmente. Use para suas próprias fotos, ativos de marca, logos ou qualquer mídia que você queira usar em posts.

### Como enviar arquivos

- **Arrastar e soltar** — arraste arquivos diretamente para a área da galeria
- **Clique para navegar** — clique na zona de upload para abrir o seletor de arquivos
- **Botão de upload** — use o botão de upload na barra de ferramentas da galeria

Veja [Enviar Mídia](/zee-studio/upload) para formatos suportados e limites de tamanho.

### Ações nas mídias enviadas

| Ação | Descrição |
|------|-----------|
| **Baixar** | Salvar o arquivo no seu dispositivo |
| **Criar Post** | Anexar a mídia a um post de redes sociais |
| **Animar** | Gerar um vídeo a partir de uma imagem estática (Imagem para Vídeo) |
| **Excluir** | Remover o arquivo enviado |

## Comportamento da galeria

- A mídia é exibida em uma grade responsiva que se adapta ao tamanho da sua tela
- Os itens são ordenados por data de criação, mais recentes primeiro
- A galeria carrega mais itens automaticamente conforme você rola (scroll infinito)
- As miniaturas são otimizadas para carregamento rápido
- Vídeos mostram um ícone de reprodução sobre a miniatura
