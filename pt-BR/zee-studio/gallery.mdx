Sua biblioteca pessoal de mídia no Zee Studio é organizada em duas abas: **Minhas Criações** para conteúdo gerado por IA e **Minha Mídia** para arquivos enviados manualmente.

## Minhas Criações

A aba **Minhas Criações** mostra todas as imagens e vídeos gerados com IA. Cada item exibe:

- Uma miniatura de pré-visualização
- O modelo de IA usado para a geração
- Há quanto tempo foi criado (por exemplo, "há 2 horas")

### Filtrando suas criações

Use os filtros acima da galeria para refinar sua visualização:

| Filtro | Mostra |
|--------|-------|
| **Tudo** | Todas as mídias geradas |
| **Imagens** | Apenas imagens geradas por IA |
| **Vídeos** | Apenas vídeos gerados por IA |

### Ações nas suas criações

Clique em qualquer item em Minhas Criações para acessar estas ações:

| Ação | Descrição |
|--------|-------------|
| **Baixar** | Salve o arquivo no seu dispositivo (JPG para imagens, MP4 para vídeos) |
| **Criar post** | Abra o editor de post com esta mídia anexada |
| **Recriar** | Carregue o mesmo modelo e configurações para gerar uma nova variação |
| **Usar prompt** | Copie o prompt original de volta para o campo de criação |
| **Ver detalhes** | Veja todos os metadados de geração — modelo, prompt, configurações, resolução |
| **Editar imagem** | Abra a imagem no editor integrado para modificações (somente imagens) |
| **Animar** | Transforme uma imagem estática em vídeo com o modo Imagem para vídeo (somente imagens) |
| **Publicar** | Compartilhe na galeria pública Explorar para a comunidade descobrir |
| **Excluir** | Remova permanentemente a mídia (com confirmação) |

### Gerações pendentes

Quando uma geração está em andamento, ela aparece em Minhas Criações com um indicador de carregamento. Você pode:

- Ver o status do progresso (na fila, processando)
- Continuar trabalhando em outras tarefas enquanto espera
- Fechar o Zee Studio e voltar mais tarde — o item estará pronto quando você retornar

## Minha Mídia

A aba **Minha Mídia** mostra os arquivos que você enviou manualmente. Use-a para suas próprias fotos, ativos de marca, logos ou qualquer mídia que queira utilizar em posts.

### Como fazer upload

- **Arrastar e soltar** — arraste arquivos diretamente para a área da galeria
- **Clique para procurar** — clique na zona de upload para abrir seu seletor de arquivos
- **Botão de upload** — use o botão de upload na barra de ferramentas da galeria

Veja [Enviar mídia](/zee-studio/upload) para formatos suportados e limites de tamanho.

### Ações nas mídias enviadas

| Ação | Descrição |
|--------|-------------|
| **Baixar** | Salve o arquivo no seu dispositivo |
| **Criar post** | Anexe a mídia a um post de rede social |
| **Animar** | Gere um vídeo a partir de uma imagem estática (Imagem para vídeo) |
| **Excluir** | Remova o arquivo enviado |

## Comportamento da galeria

- A mídia é exibida em uma grade responsiva que se adapta ao tamanho da tela
- Os itens são ordenados por data de criação, mais recentes primeiro
- A galeria carrega mais itens automaticamente conforme você rola (rolagem infinita)
- As miniaturas são otimizadas para carregamento rápido
- Os vídeos exibem um ícone de play sobreposto na miniatura
