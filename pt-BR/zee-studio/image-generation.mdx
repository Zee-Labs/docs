Gere imagens profissionais com IA diretamente no Zee Studio. Escolha entre 16 modelos otimizados para diferentes estilos — fotorrealismo, logos, arte vetorial, ilustrações e muito mais.

## Como gerar uma imagem

1. Abra o **Zee Studio** pela barra de navegação superior (clique em **Zee Studio**)
2. Selecione o modo **AI Image** no painel de criação
3. Escolha seu modo de geração:
   - **Texto para Imagem** — descreva o que você quer no campo de prompt
   - **Imagem para Imagem** — anexe uma imagem de referência e descreva as alterações desejadas
4. Digite seu prompt (ex.: "Um retrato profissional de uma mulher confiante em um escritório moderno")
5. *(Opcional)* Clique no botão **Aprimorar com IA** (✨) para otimizar seu prompt — isso melhora drasticamente os resultados
6. Clique no chip do modelo para selecionar um modelo de IA e ajustar as configurações
7. Verifique o custo de créditos exibido no botão **Gerar**
8. Clique em **Gerar**
9. Aguarde 10-30 segundos — sua imagem aparece em **Criações** quando estiver pronta

## Configurações do modelo

Clique no chip do modelo (ex.: **Nano Banana Pro**) para abrir o painel de configurações:

- **Seletor de modelo** — navegue e troque entre os modelos disponíveis
- **Proporção** — escolha entre paisagem (16:9), retrato (9:16), quadrado (1:1) e outros formatos
- **Qualidade/Velocidade** — alguns modelos oferecem níveis de qualidade (ex.: GPT Image: Baixa, Média, Alta)
- **Estilo** — alguns modelos suportam presets de estilo (ex.: Ideogram: realista, design, 3D, anime)
- **Prompt Negativo** — descreva o que você não quer na imagem (suportado por alguns modelos)

## Usando imagens de referência

No modo **Imagem para Imagem**:

1. Clique no ícone de imagem no painel de criação para anexar uma imagem de referência
2. O modelo de IA usará sua imagem como ponto de partida
3. Descreva as alterações desejadas no prompt (ex.: "Transforme isso em uma pintura em aquarela")

<Note>
Nem todos os modelos suportam imagens de referência. Verifique o painel de configurações do modelo — modelos com suporte a referência mostram uma opção de anexar imagem.
</Note>

## Gerando múltiplas imagens de uma vez

Precisa de variações do mesmo prompt? Gere até **4 imagens de uma vez** em um único clique.

1. No painel de criação, defina a **quantidade** para 2, 3 ou 4
2. Clique em **Gerar**
3. Todas as imagens aparecem na sua galeria como um único lote

Cada imagem conta como uma geração — seus créditos são deduzidos proporcionalmente. Gerar 4 imagens custa 4× o custo em créditos de uma imagem.

<Note>
A geração em lote é ótima para escolher a melhor variação. A IA gera cada imagem independentemente com o mesmo prompt, então você terá diferentes interpretações da sua ideia.
</Note>

## Dicas para melhores resultados

- **Seja específico** — "Um filhote de golden retriever em uma praia ao pôr do sol" funciona melhor que "cachorro na praia"
- **Use o Otimizador de Prompt** — ele adiciona automaticamente detalhes profissionais de fotografia (iluminação, câmera, composição)
- **Mencione o estilo** — "fotografia editorial Vogue", "still de cinema cinematográfico", "design flat minimalista"
- **Especifique a qualidade** — "ultra-fotorrealista 8K", "iluminação profissional de estúdio"
- **Experimente diferentes modelos** — cada modelo tem pontos fortes. Nano Banana se destaca em fotos, Recraft em design, Ideogram em texto

## Após a geração

Quando sua imagem estiver pronta, clique nela para:

- **Baixar** — salvar no seu dispositivo
- **Criar Post** — abrir o editor de post com a imagem anexada
- **Recriar** — reutilizar o mesmo modelo e configurações com um novo prompt
- **Usar prompt** — copiar o prompt para o campo de criação
- **Animar** — transformar a imagem em vídeo (modo Imagem para Vídeo)
- **Publicar** — compartilhar na galeria pública Explore
