Referência completa para todas as **20 ferramentas** expostas pelo servidor MCP do Postzee. Essas ferramentas estão disponíveis para qualquer agente IA compatível com MCP (Claude Code, OpenClaw, Hermes Agent ou clientes MCP diretos).

<Note>
Todas as ferramentas aceitam a chave de API embutida na sua URL MCP. Elas são escopadas a uma única organização. **Custos de geração de IA são sempre retornados em créditos, nunca em dólares.** ($1 USD = 1.000 créditos.)
</Note>

## Referência rápida

| Categoria | Ferramentas |
|-----------|-------------|
| **Contexto e Descoberta** | `POSTZEE_GET_CONTEXT`, `POSTZEE_LIST_PLANS`, `POSTZEE_LIST_CREDIT_PACKAGES` |
| **Modelos e Specs** | `POSTZEE_LIST_MODELS_DETAILED`, `POSTZEE_LIST_PLATFORM_SPECS`, `POSTZEE_GET_BEST_POSTING_TIMES` |
| **Custo e Validação** | `POSTZEE_ESTIMATE_GENERATION_COST`, `POSTZEE_VALIDATE_GENERATION` |
| **Canais e Créditos** | `POSTZEE_LIST_CHANNELS`, `POSTZEE_GET_CREDITS` |
| **Otimização de Prompt** | `POSTZEE_ENHANCE_PROMPT` |
| **Geração** | `POSTZEE_GENERATE_IMAGE`, `POSTZEE_GENERATE_VIDEO`, `POSTZEE_CHECK_JOB` |
| **HeyGen** | `POSTZEE_LIST_HEYGEN_AVATARS`, `POSTZEE_LIST_HEYGEN_VOICES`, `POSTZEE_GENERATE_HEYGEN_VIDEO` |
| **Postagem** | `POSTZEE_CREATE_POST` |
| **Legacy (deprecadas)** | `POSTZEE_LIST_IMAGE_MODELS`, `POSTZEE_LIST_VIDEO_MODELS` |

## Contexto e Descoberta

### POSTZEE_GET_CONTEXT

Agregação de contexto em uma única chamada. **Recomendada como primeira chamada de toda sessão.** Retorna plano, créditos, storage, canais, features e versão da skill em um round-trip.

**Parâmetros:** Nenhum

    "tier": "FREE | STANDARD | TEAM | PRO | ULTIMATE",
    "isLifetime": false,
    "period": "MONTHLY | YEARLY | null",
    "cancelAt": "string | null",
    "canPost": true,
    "canConnectChannels": true,
    "canUseAi": true,
    "channelsLimit": 5,
    "postsPerMonth": 400,
    "postsRemaining": 380,
    "monthlyCredits": 2000,
    "storageLimitGB": 15,
    "monthPriceUSD": 27,
    "yearPriceUSD": 252
  },
  "credits": {
    "available": 4500,
    "monthly": 2000,
    "purchased": 2500,
    "used": 0,
    "lastResetAt": "2026-04-15T10:00:00Z",
    "nextResetAt": "2026-05-15T10:00:00Z"
  },
  "storage": { "usedGB": 1.2, "limitGB": 15, "percentUsed": 8 },
  "channels": {
    "connected": 5,
    "active": 4,

    "disabled": 0,

  },

## Modelos e Specs

### POSTZEE_LIST_MODELS_DETAILED

Matriz de capacidades para todos os modelos de imagem e vídeo disponíveis — durações, resoluções, capacidades de áudio, parâmetros customizados aceitos, e tier de custo relativo.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `type` | string | Não | `"image"`, `"video"`, ou `"all"` (default) |
| `includeHeygen` | boolean | Não | Inclui a entrada sintética HeyGen. Default: `true` |

**Retorna (por modelo):**
```json
{
  "modelId": "sora-2-t2v-pro",
  "displayName": "Sora 2 Pro Text-to-Video",
  "type": "video",
  "audio": "speech-lipsync",
  "supportsImg2Img": false,
  "supportsImg2Vid": false,
  "supportsVector": false,
  "durations": [10, 15],
  "resolutions": ["720p", "1080p"],
  "aspectRatios": ["16:9", "9:16"],
  "customParamsAccepted": {
    "duration": { "type": "enum", "enum": [10, 15], "description": "..." },
    "quality": { "type": "enum", "enum": ["standard", "high"], "description": "..." }
  },
  "costTier": "high",
  "popular": true,
  "bestFor": ["cinematic dialogue", "native lip-sync", "1080p quality"],
  "notRecommendedFor": []
}
```

`costTier` é um de `very-low`, `low`, `mid`, `high`, `premium` — o custo absoluto em créditos não é exposto intencionalmente; use `POSTZEE_ESTIMATE_GENERATION_COST` em vez disso.

A entrada sintética `heygen-avatar-video` tem `costTier: "external"` e um `billingNote` indicando que o HeyGen cobra na conta HeyGen do usuário, não em créditos Postzee.

### POSTZEE_LIST_PLATFORM_SPECS

Specs por plataforma (aspect ratios, máximo de slides para carrosséis, limites de caracteres em legendas, recomendações de hashtag).

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `platforms` | string[] | Não | Filtra para plataformas específicas (ex.: `["instagram", "tiktok"]`). Omitir para todas. |

Plataformas cobertas: instagram, facebook, linkedin, x, tiktok, youtube, pinterest, threads, bluesky, reddit.

### POSTZEE_GET_BEST_POSTING_TIMES

Janelas heurísticas dos melhores horários de postagem por canal, retornadas no fuso horário da organização.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `channelIds` | string[] | Não | Filtra para canais específicos. Omitir para todos os canais ativos. |

**Retorna:**
```json
[
  {
    "channelId": "...",
    "platform": "instagram",
    "name": "My Instagram",
    "timezone": "America/Sao_Paulo",
    "bestTimes": [
      { "time": "11:00", "timezone": "America/Sao_Paulo", "nextDateLocal": "2026-05-06", "score": 90, "reason": "Lunchtime peak", "source": "heuristic" },
      { "time": "19:00", "timezone": "America/Sao_Paulo", "nextDateLocal": "2026-05-06", "score": 95, "reason": "Evening prime", "source": "heuristic" }
    ]
  }
]
```

Canais com status `disabled` ou `requiresReauth` são filtrados — recomendações só aparecem para canais que de fato podem postar.

## Custo e Validação

### POSTZEE_ESTIMATE_GENERATION_COST

Fonte única de verdade para estimativas de custo. **Sempre retorna créditos, nunca dólares.**

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `type` | string | Sim | `"image"` ou `"video"` |
| `model` | string | Sim | ID do modelo de `POSTZEE_LIST_MODELS_DETAILED` |
| `duration` | number | Não | Segundos (vídeo, para modelos com pricing por segundo) |
| `resolution` | string | Não | `"480p"`, `"540p"`, `"720p"`, `"1080p"`, `"4k"` |
| `quality` | string | Não | Sora 2 Pro: `"standard"` (720p) ou `"high"` (1080p) |
| `generateAudio` | boolean | Não | Afeta custo em Veo, Pixverse, Wan 2.5 |
| `voiceControl` | boolean | Não | Kling 2.6/3.0 Pro |
| `aspectRatio` | string | Não | Informativo |

**Retorna:**
```json
{
  "modelId": "sora-2-t2v-pro",
  "displayName": "Sora 2 Pro Text-to-Video",
  "estimatedCredits": 5000,
  "currentBalance": 4500,
  "willExceedBalance": true,
  "creditsAfter": 0,
  "shortfall": 500,
  "warnings": []
}
```

---

### POSTZEE_VALIDATE_GENERATION

Validação pré-execução — captura erros de parâmetros, créditos insuficientes, problemas de storage e bloqueios de plano **antes** de queimar créditos.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `type` | string | Sim | `"image"` ou `"video"` |
| `model` | string | Sim | ID do modelo |
| `duration` | number | Não | Segundos |
| `aspectRatio` | string | Não | ex.: `"4:5"` |
| `resolution` | string | Não | Resolução de saída |
| `quality` | string | Não | Sora Pro `"standard"` / `"high"` |
| `generateAudio` | boolean | Não | |
| `voiceControl` | boolean | Não | |
| `imageUrl` | string | Não | Obrigatório para modelos image-to-video |
| `slideCount` | number | Não | Para carrosséis — multiplica o custo |

**Retorna:**
```json
{
  "valid": true,
  "errors": [],
  "warnings": ["Storage at 85% — approaching limit."],
  "estimatedCreditsPerItem": 160,
  "totalEstimatedCredits": 1600,
  "slideCount": 10,
  "currentBalance": 4500,
  "willExceedBalance": false,
  "shortfall": 0
}
```

Quando `valid: false`, o array `errors[]` contém problemas legíveis — ex.: duração não está no conjunto permitido pelo modelo, falta `imageUrl` para modelo i2v, storage em 100%, ou modelo registrado mas não disponível ainda via MCP.

## Canais e Créditos

### POSTZEE_LIST_CHANNELS

Lista canais de redes sociais conectados com flags de ação explícitas.

**Parâmetros:** Nenhum

**Retorna:**
```json
[
  {
    "id": "channel-uuid",
    "name": "My Instagram",
    "platform": "instagram",
    "picture": "https://...",
    "profile": "https://...",
    "disabled": false,
    "requiresReauth": false,
    "canPostNow": true,
    "actionRequired": "none",
    "actionUrl": null,
    "actionMessage": null,
    "status": "active"
  }
]
```

`actionRequired` é um de `"none"` (pronto para postar), `"reconnect"` (token expirado — usuário reconecta), ou `"reenable"` (admin/billing — usuário reativa, **não** o mesmo que reconectar).

---

### POSTZEE_GET_CREDITS

Saldo de créditos leve. Subset de `POSTZEE_GET_CONTEXT.credits` — prefira o último para código novo.

**Parâmetros:** Nenhum

**Retorna:**
```json
{
  "available": 4500,
  "monthly": 2000,
  "used": 0,
  "purchased": 2500
}
```

Todos os valores em créditos ($1 USD = 1.000 créditos).

## Otimização de Prompt

### POSTZEE_ENHANCE_PROMPT

Otimiza o prompt do usuário para melhores resultados de geração de IA. **Grátis** — sem custo de créditos.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `prompt` | string | Sim | O prompt do usuário a ser otimizado |
| `mediaType` | string | Sim | `"image"` ou `"video"` |
| `model` | string | Não | ID do modelo alvo (para otimização específica do modelo) |

**Retorna:**
```json
{ "success": true, "enhancedPrompt": "A young woman in a tailored trench coat walks through..." }
```

## Geração

### POSTZEE_GENERATE_IMAGE

Gera uma imagem IA a partir de um prompt. **Operação assíncrona** — retorna um job id para fazer poll com `POSTZEE_CHECK_JOB`.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `prompt` | string | Sim | Descrição detalhada da imagem |
| `model` | string | Sim | ID do modelo de `POSTZEE_LIST_MODELS_DETAILED` (type `"image"`) |
| `aspectRatio` | string | Não | ex.: `"1:1"`, `"4:5"`, `"9:16"`, `"16:9"`. Default: `"1:1"` |
| `imageUrls` | string[] | Não | URLs de imagens de referência para image-to-image |
| `quality` | string | Não | GPT Image 2: `"low"`, `"medium"` ou `"high"`. Default: `"high"` |
| `style` | string | Não | Recraft V3/V4: `"realistic_image"`, `"digital_illustration"`, `"vector_illustration"` |
| `renderingSpeed` | string | Não | Tier do Ideogram V3: `"turbo"`, `"balanced"` ou `"quality"` |
| `negativePrompt` | string | Não | Coisas a excluir da imagem |
| `seed` | number | Não | Seed determinística opcional |

**Retorna (sucesso):**
```json
{ "success": true, "jobId": "job-uuid", "status": "processing", "message": "Image generation started..." }
```

**Retorna (falha):**
```json
{ "success": false, "error": "unsupported_model | organization_not_found | generation_failed", "message": "..." }
```

---

### POSTZEE_GENERATE_VIDEO

Gera um vídeo IA a partir de um prompt ou imagem de referência. **Operação assíncrona.**

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `prompt` | string | Sim | Descrição detalhada do vídeo |
| `model` | string | Sim | ID do modelo de `POSTZEE_LIST_MODELS_DETAILED` (type `"video"`) |
| `duration` | number | Não | Segundos. Valores permitidos variam por modelo — consulte `model.durations` |
| `aspectRatio` | string | Não | `"16:9"` ou `"9:16"`. Default: `"16:9"` |
| `imageUrl` | string | Não | URL de imagem de referência para modelos image-to-video |
| `endImageUrl` | string | Não | URL do frame final para interpolação first-last-frame (ex.: `wan-flf2v`) |
| `resolution` | string | Não | `"480p"`, `"540p"`, `"720p"`, `"1080p"`, `"4k"` (depende do modelo) |
| `quality` | string | Não | Apenas Sora 2 Pro: `"standard"` (720p) ou `"high"` (1080p) |
| `generateAudio` | boolean | Não | Gera áudio ambiente (Veo 3.1, Pixverse, Wan 2.5). Afeta créditos. |
| `voiceControl` | boolean | Não | Controle de voz (Kling 2.6/3.0 Pro). Afeta créditos. |
| `negativePrompt` | string | Não | Coisas a evitar no vídeo |
| `seed` | number | Não | Seed determinística opcional |

**Retorna:** mesma estrutura de `POSTZEE_GENERATE_IMAGE`.

<Warning>
Alguns IDs de modelo no catálogo ainda não estão disponíveis via MCP e serão rejeitados de antemão (variantes Sora 2 storyboard, alguns paths Veo 3.1). Use `POSTZEE_LIST_MODELS_DETAILED` para ver os modelos atualmente suportados — ele filtra esses casos.
</Warning>

### POSTZEE_CHECK_JOB

Faz poll do status de um job assíncrono de geração de imagem ou vídeo.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `jobId` | string | Sim | Job id de `POSTZEE_GENERATE_*` |

**Retorna (processando):**
```json
{ "jobId": "...", "status": "processing", "progress": 45 }
```

**Retorna (sucesso):**
```json
{ "jobId": "...", "status": "success", "progress": 100, "mediaUrl": "https://cdn.postzee.app/...", "mediaId": "media-uuid" }
```

**Retorna (falha):**
```json
{ "jobId": "...", "status": "failed", "error": "Generation failed: ..." }
```

**Retorna (não encontrado):**
```json
{ "jobId": "...", "status": "not_found" }
```

<Note>
Ferramentas HeyGen exigem uma chave de API HeyGen configurada na sua conta Postzee. Configure em [Settings → integração HeyGen](https://dashboard.postzee.app/settings). Geração de vídeo HeyGen **cobra na conta HeyGen do usuário, não em créditos Postzee.**
</Note>

### POSTZEE_LIST_HEYGEN_AVATARS

Lista avatares HeyGen disponíveis (filtre por gênero, idade, estilo etc. no lado do cliente).

**Parâmetros:** Nenhum
**Retorna:** Array de avatares da sua conta HeyGen.

---

### POSTZEE_LIST_HEYGEN_VOICES

Lista vozes HeyGen disponíveis.

**Parâmetros:** Nenhum
**Retorna:** Array de vozes da sua conta HeyGen, com info de idioma e gênero.

### POSTZEE_GENERATE_HEYGEN_VIDEO

Cria um vídeo de avatar com HeyGen. **Operação assíncrona** — faça poll com `POSTZEE_CHECK_JOB`.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `script` | string | Sim | Texto para o avatar falar (20-1500 caracteres) |
| `avatarId` | string | Sim | De `POSTZEE_LIST_HEYGEN_AVATARS` |
| `voiceId` | string | Sim | De `POSTZEE_LIST_HEYGEN_VOICES` |
| `aspectRatio` | string | Não | `"16:9"` ou `"9:16"`. Default: `"16:9"` |

**Retorna:** mesma estrutura de `POSTZEE_GENERATE_IMAGE` — `jobId` + `status: "processing"`. A resposta também avisa que créditos HeyGen (não Postzee) serão consumidos.

## Postagem

### POSTZEE_CREATE_POST

Cria, agenda ou publica um post em um canal de rede social.

**Parâmetros:**

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:---:|-----------|
| `type` | string | Sim | `"now"` (publica imediatamente), `"draft"` ou `"schedule"` |
| `date` | string | Apenas schedule | UTC datetime (ex.: `"2026-04-24T15:00:00Z"`) — **obrigatório** quando `type` é `"schedule"` |
| `channelId` | string | Sim | ID do canal de `POSTZEE_LIST_CHANNELS` |
| `text` | string | Sim | Conteúdo do post |
| `mediaUrls` | string[] | Não | URLs das mídias (em ordem de exibição — ordem do array = ordem dos slides em carrosséis) |

**Retorna (sucesso):**
```json
{ "success": true, "postId": "post-uuid", "message": "Post published successfully." }
```

**Retorna (falha):**
```json
{ "success": false, "error": "<error_code>", "message": "...", "upgradeUrl": "https://dashboard.postzee.app/billing" }
```

**Códigos de `error` possíveis:**

| Código | Significado |
|--------|-------------|
| `subscription_required` | O plano atual não permite publicar ou o limite mensal de posts foi atingido. `upgradeUrl` é incluído. |
| `date_required` | `type: "schedule"` foi enviado sem `date`. |
| `invalid_date` | A string `date` não é um timestamp UTC ISO válido. |
| `organization_not_found` | A chave MCP resolveu para uma organização que não existe (raro). |
| `post_failed` | Erro de provider/rede durante a publicação. Tente uma vez de novo; se persistir, surface a `message`. |

## Ferramentas legacy (deprecadas)

As ferramentas a seguir ainda funcionam mas foram superadas:

- **`POSTZEE_LIST_IMAGE_MODELS`** → use `POSTZEE_LIST_MODELS_DETAILED({ type: "image" })`
- **`POSTZEE_LIST_VIDEO_MODELS`** → use `POSTZEE_LIST_MODELS_DETAILED({ type: "video" })`

Serão mantidas por pelo menos 90 dias para retrocompatibilidade, depois removidas em uma futura versão major.

---

## Padrão de workflow assíncrono

Geração de imagem, vídeo e HeyGen são assíncronas. O workflow recomendado:

1. Valide primeiro — `POSTZEE_VALIDATE_GENERATION` para capturar erros de parâmetros e créditos insuficientes sem queimar créditos
2. Estime o custo — `POSTZEE_ESTIMATE_GENERATION_COST` se você precisar do número exato para mostrar ao usuário
3. Otimize o prompt — `POSTZEE_ENHANCE_PROMPT` (grátis)
4. Gere — `POSTZEE_GENERATE_IMAGE` / `POSTZEE_GENERATE_VIDEO` / `POSTZEE_GENERATE_HEYGEN_VIDEO` → retorna `jobId`
5. Faça poll — `POSTZEE_CHECK_JOB` a cada poucos segundos. Latências típicas: 10-60s para imagens, 30-180s para vídeos, até 5min para HeyGen
6. Em caso de sucesso — use `mediaUrl` em `POSTZEE_CREATE_POST`

## Formato de erro padrão

Todas as ferramentas write/generate retornam:

```json
{
  "success": true | false,
  "error": "<machine_code>",
  "message": "<human readable>",
  "...": "campos específicos da ferramenta em sucesso"
}
```

Faça branch no `error` machine code para lógica; use `message` para narrar ao usuário (traduzido para o idioma dele).
