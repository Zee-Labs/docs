Conecte o Postzee ao **Claude Code** para gerar imagens/vídeos com IA, criar vídeos com avatar HeyGen e publicar em mais de 30 redes sociais — tudo por conversa em linguagem natural.

## Instalar a Skill do Postzee

A forma mais fácil de conectar — instale a skill do GitHub.

### Opção 1: GitHub CLI

```bash
gh skill install Zee-Labs/postzee-skill
```

### Opção 2: Instalação manual

```bash
mkdir -p ~/.claude/skills/postzee
curl -o ~/.claude/skills/postzee/SKILL.md https://raw.githubusercontent.com/Zee-Labs/postzee-skill/main/SKILL.md
```

## Configurar a Conexão MCP

Após instalar a skill, conecte-se à sua conta Postzee:

1. Acesse [Configurações do Postzee](https://dashboard.postzee.app/settings) → aba **API Pública**
2. Copie a **URL do MCP** na seção "MCP (Model Context Protocol)"
3. Execute o seguinte comando no Claude Code:

```bash
claude mcp add postzee SUA_URL_MCP
```

Substitua `SUA_URL_MCP` pela URL completa copiada (parece com `https://api.postzee.app/mcp/.../http`).

### Verificar a conexão

```
Liste meus canais do Postzee
```

O Claude Code deve chamar a ferramenta `POSTZEE_LIST_CHANNELS` e mostrar suas contas de redes sociais conectadas.

## Exemplos de uso

Uma vez conectado, use linguagem natural para interagir com o Postzee:

### Gerar uma imagem

> "Gere uma foto profissional de uma mulher em um escritório moderno"

### Transformar uma foto (imagem-para-imagem)

> "Transforme essa foto em estilo anime" (forneça uma URL de imagem)

### Animar uma foto em vídeo

> "Anime essa foto em um vídeo cinematográfico para Reels do Instagram"

### Criar um vídeo com avatar HeyGen

> "Crie um vídeo HeyGen de um avatar apresentando nosso novo produto"

### Gerar e publicar

> "Gere uma foto de praia ao pôr do sol e poste no meu Instagram com a legenda 'Hora dourada'"

### Verificar créditos

> "Quantos créditos de IA eu tenho?"

## Workflow recomendado

Para melhores resultados:

1. **Verificar créditos** — confirme que tem créditos suficientes
2. **Otimizar prompt** — use o otimizador para melhores resultados (grátis)
3. **Gerar** — crie a imagem ou vídeo
4. **Publicar** — publique ou agende nos seus canais
