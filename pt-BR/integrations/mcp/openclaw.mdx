Conecte o Postzee ao **OpenClaw** para gerar imagens/vídeos com IA, criar vídeos com avatar HeyGen e gerenciar posts em redes sociais pelo seu agente OpenClaw.

## Instalar a Skill do Postzee

Instale a skill do ClawHub:

```bash
clawhub install postzee
```

Ou diga diretamente ao seu agente OpenClaw:

> "Instale a skill do Postzee"

## Configurar a Conexão MCP

Após instalar a skill, conecte-se à sua conta Postzee:

1. Acesse [Configurações do Postzee](https://dashboard.postzee.app/settings) → aba **API Pública**
2. Copie a **URL do MCP** na seção "MCP (Model Context Protocol)"
3. Diga ao seu agente: "Configure o Postzee com esta URL MCP: `SUA_URL_MCP`"

O agente armazenará a configuração automaticamente.

### Verificar a conexão

Diga ao seu agente:

> "Liste meus canais do Postzee"

Ele deve mostrar suas contas de redes sociais conectadas.

## Ferramentas disponíveis

Uma vez instalada, 20 ferramentas ficam disponíveis para seu agente:

| Ferramenta | Descrição |
|------------|-----------|
| `POSTZEE_LIST_CHANNELS` | Listar canais de redes sociais conectados |
| `POSTZEE_GET_CREDITS` | Verificar saldo de créditos de IA |
| `POSTZEE_LIST_IMAGE_MODELS` | Listar modelos de imagem com custos |
| `POSTZEE_LIST_VIDEO_MODELS` | Listar modelos de vídeo com custos |
| `POSTZEE_ENHANCE_PROMPT` | Otimizar prompts para melhores resultados (grátis) |
| `POSTZEE_GENERATE_IMAGE` | Gerar imagem com IA (suporta imagens de referência, aspect ratio, qualidade, estilo) |
| `POSTZEE_GENERATE_VIDEO` | Gerar vídeo com IA (suporta imagem-para-vídeo, duração, aspect ratio) |
| `POSTZEE_GENERATE_HEYGEN_VIDEO` | Criar vídeo com avatar HeyGen |
| `POSTZEE_LIST_HEYGEN_AVATARS` | Listar avatares HeyGen disponíveis |
| `POSTZEE_LIST_HEYGEN_VOICES` | Listar vozes HeyGen disponíveis |
| `POSTZEE_CHECK_JOB` | Verificar status da geração |
| `POSTZEE_CREATE_POST` | Criar, agendar ou publicar post |

## Exemplos de uso

- "Gere uma foto profissional de um produto"
- "Transforme essa foto em pintura a óleo" (envie foto pelo Telegram)
- "Anime minha foto em um vídeo para TikTok" (envie foto pelo Telegram)
- "Crie um vídeo HeyGen apresentando nossa empresa"
- "Poste uma imagem no LinkedIn com a legenda 'Mais um dia produtivo'"
- "Quantos créditos eu tenho?"
- "Gere um vídeo de 10 segundos de ondas do mar e poste no YouTube"

### Enviando imagens de referência via Telegram

Você pode enviar uma foto diretamente no chat do Telegram. O agente usará automaticamente como referência para geração imagem-para-imagem ou imagem-para-vídeo.
