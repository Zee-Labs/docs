O Postzee inclui um **servidor MCP** integrado (Model Context Protocol) que permite que agentes de IA interajam com sua conta Postzee. Gere imagens, crie vídeos, publique posts e gerencie suas redes sociais — tudo através de conversa em linguagem natural com seu agente de IA.

## O que é MCP?

O [Model Context Protocol](https://modelcontextprotocol.io) é um padrão aberto para conectar modelos de IA a ferramentas e fontes de dados externas. O servidor MCP do Postzee expõe 9 ferramentas que qualquer agente de IA compatível com MCP pode usar.

## Agentes de IA suportados

| Agente | Tipo de integração | Tempo de configuração |
|--------|-------------------|----------------------|
| **Claude Code** | Plugin (marketplace) | < 1 minuto |
| **OpenClaw** | Plugin (ClawHub) | ~ 2 minutos |
| **Qualquer cliente MCP** | Conexão HTTP direta | ~ 1 minuto |

## O que os agentes podem fazer

Com as ferramentas MCP do Postzee, um agente de IA pode:

- **Listar seus canais** — ver todas as contas de redes sociais conectadas
- **Verificar saldo de créditos** — visualizar créditos de IA disponíveis
- **Explorar modelos de IA** — listar modelos de imagem e vídeo disponíveis
- **Aprimorar prompts** — otimizar prompts para melhor geração com IA
- **Gerar imagens** — criar imagens com IA usando qualquer modelo
- **Gerar vídeos** — criar vídeos com IA usando qualquer modelo
- **Verificar status de jobs** — consultar tarefas de geração assíncronas
- **Criar posts** — publicar, agendar ou salvar rascunhos de posts em qualquer canal

## Como funciona

```
Você (conversa) → Agente de IA → Protocolo MCP → API do Postzee → Plataformas Sociais
```

1. Você descreve o que deseja em linguagem natural (ex.: "Gere uma foto de pôr do sol e poste no Instagram")
2. O agente de IA usa as ferramentas MCP do Postzee para executar a tarefa
3. O Postzee cuida da geração com IA, agendamento e publicação
4. O agente reporta os resultados

## Detalhes da conexão

| Propriedade | Valor |
|-------------|-------|
| **Nome do servidor** | Postzee |
| **Versão** | 3.0.0 |
| **Transporte** | Streamable HTTP (com fallback SSE) |
| **Endpoint** | `https://api.postzee.app/mcp/{apiKey}/messages` |
| **Autenticação** | Chave de API (embutida na URL) |
| **Ferramentas** | 9 |

## Requisitos

- Uma **conta Postzee** com plano pago (Standard ou superior)
- Uma **chave de API do Postzee** (crie uma em Configurações → aba API)
- Um **agente de IA compatível com MCP** (Claude Code, OpenClaw ou qualquer cliente MCP)
