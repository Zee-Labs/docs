O Postzee inclui um **servidor MCP** (Model Context Protocol) integrado que permite agentes de IA interagirem com sua conta. Gere imagens, crie vídeos, crie vídeos com avatar HeyGen, publique posts e gerencie suas redes sociais — tudo por conversa em linguagem natural.

## O que é MCP?

O [Model Context Protocol](https://modelcontextprotocol.io) é um padrão aberto para conectar modelos de IA a ferramentas e fontes de dados externas. O servidor MCP do Postzee expõe **12 ferramentas** que qualquer agente de IA compatível pode usar.

## Agentes de IA suportados

| Agente | Tipo de integração | Tempo de setup |
|--------|-------------------|----------------|
| **Claude Code** | Skill (GitHub) | < 1 minuto |
| **OpenClaw** | Skill (ClawHub) | < 1 minuto |
| **Hermes Agent** | Skill (GitHub) | < 1 minuto |
| **Qualquer cliente MCP** | Conexão HTTP direta | ~ 1 minuto |

## O que os agentes podem fazer

- **Gerar imagens** — com fotos de referência, aspect ratio, qualidade e estilo
- **Gerar vídeos** — de texto ou animando uma foto (imagem-para-vídeo)
- **Criar vídeos com avatar HeyGen** — avatares de IA falando seu roteiro
- **Otimizar prompts** — para melhores resultados (grátis)
- **Listar modelos de IA** — com custos em créditos
- **Verificar saldo** — créditos disponíveis
- **Listar canais** — contas de redes sociais conectadas
- **Criar posts** — publicar, agendar ou salvar como rascunho

## Como funciona

```
Você (conversa) → Agente de IA → Protocolo MCP → API Postzee → Redes Sociais
```

## Detalhes da conexão

| Propriedade | Valor |
|-------------|-------|
| **Nome** | Postzee |
| **Versão** | 3.1.0 |
| **Transporte** | HTTP Streamable (recomendado) + SSE (legacy) |
| **Endpoint HTTP** | `https://api.postzee.app/mcp/{apiKey}/http` |
| **Endpoint SSE** | `https://api.postzee.app/mcp/{apiKey}/sse` (legacy) |
| **Autenticação** | API key (embutida na URL) |
| **Ferramentas** | 12 |

## Requisitos

- Uma **conta Postzee** (gratuita ou paga)
- A **URL do MCP** (copie em Configurações → API Pública → seção MCP)
- Um **agente de IA compatível** (Claude Code, OpenClaw, Hermes Agent ou qualquer cliente MCP)
