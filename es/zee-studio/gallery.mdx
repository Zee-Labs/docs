Tu biblioteca de medios personal en Zee Studio está organizada en dos pestañas: **Creations** para contenido generado con IA y **Files** para archivos subidos manualmente.

## Creations

La pestaña **Creations** muestra todas las imágenes y videos que has generado con IA. Cada elemento muestra:

- Una vista previa en miniatura
- El modelo de IA utilizado para la generación
- Hace cuánto tiempo fue creado (ej., "hace 2 horas")

### Filtrar tus creaciones

Usa las pastillas de filtro encima de la galería para acotar tu vista:

| Filtro | Muestra |
|--------|-------|
| **All** | Todos los medios generados |
| **Images** | Solo imágenes generadas con IA |
| **Videos** | Solo videos generados con IA |

### Acciones sobre tus creaciones

Haz clic en cualquier elemento de Creations para acceder a estas acciones:

| Acción | Descripción |
|--------|-------------|
| **Descargar** | Guardar el archivo en tu dispositivo (JPG para imágenes, MP4 para videos) |
| **Crear Post** | Abrir el editor de posts con este medio adjunto |
| **Recrear** | Cargar el mismo modelo y configuración para generar una nueva variación |
| **Usar Prompt** | Copiar el prompt original de vuelta al campo de creación |
| **Ver Detalles** | Ver metadatos completos de generación — modelo, prompt, configuración, resolución |
| **Editar Imagen** | Abrir la imagen en el editor integrado para modificaciones (solo imágenes) |
| **Animar** | Transformar una imagen estática en un video usando el modo Imagen a Video (solo imágenes) |
| **Publicar** | Compartir en la galería pública Explore para que la comunidad la descubra |
| **Eliminar** | Eliminar permanentemente el medio (con confirmación) |

### Generaciones pendientes

Cuando una generación está en progreso, aparece en Creations con un indicador de carga. Puedes:

- Ver el estado del progreso (en cola, procesando)
- Continuar trabajando en otras tareas mientras esperas
- Cerrar Zee Studio y volver más tarde — el elemento estará listo cuando regreses

## Files

La pestaña **Files** muestra archivos que has subido manualmente. Úsala para tus propias fotos, activos de marca, logos o cualquier medio que quieras usar en posts.

### Cómo subir

- **Arrastrar y soltar** — arrastra archivos directamente al área de la galería
- **Clic para navegar** — haz clic en la zona de subida para abrir el selector de archivos
- **Botón de subida** — usa el botón de subida en la barra de herramientas de la galería

Consulta [Subir Medios](/zee-studio/upload) para formatos compatibles y límites de tamaño.

### Acciones sobre medios subidos

| Acción | Descripción |
|--------|-------------|
| **Descargar** | Guardar el archivo en tu dispositivo |
| **Crear Post** | Adjuntar el medio a un post en redes sociales |
| **Animar** | Generar un video a partir de una imagen estática (Imagen a Video) |
| **Eliminar** | Eliminar el archivo subido |

## Comportamiento de la galería

- Los medios se muestran en una cuadrícula responsiva que se adapta al tamaño de tu pantalla
- Los elementos están ordenados por fecha de creación, los más recientes primero
- La galería carga más elementos automáticamente al hacer scroll (scroll infinito)
- Las miniaturas están optimizadas para carga rápida
- Los videos muestran un icono de reproducción superpuesto en su miniatura
