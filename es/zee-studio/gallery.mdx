Tu biblioteca personal de medios en Zee Studio se organiza en dos pestañas: **Mis Creaciones** para el contenido generado por IA y **Mis Medios** para los archivos subidos manualmente.

## Mis Creaciones

La pestaña **Mis Creaciones** muestra todas las imágenes y vídeos que has generado con IA. Cada elemento muestra:

- Una miniatura de vista previa
- El modelo de IA usado para la generación
- Cuánto tiempo hace que se creó (por ejemplo, "hace 2 horas")

### Filtrar tus creaciones

Usa las píldoras de filtro encima de la galería para acotar la vista:

| Filtro | Muestra |
|--------|-------|
| **Todo** | Todos los medios generados |
| **Imágenes** | Solo imágenes generadas por IA |
| **Vídeos** | Solo vídeos generados por IA |

### Acciones sobre tus creaciones

Haz clic en cualquier elemento de Mis Creaciones para acceder a estas acciones:

| Acción | Descripción |
|--------|-------------|
| **Descargar** | Guarda el archivo en tu dispositivo (JPG para imágenes, MP4 para vídeos) |
| **Crear post** | Abre el editor de post con este medio adjunto |
| **Recrear** | Carga el mismo modelo y ajustes para generar una nueva variación |
| **Usar prompt** | Copia el prompt original al campo de creación |
| **Ver detalles** | Consulta todos los metadatos de generación — modelo, prompt, ajustes, resolución |
| **Editar imagen** | Abre la imagen en el editor integrado para modificarla (solo imágenes) |
| **Animar** | Transforma una imagen fija en un vídeo usando el modo Imagen a vídeo (solo imágenes) |
| **Publicar** | Compártelo en la galería pública Explorar para que la comunidad lo descubra |
| **Eliminar** | Elimina el medio de forma permanente (con confirmación) |

### Generaciones pendientes

Cuando una generación está en curso, aparece en Mis Creaciones con un indicador de carga. Puedes:

- Ver el estado de progreso (en cola, procesando)
- Seguir trabajando en otras tareas mientras esperas
- Cerrar Zee Studio y volver más tarde — el elemento estará listo cuando regreses

## Mis Medios

La pestaña **Mis Medios** muestra los archivos que has subido manualmente. Úsala para tus propias fotos, recursos de marca, logos o cualquier medio que quieras usar en publicaciones.

### Cómo subir archivos

- **Arrastrar y soltar** — arrastra archivos directamente al área de la galería
- **Hacer clic para examinar** — haz clic en la zona de subida para abrir el selector de archivos
- **Botón de subida** — usa el botón de subida en la barra de herramientas de la galería

Consulta [Subir medios](/zee-studio/upload) para conocer los formatos admitidos y los límites de tamaño.

### Acciones sobre los medios subidos

| Acción | Descripción |
|--------|-------------|
| **Descargar** | Guarda el archivo en tu dispositivo |
| **Crear post** | Adjunta el medio a un post en redes sociales |
| **Animar** | Genera un vídeo a partir de una imagen fija (Imagen a vídeo) |
| **Eliminar** | Elimina el archivo subido |

## Comportamiento de la galería

- Los medios se muestran en una cuadrícula responsiva que se adapta al tamaño de tu pantalla
- Los elementos se ordenan por fecha de creación, los más recientes primero
- La galería carga más elementos automáticamente al desplazarte (scroll infinito)
- Las miniaturas están optimizadas para una carga rápida
- Los vídeos muestran un icono de reproducción superpuesto en su miniatura
