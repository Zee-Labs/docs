Conecta Postzee a **OpenClaw** para generar imágenes/videos con IA, crear videos con avatar HeyGen y gestionar publicaciones en redes sociales a través de tu agente OpenClaw.

## Instalar la Skill de Postzee

Instala la skill desde ClawHub:

```bash
clawhub install postzee
```

O dile directamente a tu agente OpenClaw:

> "Instala la skill de Postzee"

## Configurar la Conexión MCP

1. Ve a [Configuración de Postzee](https://dashboard.postzee.app/settings) → pestaña **API Pública**
2. Copia la **URL del MCP** en la sección "MCP (Model Context Protocol)"
3. Dile a tu agente: "Configura Postzee con esta URL MCP: `TU_URL_MCP`"

### Verificar la conexión

> "Lista mis canales de Postzee"

## Herramientas disponibles

12 herramientas disponibles para tu agente:

| Herramienta | Descripción |
|-------------|-------------|
| `POSTZEE_LIST_CHANNELS` | Listar canales de redes sociales |
| `POSTZEE_GET_CREDITS` | Verificar saldo de créditos |
| `POSTZEE_LIST_IMAGE_MODELS` | Listar modelos de imagen con costos |
| `POSTZEE_LIST_VIDEO_MODELS` | Listar modelos de video con costos |
| `POSTZEE_ENHANCE_PROMPT` | Optimizar prompts (gratis) |
| `POSTZEE_GENERATE_IMAGE` | Generar imagen con IA (soporta imágenes de referencia, aspect ratio, calidad, estilo) |
| `POSTZEE_GENERATE_VIDEO` | Generar video con IA (soporta imagen a video, duración, aspect ratio) |
| `POSTZEE_GENERATE_HEYGEN_VIDEO` | Crear video con avatar HeyGen |
| `POSTZEE_LIST_HEYGEN_AVATARS` | Listar avatares HeyGen |
| `POSTZEE_LIST_HEYGEN_VOICES` | Listar voces HeyGen |
| `POSTZEE_CHECK_JOB` | Verificar estado de generación |
| `POSTZEE_CREATE_POST` | Crear, programar o publicar post |

## Ejemplos de uso

- "Genera una foto profesional de un producto"
- "Transforma esta foto en pintura al óleo" (envía foto por Telegram)
- "Anima mi foto en un video para TikTok" (envía foto por Telegram)
- "Crea un video HeyGen presentando nuestra empresa"
- "Publica una imagen en LinkedIn con el texto 'Otro día productivo'"
- "¿Cuántos créditos tengo?"

### Enviar imágenes de referencia por Telegram

Puedes enviar una foto directamente en el chat de Telegram. El agente la usará automáticamente como referencia.
