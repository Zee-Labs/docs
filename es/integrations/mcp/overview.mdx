Postzee incluye un **servidor MCP** (Model Context Protocol) integrado que permite a agentes de IA interactuar con tu cuenta. Genera imágenes, crea videos, crea videos con avatar HeyGen, publica posts y gestiona tus redes sociales — todo por conversación en lenguaje natural.

## ¿Qué es MCP?

El [Model Context Protocol](https://modelcontextprotocol.io) es un estándar abierto para conectar modelos de IA a herramientas externas. El servidor MCP de Postzee expone **20 herramientas**.

## Agentes de IA compatibles

| Agente | Tipo de integración | Tiempo de setup |
|--------|-------------------|-----------------|
| **Claude Code** | Skill (GitHub) | < 1 minuto |
| **OpenClaw** | Skill (ClawHub) | < 1 minuto |
| **Hermes Agent** | Skill (GitHub) | < 1 minuto |
| **Cualquier cliente MCP** | Conexión HTTP directa | ~ 1 minuto |

## Detalles de conexión

| Propiedad | Valor |
|-----------|-------|
| **Versión** | 3.1.0 |
| **Transporte** | HTTP Streamable (recomendado) + SSE (legacy) |
| **Endpoint HTTP** | `https://api.postzee.app/mcp/{apiKey}/http` |
| **Herramientas** | 12 |

## Requisitos

- Una **cuenta Postzee** (gratuita o de pago)
- La **URL del MCP** (copia en Configuración → API Pública → sección MCP)
