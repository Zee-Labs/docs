Conecta Postzee a **Claude Code** para generar imágenes/videos con IA, crear videos con avatar HeyGen y publicar en más de 30 redes sociales — todo a través de conversación en lenguaje natural.

## Instalar la Skill de Postzee

La forma más fácil de conectar — instala la skill desde GitHub.

### Opción 1: GitHub CLI

```bash
gh skill install Zee-Labs/postzee-skill
```

### Opción 2: Instalación manual

```bash
mkdir -p ~/.claude/skills/postzee
curl -o ~/.claude/skills/postzee/SKILL.md https://raw.githubusercontent.com/Zee-Labs/postzee-skill/main/SKILL.md
```

## Configurar la Conexión MCP

Después de instalar la skill, conecta tu cuenta Postzee:

1. Ve a [Configuración de Postzee](https://dashboard.postzee.app/settings) → pestaña **API Pública**
2. Copia la **URL del MCP** en la sección "MCP (Model Context Protocol)"
3. Ejecuta el siguiente comando en Claude Code:

```bash
claude mcp add postzee TU_URL_MCP
```

Reemplaza `TU_URL_MCP` con la URL completa copiada (se ve como `https://api.postzee.app/mcp/.../http`).

### Verificar la conexión

```
Lista mis canales de Postzee
```

Claude Code debería llamar a la herramienta `POSTZEE_LIST_CHANNELS` y mostrar tus cuentas de redes sociales conectadas.

## Ejemplos de uso

Una vez conectado, usa lenguaje natural para interactuar con Postzee:

### Generar una imagen

> "Genera una foto profesional de una mujer en una oficina moderna"

### Transformar una foto (imagen a imagen)

> "Transforma esta foto en estilo anime" (proporciona una URL de imagen)

### Animar una foto en video

> "Anima esta foto en un video cinematográfico para Reels de Instagram"

### Crear un video con avatar HeyGen

> "Crea un video HeyGen de un avatar presentando nuestro nuevo producto"

### Generar y publicar

> "Genera una foto de playa al atardecer y publícala en mi Instagram con el texto 'Hora dorada'"

### Verificar créditos

> "¿Cuántos créditos de IA tengo?"

## Workflow recomendado

Para mejores resultados:

1. **Verificar créditos** — confirma que tienes créditos suficientes
2. **Optimizar prompt** — usa el optimizador para mejores resultados (gratis)
3. **Generar** — crea la imagen o video
4. **Publicar** — publica o programa en tus canales
