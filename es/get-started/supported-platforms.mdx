## Características de las plataformas

Todas las plataformas soportan:
- **Contenido de texto** con formato específico de cada plataforma
- **Adjuntos de imagen** (individual o múltiples donde sea compatible)
- **Programación** — publicar ahora o programar para después
- **Borrador** — guardar contenido sin publicar

Algunas plataformas también soportan:
- **Subida de videos** (Instagram Reels, TikTok, YouTube, etc.)
- **Posts de carrusel** (Instagram, LinkedIn)
- **Stories** (Instagram, Facebook)
- **Configuraciones específicas de plataforma** (privacidad de TikTok, visibilidad de YouTube, etc.)

<Note>
Cada plataforma requiere una conexión OAuth independiente. Ve a **Canales** → **Agregar Canal** para conectar tus cuentas. Algunas plataformas (como Instagram y Facebook) usan la misma autorización de Facebook Business.
</Note>
