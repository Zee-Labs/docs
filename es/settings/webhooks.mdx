Los webhooks te permiten recibir notificaciones en tiempo real de Postzee cada vez que se publica una publicación. Crea integraciones y automatizaciones personalizadas conectando Postzee a tus propios sistemas.

## Cómo acceder

Abre el panel de la cuenta (haz clic en tu avatar en la esquina superior derecha) → **Configuración** → pestaña **Webhooks**.

## ¿Qué es un webhook?

Un webhook es una URL a la que Postzee llama (HTTP POST) cuando se publican publicaciones en tu cuenta. Postzee envía un payload JSON con los detalles de la publicación y tu servidor puede reaccionar como prefieras.

Casos de uso comunes:

- **Notificar a tu equipo en Slack/Discord** cuando se publica una publicación
- **Actualizar un CRM** cuando el contenido de un cliente se publica
- **Registrar eventos** en tu sistema de analítica o monitoreo
- **Sincronizar datos** con una base de datos personalizada

## Cómo configurar

1. Ve a **Configuración** → **Webhooks**
2. Haz clic en **Agregar Webhook**
3. Ingresa:
   - **URL** — el endpoint que tu servidor expone para recibir los eventos
   - **Nombre** — una etiqueta amigable para este webhook
   - **Canales** — elige **Todos los canales** o selecciona canales específicos para monitorear
4. Haz clic en **Guardar**

Postzee comenzará a enviar las publicaciones publicadas a tu URL.

## Filtrado por canal

Puedes limitar cada webhook a canales específicos:

- **Todos los canales** — recibe eventos para cada canal conectado
- **Canales específicos** — solo se dispara cuando las publicaciones se publican en los canales seleccionados

Esto es útil cuando deseas webhooks distintos para diferentes plataformas o clientes.

## Formato del payload

Cada entrega de webhook es un HTTP POST con el header `Content-Type: application/json`. El cuerpo es un arreglo JSON de las publicaciones publicadas recientemente:

```json
[
  {
    "id": "post-uuid",
    "content": "Post text content...",
    "publishDate": "2026-04-29T15:00:00Z",
    "integration": {
      "id": "channel-uuid",
      "name": "My Instagram",
      "providerIdentifier": "instagram"
    }
  }
]
```

Los webhooks se entregan en lote: si varias publicaciones se publican aproximadamente al mismo tiempo, pueden entregarse juntas en una sola solicitud.

El soporte de webhooks varía según el plan:

|------|----------|

| Standard | 2 |
| Team | 10 |
| Pro | 30 |
| Ultimate | 10,000 |
