Consulta tu saldo de créditos de IA y mejora prompts a través de la API.

## Obtener saldo de créditos

```
GET /public/v1/credits
```

Devuelve tu saldo actual de créditos.

### Solicitud

```bash
curl -H "Authorization: YOUR_API_KEY" \
  https://api.postzee.app/public/v1/credits
```

### Respuesta

```json
{
  "monthlyCredits": 5.0,
  "purchasedCredits": 2.0,
  "usedCredits": 0.5,
  "availableCredits": 6.5,
  "lastReset": "2026-04-01T00:00:00.000Z"
}
```

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `monthlyCredits` | number | Créditos mensuales de tu plan (en USD) |
| `purchasedCredits` | number | Créditos comprados que nunca expiran (en USD) |
| `usedCredits` | number | Créditos usados en el ciclo de facturación actual (en USD) |
| `availableCredits` | number | Total de créditos disponibles — mensuales + comprados - usados (en USD) |
| `lastReset` | string | Timestamp ISO del último reinicio mensual de créditos |

<Note>
Todos los valores de créditos están en USD. Para convertir a conteo de créditos: `valor x 1,000`. Por ejemplo, `availableCredits: 6.5` = `6,500` créditos disponibles.
</Note>

## Mejorar prompt

```
POST /public/v1/enhance-prompt
```

Optimiza un prompt para mejores resultados de generación IA. Este endpoint es **gratuito** — no se consumen créditos.

**Límite de tasa:** 10 solicitudes por minuto.

### Solicitud

```bash
curl -X POST \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "a cat on a windowsill",
    "mediaType": "image"
  }' \
  https://api.postzee.app/public/v1/enhance-prompt
```

### Parámetros

| Parámetro | Tipo | Obligatorio | Descripción |
|-----------|------|:---:|-------------|
| `prompt` | string | Sí | El prompt a mejorar (máx. 2,000 caracteres) |
| `mediaType` | string | Sí | `"image"` o `"video"` |
| `model` | string | No | ID del modelo de IA de destino para optimización específica del modelo |
| `stylePreferences` | string | No | Indicaciones de estilo (máx. 500 caracteres) |
| `sourceImageUrls` | string[] | No | URLs de imágenes de referencia (máx. 5) |

### Respuesta

```json
{
  "enhancedPrompt": "A fluffy ginger cat perched on a sunlit windowsill, warm afternoon light casting soft shadows through lace curtains...",
  "success": true
}
```

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `enhancedPrompt` | string | El prompt optimizado |
| `success` | boolean | Si la mejora fue exitosa |
| `error` | string | Mensaje de error (solo si `success` es `false`) |
