Your personal media library in Zee Studio is organized into two tabs: **Creations** for AI-generated content and **Files** for manually uploaded files.

## Creations

The **Creations** tab shows all images and videos you've generated with AI. Each item displays:

- A thumbnail preview
- The AI model used for generation
- How long ago it was created (e.g., "2 hours ago")

### Filtering your creations

Use the filter pills above the gallery to narrow your view:

| Filter | Shows |
|--------|-------|
| **All** | All generated media |
| **Images** | Only AI-generated images |
| **Videos** | Only AI-generated videos |

### Actions on your creations

Click on any item in Creations to access these actions:

| Action | Description |
|--------|-------------|
| **Download** | Save the file to your device (JPG for images, MP4 for videos) |
| **Create Post** | Open the post editor with this media attached |
| **Recreate** | Load the same model and settings to generate a new variation |
| **Use Prompt** | Copy the original prompt back to the creator field |
| **View Details** | See full generation metadata — model, prompt, settings, resolution |
| **Edit Image** | Open the image in the built-in editor for modifications (images only) |
| **Animate** | Transform a still image into a video using Image to Video mode (images only) |
| **Publish** | Share to the public Explore gallery for the community to discover |
| **Delete** | Permanently remove the media (with confirmation) |

### Pending generations

When a generation is in progress, it appears in Creations with a loading indicator. You can:

- See the progress status (queued, processing)
- Continue working on other tasks while waiting
- Close Zee Studio and come back later — the item will be ready when you return

## Files

The **Files** tab shows files you've uploaded manually. Use this for your own photos, brand assets, logos, or any media you want to use in posts.

### How to upload

- **Drag and drop** — drag files directly onto the gallery area
- **Click to browse** — click the upload zone to open your file picker
- **Upload button** — use the upload button in the gallery toolbar

See [Upload Media](/zee-studio/upload) for supported formats and size limits.

### Actions on uploaded media

| Action | Description |
|--------|-------------|
| **Download** | Save the file to your device |
| **Create Post** | Attach the media to a social media post |
| **Animate** | Generate a video from a still image (Image to Video) |
| **Delete** | Remove the uploaded file |

## Gallery behavior

- Media is displayed in a responsive grid that adapts to your screen size
- Items are sorted by creation date, newest first
- The gallery loads more items automatically as you scroll (infinite scroll)
- Thumbnails are optimized for fast loading
- Videos show a play icon overlay on their thumbnail
